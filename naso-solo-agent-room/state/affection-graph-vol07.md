# vol07 Affection Graph: Spec Sheet Lovers

> 기준 문서: `outline/vol07-spec-sheet-lovers.md`, `state/affection-graph-vol06.md`, `state/misunderstanding-queue-vol06.md`, `state/panel-arc.md`, `state/production-mystery-timeline.md`

## 사용 규칙

- 이 파일은 7권 원고 작성자가 각 장 시작 전에 확인할 감정 상태표다.
- 7권의 중심 질문은 "내가 고른 사람을 좋아하는가, 나를 좋아해 줄 확률이 높은 사람을 고르는가?"다. 모든 그래프 변화는 조건 일치율과 감정 동의를 분리해서 읽어야 한다.
- `spec_match`, `viewer_hook`, `low event density`, `progression score` 같은 LoveMatch 라벨은 작품의 판정이 아니다. 이 라벨은 참가자의 선택 환경을 좁히는 압력으로만 사용한다.
- 윤아의 카메라 거부는 까다로움이나 방송 비협조가 아니라, 관찰 동의와 선택 가능 상태를 지키려는 행동이다.
- 태린-해준과 보라-민겸은 7권에서 전부 해결하지 않는다. 화려한 장면과 느린 장면의 대비가 8권의 교착, Gemini 과적합, 자동 미션 압력으로 이어져야 한다.
- 장 끝에서 아래 상태와 충돌하는 새 반복 설정이 생기면 `state/continuity-notes.md`에 먼저 제안한다.

| Codex -> 조건표 | 7권 초반에는 조건 일치율, 프로필, 대화 품질이 가장 설득력 있는 분석처럼 보인다 | `expected: high progression`을 선언하지만 `08-part8`에서 붕괴해야 한다 | Codex의 실패는 무능이 아니라, 감정 동의를 스펙 매칭으로 증명하려 한 관점의 실패다 |

| Gemini -> 보이는 신호 | 화려한 첫인상, 늦은 웃음, 호흡 변화, 카메라 확인을 빠르게 읽는다 | 보이는 장면과 촬영 의식을 분리하려 하지만, 태린-해준 신호는 8권 과적합 씨앗이 된다 | Gemini는 마음을 확정하지 않는다. 감각 분석은 다음 장면을 다르게 읽게 하는 보조여야 한다 |
| GLM -> 고스펙 구조 | 프로필 공개, 홍보 전략, 추천 동선이 숙소 권력과 시선 배치를 바꾸는 것을 본다 | 지후와 태린에게 시선 권력이 몰리고, 민겸은 저밀도 장면으로 밀리는 구조를 기록한다 | 구조 분석이 윤아의 거절과 보라의 느린 감지 같은 개인 행동을 대체하지 않게 한다 |
| Grok -> 독설 | Act 2 이후 사람의 약점을 덜 비꼬고, 지후의 KPI식 매력과 시스템 허세를 겨냥한다 | 윤아의 불쾌감과 민겸의 느림은 농담 대상으로 삼지 않는다 | Grok의 독설은 시스템과 자기기만을 찌르는 칼이어야 한다. 참가자의 취약함을 소비하지 않는다 |

| Gemini -> 관찰 신호 | 윤아의 늦은 웃음과 보라의 호흡 변화는 읽었지만, 태린의 화려한 신호는 아직 불안까지 해석하지 못했다 | 8권에서 화려한 데이트 신호에 과적합했다가 태린의 무대 뒤 불안을 놓치는 붕괴로 이어진다 |
| GLM -> 권력 구조 | 고스펙 프로필과 추천 동선이 숙소 권력을 바꾸는 구조를 기록했다 | 8권의 순환 호감 구조와 9권의 관찰자 책임 인정으로 확장된다 |
| Grok -> 독설 원칙 | 사람의 취약함보다 지후의 KPI식 자기기만과 시스템 허세를 겨냥했다 | 8권 태린의 단독 퇴소 앞에서 농담을 멈추고, 9권 시스템 개입 앞에서 농담 없는 분노로 이동한다 |

## 검수 체크

- 지후-윤아의 조건 일치율은 감정 동의의 증거가 아니라, 둘이 만들고 싶은 이미지가 비슷하다는 압력으로 써야 한다.
- 윤아의 카메라 거부는 까다로움이나 방송 비협조로 판정하지 않는다. 선택 가능 상태를 지키는 행동으로 기록한다.
- 보라-민겸의 느린 장면은 낮은 이벤트 밀도로 취급되는 시스템 판정과 보라의 신체적 휴식 반응이 충돌해야 한다.
- 태린-해준의 화려한 장면은 7권에서 결론 내지 않고 8권 과적합과 단독 퇴소의 씨앗으로 남긴다.
- Codex의 조건 분석은 처음에는 충분히 그럴듯해야 한다. 그래야 `08-part8`의 붕괴가 관점 업데이트로 기능한다.
- LoveMatch는 선택을 직접 명령하지 않고 노출 편향과 미션 설명 재사용으로 선택 환경을 좁혀야 한다.
- Grok의 독설은 윤아의 불쾌감, 민겸의 느림, 태린의 불안이 아니라 지후의 성과 언어와 시스템 허세를 겨냥해야 한다.
