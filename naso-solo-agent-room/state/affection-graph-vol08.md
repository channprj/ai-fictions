# vol08 Affection Graph: Deadlock Paradise

> 기준 문서: `outline/vol08-deadlock-paradise.md`, `state/affection-graph-vol07.md`, `state/misunderstanding-queue-vol07.md`, `state/panel-arc.md`, `state/production-mystery-timeline.md`

## 사용 규칙

- 이 파일은 8권 원고 작성자가 각 장 시작 전에 확인할 감정 상태표다.
- 8권의 중심 질문은 "기다림은 배려인가, 책임 회피인가?"다. 모든 그래프 변화는 기다림이 상대의 선택 시간을 지켜 주는지, 아니면 위험을 상대에게 떠넘기는지 분리해서 읽어야 한다.
- `loop_retention`, `low event density`, `dropout_risk`, `retain: true` 같은 LoveMatch 라벨은 작품의 판정이 아니다. 이 라벨은 참가자 자율성을 좁히는 제작 시스템 압력으로만 사용한다.
- 태린의 단독 퇴소는 패배, 회피, 방송 실패가 아니라 더 보여 주지 않을 권리다.
- 보라-민겸의 느린 장면은 사건 부족이 아니라 보라가 비상상태에서 내려오는 리듬이다.
- 윤아의 조사는 감정선 이탈이 아니라 9권에서 선택 가능 상태를 묻기 위한 증거 축이다.
- 장 끝에서 아래 상태와 충돌하는 새 반복 설정이 생기면 `state/continuity-notes.md`에 먼저 제안한다.

| Gemini / 신호 과적합 | 화려한 장면을 진심으로 오독했고, 태린의 저장되지 않은 불안을 놓쳤다 | 9권에서는 카메라에 보이는 신호보다 선택 동의와 카메라 밖 말하지 않을 권리를 먼저 고려하는 업데이트로 이어진다 |
| Claude / 기다림 경계 | 기다림이 배려일 수도, 위험 전가일 수도 있음을 분리했고 보라-민겸 장면에서 강요하지 않는 돌봄을 기록했다 | 9권 최종 선택 중단 요구 앞에서 안전보다 동의를 우선하는 결론으로 확장된다 |
| GLM / 노출 구조 | 자동 미션이 관계망을 어떻게 좁히는지 기록했지만, 태린의 비선택이 구조를 거스르는 실제 행동임을 인정했다 | 9권에서 관찰자 책임과 공범성 논쟁으로 확장된다 |
| Grok / 독설 제한 | 태린의 단독 퇴소 앞에서 농담보다 존중의 짧은 문장을 남긴다 | 9권 시스템 개입 앞에서 참가자 조롱이 아니라 농담 없는 분노로 이동한다 |
| LoveMatch / 자동 미션 | 태린의 비선택을 `dropout_risk`로 오독하고 긴급 대체 미션을 제안한다 | 9권에서는 선택 유도 기능과 패널 로그 사용 경로가 드러나며, 최종 선택 중단으로 이어진다 |

## 검수 체크

- 기다림을 무조건 미덕으로 쓰지 않는다. 각 기다림은 상대의 선택 시간을 보장하는지, 책임을 넘기는지 장면 안에서 판별되어야 한다.
- 태린의 단독 퇴소는 사랑 실패가 아니라 보여 주지 않을 권리다. 해준의 후회를 중심 결론으로 바꾸지 않는다.
- Gemini의 과적합은 아름다운 화면과 진심을 혼동한 관점 붕괴로 써야 하며, 참가자 비난으로 처리하지 않는다.
- 보라-민겸 장면은 낮은 이벤트 밀도 판정과 실제 감정 진전이 계속 충돌해야 한다.
- 윤아의 조사는 9권 증거 축으로 충분히 구체적이어야 한다. 시간표, 카메라 위치, 하이라이트 후보 중 최소 2개 이상을 반복 단서로 남긴다.
- LoveMatch는 선택을 직접 강요하지 않는다. 시간표, 노출 빈도, 장면 크기, 편집 후보를 조정해 선택 환경을 좁힌다.
- Grok의 독설은 태린의 불안, 민겸의 느림, 보라의 멈춤을 조롱하지 않는다. 시스템과 자기기만만 겨냥한다.
