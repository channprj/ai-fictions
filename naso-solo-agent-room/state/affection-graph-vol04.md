# vol04 Affection Graph: Legacy Heart Migration

> 기준 문서: `outline/vol04-legacy-heart-migration.md`, `state/affection-graph-vol03.md`, `state/misunderstanding-queue-vol03.md`, `state/panel-arc.md`, `state/production-mystery-timeline.md`

## 사용 규칙

- 이 파일은 4권 원고 작성자가 각 장 시작 전에 확인할 감정 상태표다.
- 4권의 중심 질문은 "과거를 설명해야 사랑받을 수 있다면, 그 사랑은 시작 전부터 재판인가?"다. 모든 그래프 변화는 이해, 보호, 심문, 회피를 구분해서 읽어야 한다.
- Act 2의 관계는 설렘만으로 움직이지 않는다. 자녀, 가족 반응, 직업, 과거의 실패, 사후 책임이 감정과 같은 표 위에 놓인다.
- LoveMatch의 `risk`, `safe`, `stable` 라벨은 작품의 판정이 아니다. 참가자가 자기 과거를 어떤 문장으로 말할 권리를 갖는지가 우선이다.
- 장 끝에서 아래 상태와 충돌하는 새 반복 설정이 생기면 `state/continuity-notes.md`에 먼저 제안한다.

| ---- | ------------- | --------------------- | -------------- |
| 유찬 -> 세경 | 세경의 명확함에 끌리지만 아이와 과거를 이유로 먼저 속도를 낮추려 한다 | Claude는 유찬의 조심성을 안전한 배려로 보고, Codex는 책임감과 회피를 분리하려 한다 | 유찬의 따뜻함은 진짜지만 자기 행복을 사치로 여기는 습관도 같은 행동에서 나와야 한다 |
| 세경 -> 유찬 | 유찬의 조용한 배려를 알아보지만 자기 과거가 심사 대상이 될까 경계한다 | Gemini는 짧은 웃음과 손가락 힘에서 긴장을 읽고, Claude는 질문 순서를 경계한다 | 세경은 차가운 사람이 아니라 자기 서사권을 잃지 않으려는 사람이다 |
| 유찬 -> 아이 / 자기 행복 | 아이를 책임의 중심에 두지만, 그 책임 뒤에 자기 욕망을 숨긴다 | GLM은 가족 조건이 관계망을 바꾸는 핵심 노드라고 본다 | 아이는 로맨스 장애물이 아니라 유찬의 선택 비용을 드러내는 현실 조건이어야 한다 |
| 현오 -> 미리 | 능숙한 요리와 농담으로 미리를 편하게 만들려 하지만 무대 없는 진심을 두려워한다 | Grok은 처음에는 분위기 관리 자동화로 찌르다가 농담 뒤 실패 공포를 본다 | 현오의 매력은 과장하지 말고, 퍼포먼스가 실패 공포의 방어라는 양면을 유지한다 |
| 미리 -> 현오 | 현오의 세심함에 끌리지만 또 누군가의 작품이나 평가 대상으로 놓일까 경계한다 | Gemini는 늦게 닫히는 웃음과 목소리 균열을 잡는다 | 미리는 까다로운 사람이 아니라 자기 이름을 남의 라벨로 덮지 않으려는 사람이다 |
| 준하 -> 라온 | 안정적이고 잘 들어주는 라온에게 편안함을 느끼지만 그 편안함을 호감으로 빨리 분류한다 | Codex는 조건 안정성과 감정 진전을 같은 방향으로 오독할 수 있다 | 준하-라온 축은 4권에서 해결하지 말고 5권의 조건표 붕괴를 위한 압력으로 남긴다 |
| 라온 -> 준하 / 숙소 | 모두를 편하게 만들지만 자기 컵이 비어 있는 줄도 늦게 알 만큼 자기 기대를 미룬다 | GLM은 incoming signal 증가를 구조로 보고, Claude는 안전한 역할의 비용을 봐야 한다 | 라온은 편리한 상담자가 아니라 기대를 말하지 못하는 사람으로 설계한다 |
| 한서윤 / 백이현 -> LoveMatch | 3권 이상치 감사 파일 이후 새 위험 변수 대시보드를 보호 기능으로 믿고 싶다 | 백이현은 라벨을 중립 메타데이터로 보고, 한서윤은 참가자 말이 라벨보다 늦게 도착하는 점을 불편해한다 | 제작 시스템은 로맨스 판정을 대체하지 말고 선택 환경을 좁히는 압력으로만 쓴다 |
| Claude -> 세경 | 세경의 상처 노출을 줄이는 쪽으로 기울며 보호와 침묵을 혼동한다 | Grok과 Gemini가 "묻지 않음"도 고립이 될 수 있다고 반박한다 | Claude의 오류는 설교가 아니라 실제로 좋은 의도에서 생긴 실패여야 한다 |

## 검수 체크

- 세경의 경계는 차가움이 아니라 자기 서사권을 지키는 행동이다.
- 유찬의 아이는 장애물 장치가 아니라 책임과 회피가 같은 말로 포장되는 현실 조건이다.
- 현오의 농담은 웃기기 위한 장식이 아니라 실패 공포를 숨기는 방어다.
- 라온의 경청은 안전한 미덕이지만, 자기 기대를 지우는 비용을 남겨야 한다.
- Claude의 보호 오류는 악의가 아니라 좋은 의도에서 나온 판단 실패여야 한다.
- LoveMatch의 위험 변수는 참가자 보호처럼 보이되 실제로는 선택 환경을 좁히는 압력으로 이어져야 한다.
