# 캐논 업데이트 대기열

> 목적: outline과 원고 작성 중 새로 생기는 반복 설정을 바로 고정 캐논으로 만들지 않고, 검토 가능한 대기열에 모아 12권 장편의 모순을 줄인다.

## 사용 규칙

- `PRD.md`는 제품 범위와 권리 경계를 정하고, `BIBLE.md`는 승인된 고정 캐논을 정한다.
- 이 파일의 항목은 승인 전 기록이다. `approved-for-bible` 상태가 되기 전까지는 전체 시리즈 캐논으로 간주하지 않는다.
- 새 outline 또는 장 원고가 패널 말투, 참가자 과거, 제작 시스템 동작, 반복 장소, 권리 리스크를 추가하면 이 파일에 먼저 기록한다.
- 한 권 내부에서만 쓰이는 임시 디테일은 해당 권 outline 또는 권별 state 파일에 둔다. 다른 권에서 다시 쓰일 가능성이 있을 때만 이 대기열에 올린다.
- 기존 문서와 충돌하는 항목은 새 설정을 우선하지 않는다. 충돌 항목에는 어떤 문서의 어느 원칙과 부딪히는지 적고, 검토 후 수정하거나 폐기한다.
- 승인된 항목은 `BIBLE.md`, 관련 outline, 관련 state 파일 중 필요한 위치로 옮긴 뒤 이 파일에 반영 결과를 남긴다.

## 우선순위

캐논 충돌이 생기면 아래 순서로 판단한다.

| 순위 | 문서 | 판단 기준 |
| ---- | ---- | --------- |
| 1 | `PRD.md` | 권리 경계, 12권 구조, 핵심 주제, 금지사항 |
| 2 | `BIBLE.md` | 고정 캐논, 패널 말투, 세계 규칙, 장면 엔진 |
| 3 | `state/panel-arc.md` | 패널별 붕괴와 업데이트, 관계 변화 |
| 4 | `state/production-mystery-timeline.md` | LoveMatch 개입 단서, 제작진 책임선 |
| 5 | 권별 outline | 해당 권의 장면 목적, 갈등, 후킹 |
| 6 | 권별 state 파일 | 현재 권의 감정 방향과 오해 상태 |
| 7 | 원고 초안 | 장면 안에서 새로 생긴 세부 묘사 |
| 8 | 이 파일 | 승인 전 제안과 검토 기록 |

## 상태값

| 상태 | 의미 | 다음 행동 |
| ---- | ---- | --------- |
| `proposed` | 새 설정이 제안되었지만 아직 검토되지 않았다 | 충돌 문서와 반복 사용 가능성을 확인한다 |
| `needs-revision` | 방향은 유효하지만 기존 캐논 또는 권리 경계와 일부 충돌한다 | 수정안을 남긴 뒤 다시 검토한다 |
| `approved-for-bible` | 전체 시리즈 캐논으로 승격할 수 있다 | `BIBLE.md` 또는 관련 상태 파일에 반영한다 |
| `approved-local` | 특정 권 안에서는 유효하지만 시리즈 전체 캐논은 아니다 | 권별 outline 또는 state 파일에만 유지한다 |
| `rejected` | 캐논, 권리 경계, 톤 원칙과 맞지 않는다 | 같은 형태로 재사용하지 않는다 |
| `superseded` | 더 나은 항목으로 대체되었다 | 대체 항목 ID를 남긴다 |

## 항목 형식

새 항목은 아래 형식을 따른다.

```text
id: C-volNN-###
scope: series | act | volume | chapter
source: outline/volNN-... 또는 volNN/파일명
status: proposed
proposed_canon: 새로 반복될 가능성이 있는 설정
why_it_matters: 감정선, 패널 아크, 제작 미스터리, 권리 경계 중 어떤 문제를 해결하는지
conflicts_to_check: 확인해야 할 기존 문서 또는 원칙
resolution: 검토 결과와 반영 위치
```

ID 규칙:

- `C-vol01-001`처럼 권 번호와 세 자리 순번을 쓴다.
- 여러 권에 걸친 항목은 처음 발견된 권 번호를 붙인다.
- 패널 전용 항목은 `proposed_canon` 첫머리에 패널 이름을 명시한다.
- LoveMatch 관련 항목은 `conflicts_to_check`에 `state/production-mystery-timeline.md`를 반드시 포함한다.

## 승격 체크

항목을 `approved-for-bible`로 바꾸기 전에 다음 질문을 통과해야 한다.

- 실제 방송 회차, 실제 출연자, 실제 방송 대사와 닮지 않았는가.
- 참가자의 상처를 패널 농담이나 시스템 장치의 재료로만 쓰지 않는가.
- `사랑은 예측이 아니라 책임 있는 선택이다`라는 주제와 충돌하지 않는가.
- 패널 5명 중 누군가를 만능 해설자나 줄거리 요약기로 만들지 않는가.
- LoveMatch가 선택을 직접 명령하지 않고 선택 환경에만 영향을 주는가.
- 해당 설정이 한 권의 장면 재미를 넘어 다음 권에서도 유지될 필요가 있는가.

## 현재 대기열

현재 승인 대기 중인 새 캐논 항목은 없다. Phase 2의 1권 outline 작성부터 새 항목을 이 형식으로 누적한다.
