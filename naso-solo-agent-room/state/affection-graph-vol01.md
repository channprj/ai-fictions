# vol01 Affection Graph: First Impression Build

> 기준 문서: `outline/vol01-first-impression-build.md`, `state/panel-arc.md`, `state/production-mystery-timeline.md`

## 사용 규칙

- 이 파일은 1권 원고 작성자가 각 장 시작 전에 확인할 감정 상태표다.
- 참가자 내면은 독자에게만 공개될 수 있다. 패널은 카메라, 마이크, 제한된 메타데이터로 확인 가능한 신호만 말해야 한다.
- 그래프 변화는 말보다 행동, 동선, 침묵, 선택 미션 결과로 증명한다.
- 장 끝에서 아래 상태와 충돌하는 새 반복 설정이 생기면 `state/continuity-notes.md`에 먼저 제안한다.

| 선우 -> 하린 | 늦지만 분명한 행동을 남긴 상태 | 메모가 말보다 먼저 도착했고, 이제 직접 묻겠다는 약속을 지켜야 한다 |

| 하린 -> 도윤 | 빠른 신호에 흔들렸지만 검증되지 않은 상태 | 도윤의 확신이 낮의 책임으로 이어지는지 확인한다 |

| 은채 -> 태오 | 현실적 경계와 호기심이 공존 | 태오가 허세를 내려놓는지 기다린다 |

## 검수 체크

- 선우-하린 그래프는 1권 안에서 완전 해결하지 않는다. 해결이 아니라 2권 직접 질문의 압력을 만든다.
- 도윤은 삼각 구도의 장치가 아니라 성과형 선택 습관을 가진 사람으로 유지한다.
- 다정의 정보 허브화는 편리한 설명 도구가 아니라 감정 피로와 윤리 비용을 만들어야 한다.
- 패널 그래프와 실제 감정 그래프가 다를 수 있음을 유지한다.
