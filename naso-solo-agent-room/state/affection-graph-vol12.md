# vol12 Affection Graph: No Final Algorithm

> 기준 문서: `outline/vol12-no-final-algorithm.md`, `state/affection-graph-vol11.md`, `state/misunderstanding-queue-vol11.md`, `state/panel-arc.md`, `state/production-mystery-timeline.md`

## 사용 규칙

- 이 파일은 12권 원고 작성자가 각 장 시작 전에 확인할 마지막 관계 상태표다.
- 12권의 중심 질문은 "사랑을 가장 잘 이해한 순간, 우리는 왜 아무것도 예측하지 않기로 하는가?"다. 모든 관계 변화는 커플 성사율이 아니라 선택권, 말하지 않을 권리, 선택 이후 사후 책임으로 판단해야 한다.
- `last-season/no-dashboard`는 방치가 아니다. 호감도, 랭킹, 성공률, 선택 유도는 꺼지지만 안전 동선, 휴식 요청, 원본 열람권, 사후 상담, 위험 대응은 남아야 한다.
- 마지막 기수의 참가자는 주원, 해나, 태경, 서림, 도하, 민아다. 이들은 12권 outline에서 고정된 최종 기수이며, 기존 방송 참가자나 실제 인물을 참조하지 않는다.
- 최종권의 승리 조건은 커플 수가 아니라 자율성 보존이다. 성공, 유예, 비선택, 즉답 거부는 모두 선택 가능 상태를 지켰는지로 평가한다.
- 패널은 더 정확한 예측자가 되는 것이 아니라 예측하지 않는 관찰을 행동으로 증명해야 한다. `[NO PREDICTION]`은 정보 부족이 아니라 선택 유도 거부 원칙이다.
- 장 끝에서 아래 상태와 충돌하는 새 반복 설정이 생기면 `state/continuity-notes.md`에 먼저 제안한다.

## 초기 그래프

| 관계 | 12권 시작 상태 | 패널이 처음 읽는 방향 | 원고 작성 주의 |
| ---- | -------------- | ---------------------- | -------------- |
| 주원 -> 해나 | 좋은 선택은 좋은 경로에서 나온다고 믿고, 해나를 편하게 하려면 먼저 동선을 설계해야 한다고 생각한다 | Codex는 도움과 통제를 아직 열린 이슈로 두고, GLM은 경로 설계가 선택 환경을 좁힐 수 있음을 본다 | 주원의 배려를 악의로 만들지 않는다. 문제는 선의가 상대의 멈춤을 대신 결정하는 순간이다 |
| 해나 -> 주원 | 주원의 친절이 고맙지만, 피곤할 때도 괜찮다고 맞춰 주는 습관 때문에 쉬고 싶다는 말을 늦춘다 | Gemini는 해나의 짧은 숨을 보지만 확정하지 않고, Claude는 피로를 말할 권리를 안전 절차로 본다 | 해나를 수동적인 사람으로 쓰지 않는다. 자기 짐, 자기 속도, 자기 휴식 시간을 직접 말하게 해야 한다 |
| 태경 -> 서림 | 침묵을 잘 듣지만 자기 마음은 침묵 뒤에 숨긴다. 말하면 관계가 깨질까 봐 짧은 문장도 미룬다 | Gemini는 침묵의 질감을 보되 마음을 채우지 않고, Codex는 회피와 말하지 않을 권리를 구분하려 한다 | 침묵을 신비화하지 않는다. 태경은 마지막에 해석을 맡기지 않고 자기 책임 문장을 말해야 한다 |
| 서림 -> 태경 | 설명과 증명을 요구받아 온 피로가 있어, 좋아해도 모든 이유를 말하고 싶지는 않다 | Claude는 말하지 않을 권리를 동의 범위로 보고, GLM은 증명 압력이 구조적으로 반복되는 순간을 본다 | 서림의 단호함을 차가움으로 쓰지 않는다. 말하지 않음은 관계 거부가 아니라 경계 설정일 수 있다 |
| 도하 -> 민아 | 실패를 피하려고 모든 가능성을 확인하려 하고, 민아의 다정함을 호감 신호로 확정하고 싶어 한다 | Codex는 테스트 케이스 사고가 관계를 소모시키는 방식을 보고, Grok은 장난 뒤의 불안을 찌른다 | 도하의 장난은 가볍지만 상대를 실험 대상으로 만들면 비용이 생겨야 한다 |
| 민아 -> 도하 | 다정함과 선택 약속을 혼동받아 온 탓에, 친절을 줄이기보다 늦게 경계를 세우는 습관이 있다 | Claude는 친절과 동의의 차이를 보고, Gemini는 민아가 미소를 거두는 작은 순간을 유보한다 | 민아의 다정함을 보상처럼 지급하지 않는다. 경계는 관계를 끊는 것이 아니라 사람으로 남는 절차다 |
| 나은결 -> 마지막 촬영 | 무예측 촬영을 승인했지만 프로그램 생존과 마지막 회차 성과를 완전히 내려놓지는 못했다 | GLM은 승인 라인이 강태산의 내부 대시보드 압박에 취약하다고 본다 | 나은결을 갑작스러운 선역으로 만들지 않는다. 흔들림 끝에 비용 있는 결재를 해야 한다 |
| 한서윤 -> 참가자 | 11권에서 사후 절차를 정리했고, 마지막 기수에게도 원본 열람권과 말하지 않을 권리를 선제 조건으로 남기려 한다 | Claude는 절차가 참가자에게 실제로 닿는지 보고, Grok은 좋은 문장으로 도망치는 순간을 겨냥한다 | 서윤의 역할은 감동 문구가 아니라 참가자가 방송 이후에도 쓸 수 있는 절차를 남기는 것이다 |
| 백이현 -> LoveMatch | 예측 제거 패치 후보를 잠가 두었지만, 최종 대시보드 자동 산출 트리거가 남아 있음을 안다 | Codex는 `remove`와 `keep`을 분리하고, GLM은 내부 지표 누수 경로를 본다 | 백이현은 폭로 영웅이 아니라 운영 책임자다. 패치, 감사, 공개 범위를 분리해야 한다 |
| 강태산 -> 대시보드 | 성공률 대시보드는 조작이 아니라 내부 리스크 관리라고 믿고 마지막 복구를 밀어붙인다 | GLM은 내부 지표가 선택 환경으로 새는 경로를 보고, Grok은 참가자가 아니라 버튼과 책임 회피 문구를 비꼰다 | 강태산은 무능한 악역이 아니다. 숫자로 프로그램을 방어하는 논리가 설득력 있어야 한다 |
| Claude -> 최종 안전 | 안전은 선택을 제거하는 것이 아니라 선택 이후 사람이 부서지지 않게 하는 절차라는 결론에 도달한다 | 원본 열람권, 사후 상담, 말하지 않을 권리, 안전 동선을 선택권 보존의 하부 구조로 묶는다 | Claude가 설교하지 않게 한다. 최종권에서는 윤리 사유와 남길 기능의 범위로 말해야 한다 |
| Codex -> 최종 패치 | 예측 기능 제거 패치 후보를 실제 커밋해야 한다. 더 많은 테스트보다 멈춰야 할 기능을 끊는 일이 남았다 | `remove: prediction/ranking/nudge/success_rate_display`, `keep: safety_route/aftercare/raw_footage_review/risk_response`로 분리한다 | Codex는 줄거리 요약자가 아니라 영향 경로를 끊는 실행자다 |
| Gemini -> 최종 관찰 | 지표 없는 얼굴을 보았고, 이제 화면을 끄고 보이지 않기로 한 마음을 빈칸으로 남겨야 한다 | 침묵, 짧은 숨, 꺼진 녹음기 같은 신호를 결론으로 과적합하지 않는다 | Gemini가 최종권에서 다시 독심술사가 되면 안 된다. 유보 자체가 성장이다 |
| GLM -> 최종 구조 | 선택 유도 기능 중단의 영향 범위를 기록하고, 남길 보호 기능이 다시 통제로 변하지 않도록 경계한다 | 내부 지표, 카메라 위치, 미션 순서, 패널 발화가 선택 환경으로 누수되는 경로를 분리한다 | 구조 분석이 참가자의 작은 문장을 삼키면 안 된다. 주원, 해나, 서림의 직접 발화를 우선한다 |
| Grok -> 최종 농담 | 사람을 향한 조롱을 내려놓고, 마지막 독설을 시스템 버튼과 쓸모없는 책임 회피 문구에만 쓴다 | 버튼을 비꼬되 인간의 실패, 비선택, 침묵은 웃음거리로 만들지 않는다 | Grok의 마지막 대사는 웃겨도 잔인하면 안 된다. 농담의 방향이 인간에서 시스템으로 옮겨야 한다 |
| LoveMatch -> 보호 기능 | `last-season/no-dashboard`로 시작하지만 `final_dashboard_attempt: armed`가 남아 있다 | `[LOVE MATCH WARNING]`은 예측 제거와 보호 기능 유지가 동시에 필요함을 보여 준다 | 시스템을 완전히 폐기하지 않는다. 결말은 선택 유도 제한과 보호 기능 재배치다 |
| 관찰실 -> 로그 종료 | 관찰실은 마지막 권에서 예측 대신 기록을 남기고 서버를 종료해야 한다 | `[NO FINAL ALGORITHM]`은 `prediction: none`, `autonomy: preserved`, `care: retained`로 닫힌다 | 종료는 세계의 끝이 아니다. 참가자는 카메라 밖으로 계속 살아가야 한다 |

## 검수 체크

- 최종권의 성공 기준은 커플 수가 아니라 자율성 보존이어야 한다.
- `last-season/no-dashboard`를 안전 기능까지 꺼진 상태로 쓰지 않는다. 보호 기능은 반드시 남아야 한다.
- 주원, 도하의 선의를 악의로 단순화하지 않는다. 다만 배려와 테스트가 상대의 선택권을 압박하는 비용은 보여야 한다.
- 해나, 서림, 민아는 거절하거나 멈추거나 말하지 않을 때 관계를 망치는 사람이 아니다. 이들의 경계가 최종권의 주제다.
- 태경의 침묵은 낭만적 장식이 아니다. 마지막에는 해석을 상대에게 맡기지 않는 짧은 책임 문장이 필요하다.
- 강태산의 내부 대시보드 주장은 설득력 있게 쓰되, 내부 지표가 선택 환경으로 누수되는 경로를 반드시 반박 근거로 둔다.
- 패널은 최종권에서 예측 정확도를 겨루지 않는다. 예측하지 않는 선택을 실무적으로 수행해야 한다.
- Grok의 마지막 농담은 사람에게 향하지 않는다. 버튼, 라벨, 책임 회피 문구만 겨냥한다.
- 시스템 결말은 완전 폐기가 아니라 제한이다. 예측/랭킹/넛지는 종료하고 안전/사후 케어/원본 열람/위험 대응은 유지한다.
