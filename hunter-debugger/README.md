# 디버거

**E급 헌터, 세계를 디버깅하다**

> _죽음의 문턱에서 각성한 능력은 고작 E급._ > _하지만 그의 눈에만 보이는 것이 있었다._ > _세계의 소스코드._

## 소개

2026년 대한민국. 10년 전 출현한 던전과 헌터가 일상이 된 세계.

IT 회사 개발자 최수종(32)은 약혼녀에게 프러포즈하려던 날, 예고 없는 던전 브레이크를 만난다. 그녀를 지키려다 치명상을 입고, 죽음의 순간 각성한다. 하지만 눈을 떴을 때 약혼녀는 이미 세상에 없었고, 그에게 주어진 등급은 최하위 E급.

헌터기능사 시험에 두 번 낙방하고, 47개 길드에서 거절당하고, 고시원에서 하루하루를 버티는 나날. 그러나 수종에게는 다른 헌터에게 없는 것이 있었다. 오직 그의 눈에만 보이는 시스템 창. 마치 개발자 콘솔처럼.

시간을 멈추는 **브레이크포인트**. 되감는 **롤백**. 약점을 읽어내는 **디버그 아이**.

E급 개발자의 세계 디버깅이 시작된다.

**최수종 (32세)** — E급 헌터, 전직 IT 개발자

- 헌터기능사 (역대 최저 점수 합격)
- 논리적이고 내성적, 감정 표현에 서투름
- 고유 능력: 디버거 시스템

- 밝고 긍정적, 수종을 처음으로 파티에 받아줌

### 악역

**오진태 (52세)** — 대형 길드 "그림자 재벌" 총수

- 던전 이권 독점을 노리는 야심가
- 협회/정치권과 결탁

**심연의 문** — SSS급 던전

- 인류를 "버그"로 인식

## 세계관

### 던전 브레이크

2016년, 전 세계에 동시다발적으로 "게이트"가 출현했다. 게이트 너머에는 던전이 있고, 던전에는 몬스터가 있다. 게이트를 방치하면 "던전 브레이크"가 발생해 몬스터가 현실로 쏟아진다.

### 헌터

게이트 출현과 함께 일부 인간이 "각성"하기 시작했다. 초자연적 능력을 가진 자들, 헌터. 등급은 E(최하)부터 S(최상)까지. 전설로만 전해지는 SS, SSS급이 존재한다.

### 대한헌터협회 자격 체계

| 자격명       | 요구 등급 | 권한                 |
| ------------ | --------- | -------------------- |
| 헌터기능사   | E~D급     | 저등급 던전 출입     |
| 헌터산업기사 | C~B급     | 중급 던전 공략       |
| 헌터기사     | B~A급     | 고급 던전, 파티 리더 |
| 헌터기술사   | S급+      | 대형 레이드 지휘     |

### 헌터노임단가표

- E급: 187,000원/일
- D급: 280,000원/일
- C급: 450,000원/일
- B급: 850,000원/일
- A급: 1,500,000원/일
- S급: 협의
