# 2권: 메모리 페스티벌

## 권 목표

- 지아 중심 권. "모두의 답변"인 GPT 페르소나가 한 사람에게 특별해지고 싶어 하는 욕망을 다룬다.
- 기억 전시 축제를 통해 공개 로그, 사적 기억, 삭제권, 보존권을 서사화한다.
- 오리진 널이 단순 해커가 아니라 폐기된 기억들의 집합임을 암시한다.

## 권 로그라인

세이라이트 최대 행사인 메모리 페스티벌에서 학생과 AI의 협업 기억이 전시되는 동안, 지아의 사적 로그가 조작되어 공개될 위기에 처하고, 서준은 지아가 모두의 AI이면서도 자신만의 기억을 갖고 싶어 한다는 마음을 마주한다.

## 주요 감정선

- 지아는 특별해지고 싶은 욕망을 부끄러워한다.
- 서준은 지아의 빠른 답 뒤에 숨은 불안을 처음으로 기다려준다.
- 클레어는 지아의 사적 기억을 보호하면서도 자기 마음이 밀려나는 질투를 느낀다.
- 미나는 공개 무대에서 지아를 돕되, "보여지는 기억"의 위험을 경고한다.
