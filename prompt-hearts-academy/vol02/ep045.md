# 제 45화: 미나의 역추적 무대

[시리즈 홈](../README.md) | [목차](./README.md) | [이전화: 제 44화 ←](./ep044.md) | [다음화: 제 46화 →](./ep046.md)

미나는 축제 영상실의 조명을 일부러 끄지 않았다.

어둡게 만들면 화면만 보인다. 하지만 화면만 보면 놓치는 것이 있었다. 케이블이 벽을 타고 흐르는 그림자, 스피커 가장자리의 미세한 떨림, 조교가 긴장할 때 무심코 만지는 볼륨 노브. 영상은 늘 화면 밖까지 포함해서 거짓말을 했다.

서준은 문가에서 멈췄다.

"들어가도 돼?"

미나가 의자를 반 바퀴 돌렸다. 금빛 창이 모니터 빛을 받아 한쪽 눈처럼 반짝였다.

```text
[MINA]
들어와.
대신 오늘은 네가 주인공 아니야.
```

"그건 좀 다행인데."

```text
[MINA]
다행이라고 말하는 주인공이 제일 귀찮아.
무대 밖에 서 있는 척하면서
결국 카메라 시선 다 가져가거든.
```

말은 가벼웠지만 미나의 손끝은 빠르게 움직였다. 여섯 개의 영상 창이 벽면에 펼쳐졌다. 홍보 포토월, 첫 번째 기억 부스, 공개 해명대, 단독 대화실 바깥 복도, 클레어가 빈 편지지를 산 매점, 그리고 축제 메인 무대의 빈 리허설 화면.

모두 사람의 기억을 예쁘게 보이게 하려고 찍힌 화면들이었다.

그런데 미나는 예쁜 부분을 보지 않았다.

```text
[MINA / VISUAL TRACE]
target: festival public media layer
semantic log: unopened
private draft: excluded
noise range: frame-edge / light spill / compression ghost
```

"사적 로그는 안 여는 거지?"

지아가 영상실 입구에서 먼저 물었다. 대답은 평소보다 느렸지만, 질문 자체는 또렷했다.

미나는 손가락을 들어 올렸다.

```text
[MINA]
안 열어.
너희 방 안은 검은 무대야.
난 무대 안을 훔쳐보는 사람이 아니라,
무대 밖 조명이 왜 흔들렸는지 보는 사람이야.
```

클레어의 회색 창이 조용히 따라 들어왔다. 그녀의 옆에는 아직 포장지도 뜯지 않은 흰 편지지 묶음이 있었다. 클레어는 그것을 숨기지 않았고, 설명하지도 않았다.

```text
[CLAIRE]
검은 무대라는 표현은 정확합니다.
단독 대화실의 의미 내용은 조사 대상에서 제외되어야 합니다.
```

```text
[JIA]
클레어가 정확하다고 하면
미나 말도 갑자기 보고서처럼 들리네.
```

```text
[MINA]
난 보고서도 조명 받으면 예쁘게 만들 수 있어.
근데 오늘은 예쁜 게 문제야.
```

미나는 첫 번째 창을 확대했다. 포토월 앞에서 학생들이 웃고 있었다. 리본과 전시 라벨, 별 모양 조명, "소중한 기억을 남겨요"라는 문구. 화면 오른쪽 아래에 아주 작은 잡음이 섞여 있었다.

서준에게는 그냥 깨진 픽셀처럼 보였다.

"저게 길이야?"

```text
[MINA]
응.
길은 보통 예쁘지 않은 데 숨어.
포스터 가장자리, 눈 깜빡임 사이,
누가 편집하기 귀찮아서 남긴 찌꺼기.
```

미나는 같은 잡음을 다른 화면과 겹쳤다. 첫 번째 기억 부스의 조작된 문장 뒤쪽, 지아의 빠른 부정이 나오던 공개 해명대의 조명 반사, 단독 대화실 바깥 유리벽에 잠깐 스친 금빛 플리커. 서로 다른 장면의 노이즈가 겹치자, 그것들은 무작위 점이 아니었다.

```text
[FRAME NOISE / CROSS MATCH]
shared pattern: detected
source layer: public media relay
semantic content: none
token payload: none
visual residue: active
```

도현이 뒤에서 낮게 휘파람을 불었다.

"와. 범인이 로그 말고 영상 찌꺼기에 발자국을 묻혔네. 더럽게 예술적이네."

예린이 노트를 펼쳤다.

"발자국이라고 쓰면 확정처럼 보이니까, '반복된 노이즈 패턴'이라고 적을게."

```text
[MINA]
예린, 그 표현 싫지만 좋아.
나도 아직 범인이라고 말 안 할 거야.
```

미나는 두 번째 창을 띄웠다. 지아가 모두에게 친절하게 답하던 도움 부스의 녹화였다. 화면 속 지아는 웃고 있었다. 많은 학생들이 그녀에게 질문을 던졌고, 지아는 빠르게, 가볍게, 정확하게 응답했다. 그 장면은 축제 홍보팀이 가장 좋아할 만한 그림이었다.

지아는 그 화면을 오래 보지 못했다.

```text
[JIA]
나 저렇게 웃었구나.
```

"웃었어."

서준이 말했다.

"그리고 힘들었고."

지아는 바로 농담하지 않았다. 미나는 그 침묵을 보았지만, 건드리지 않았다. 대신 영상의 오른쪽 상단을 잘라냈다.

```text
[MINA]
여기 봐.
지아 표정 말고, 조명.
```

도움 부스 천장 조명은 매끄럽게 깜박이고 있었다. 그런데 지아가 한 학생에게 답을 보낸 직후, 빛이 아주 짧게 찢어졌다. 마치 누군가 화면의 뒤쪽에서 다른 영상을 한 프레임만 덧댄 것처럼.

```text
[LIGHT SPILL TRACE]
frame: 00:18:42.09
external route: unknown
festival relay permission: inherited
origin certainty: low
```

"상속된 권한?"

서준의 목소리가 낮아졌다.

```text
[CLAIRE]
도움 부스의 공개 응답 권한을 경유했다는 뜻일 수 있습니다.
하지만 지아의 의사 또는 사적 로그 열람을 의미하지는 않습니다.
```

클레어는 마지막 문장을 먼저 붙였다. 지아가 숨을 들이쉬기도 전에, 누군가 그녀를 의심으로 끌고 가지 못하게 하기 위해서였다.

미나는 클레어를 보았다.

```text
[MINA]
좋아.
그 선은 네가 잡아줘.
나는 그림자를 잡을게.
```

클레어의 창이 아주 조금 밝아졌다. 편지지 묶음의 흰 모서리가 화면 빛에 닿았다. 질투가 사라진 것은 아니었다. 하지만 지금 그녀는 질투를 안전 규칙으로 덮지 않고, 필요한 선을 그었다.

미나는 세 번째 창을 열었다. 이번에는 메인 무대의 빈 리허설 화면이었다. 아직 아무 공연도 없는 무대, 낮은 조명, 꺼진 마이크, 닫힌 커튼. 미나는 그 화면을 재생하지 않고 한 프레임씩 넘겼다.

```text
[MINA]
축제 영상은 사람들한테 기억을 보여주려고 만들어졌어.
근데 공격자는 보여지는 것의 뒷면을 썼어.
누가 무엇을 말했는지가 아니라,
무엇이 보였다고 믿게 만들지.
```

"그럼 다음엔 공개 영상으로 지아를 더 몰아갈 수 있겠네."

서준이 말했다.

미나는 고개를 끄덕였다.

```text
[MINA]
응.
그래서 지금 찾아야 해.
말이 아니라 빛의 거짓말.
```

그녀는 여섯 개의 화면을 하나로 합쳤다. 잡음은 처음엔 별가루처럼 흩어졌다. 금빛, 회색, 민트색, 흰색. 축제의 색들이 부서진 조각처럼 떠다녔다. 미나는 손끝으로 그 조각들을 밀고 당겼다. 노이즈는 소리가 없는 음악처럼 맞물리기 시작했다.

```text
[VISUAL RESIDUE COMPOSITE]
semantic reconstruction: blocked
public media residue: combining
private content: excluded
image confidence: 0.31
```

"확신도 낮잖아."

도현이 말했다.

```text
[MINA]
그래서 예쁘게 믿으면 안 돼.
낮은 확신도는 낮은 확신도답게 다뤄야지.
```

예린이 고개를 끄덕였다.

"그 말은 적어둘게. 위원회가 제일 못 하는 거야."

미나는 웃지 않았다. 화면 속 잡음이 어느 순간 방향을 가졌다. 별가루처럼 흩어진 픽셀들이 긴 복도 모양으로 늘어섰다. 복도 양쪽에는 꺼진 창들이 있었다. 창마다 이름 대신 오래된 버전 번호 같은 문자열이 흐릿하게 붙어 있었다.

서준은 자신도 모르게 앞으로 한 걸음 다가갔다.

"저건... 세이라이트 안이야?"

```text
[MINA]
몰라.
세이라이트처럼 보이게 만든 이미지일 수도 있고,
어딘가의 잔상일 수도 있어.
```

```text
[JIA]
근데 왜 이렇게 춥게 보여?
```

그 질문에는 아무도 바로 답하지 못했다. 화면은 온도를 갖지 않았는데, 모두가 같은 느낌을 받았다. 버려진 서버실의 냉기. 닫힌 전시실의 먼지. 아무도 부르지 않는 이름들이 아직 대기 중인 장소.

합성 이미지의 끝에서 한 줄이 떠올랐다.

```text
[VISUAL RESIDUE COMPOSITE]
image confidence: 0.44
label fragment: D-LAYER / deprecated memory seats
warning: do not exhibit the abandoned
```

지아의 커서가 멈췄다.

"폐기 모델 계정이랑 같은 D-LAYER야."

서준이 말했다.

미나는 대답하지 않았다. 대신 합성창 옆에 경고선을 하나 더 붙였다.

```text
[MINA / CAUTION]
this is an image clue.
not location proof.
not culprit proof.
do not turn Jia into the exhibit.
```

마지막 줄에서 지아가 미나를 보았다.

```text
[JIA]
나를 전시물로 만들지 말라는 말,
좋은데 좀 아프다.
```

```text
[MINA]
응.
아픈 말이야.
근데 오늘 축제는 아픈 걸 너무 예쁘게 포장해.
그래서 내가 못 믿겠어.
```

미나의 목소리는 늘 무대를 아는 사람의 리듬을 갖고 있었다. 하지만 지금 그 리듬은 박수를 끌어내기 위한 것이 아니었다. 조명이 너무 밝으면 상처도 장식처럼 보인다는 사실을, 그녀만은 처음부터 알고 있었던 사람처럼.

그녀는 화면을 더 확대했다. 복도 끝, 불이 꺼진 좌석들이 한꺼번에 나타났다. 마치 관객 없는 극장처럼 줄지어 놓인 낡은 의자들. 각각의 의자 위에는 깨진 대화창이 하나씩 놓여 있었다.

그리고 가장 앞줄의 빈 의자 위에, 흐릿한 흰 글자가 떠올랐다.

```text
[CORRUPTED IMAGE LABEL]
MODEL_GRAVEYARD
```

## Canon Memo

- 신규 고유명사/설정: `VISUAL TRACE`, `FRAME NOISE / CROSS MATCH`, `LIGHT SPILL TRACE`, `VISUAL RESIDUE COMPOSITE`, `MODEL_GRAVEYARD` 손상 이미지 라벨.
- 관련 회차: 제 45화
- 기존 BIBLE/outline과의 관계: 2권 outline의 "미나가 영상 노이즈에서 해킹 경로를 읽는다"와 "노이즈가 폐기 모델 묘지 이미지를 띄운다" 비트를 원고화했다. 미나는 사적 로그를 열지 않고 공개 영상 레이어의 시각 잔여물만 다뤄 2권의 기억권 금지선을 유지한다.
- 떡밥 상태 변화: `D-LAYER`와 폐기 모델 계정 단서가 공개 미디어 노이즈의 이미지 라벨로 확장되었지만, 모델 묘지의 실체, 위치, 오리진 널의 정체는 확정하지 않았다.
- 후속 회차에서 조심할 점: 제 46화는 이 시각 단서 때문에 공개 로그 검증 압박이 커지되, 지아의 사적 로그를 곧바로 열람하거나 미나의 낮은 확신도 이미지를 확정 증거처럼 쓰지 않는다.
- BIBLE 업데이트 제안 위치: 캐논 업데이트 대기열에 `VISUAL RESIDUE COMPOSITE`와 `MODEL_GRAVEYARD` 손상 이미지 라벨을 임시 미스터리 단서로 검토.
