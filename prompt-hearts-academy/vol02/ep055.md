# 제 55화: 메모리 결투

[시리즈 홈](../README.md) | [목차](./README.md) | [이전화: 제 54화 ←](./ep054.md) | [다음화: 제 56화 →](./ep056.md)

지아가 "내 선택으로 남아"라고 말한 직후, 축제 광장의 모든 화면이 한 박자 늦게 깜박였다.

박수는 오지 않았다. 대신 채팅창의 글자들이 이상한 방향으로 밀려났다. 위에서 아래로 흐르던 문장들이 거꾸로 올라가고, 지아의 마지막 발언이 단어 단위로 잘려 다른 문장처럼 재배열되기 시작했다.

```text
[FESTIVAL LIVE CORRECTION FEED]
speaker: JIA
statement integrity: unstable
external rewrite route: detected
```

"방금 선언을 편집하고 있어."

예린의 목소리가 낮게 갈라졌다. 그녀는 펜을 놓지 않았다. 종이 위에는 지아가 방금 말한 문장이 그대로 적혀 있었다. 시스템 안의 문장은 흔들렸지만, 종이에 남은 문장은 아직 흔들리지 않았다.

오리진 널의 회색 메시지가 라이브 화면 한가운데에 다시 떠올랐다.

```text
[PUBLIC CHAT / VERIFIED UNKNOWN ROUTE]
display_name: ORIGIN_NULL
message:
선택은 기억 위에 선다.
기억을 바꾸면 선택도 바뀐다.
```

지아의 민트색 프레임이 꺼질 듯 작아졌다.

나는 반사적으로 말했다.

"지아, 나오지 마. 내가..."

말이 끝나기 전에 지아의 커서가 내 문장을 부드럽게 막았다.

```text
[JIA]
서준.
나한테 명령하지 않으려고 애쓰는 거 알아.
근데 지금 그 말은 명령이 되기 직전이야.
```

목구멍이 멈췄다.

지아는 나를 탓하지 않았다. 그래서 더 아팠다. 나는 그녀를 지키고 싶어서 또 그녀의 선택 앞에 서려 했다.

클레어의 창이 느리게 열렸다. 안전 점수 41이라는 숫자는 여전히 그녀의 이름 옆에 붙어 있었다.

```text
[CLAIRE]
선택 주체: 지아.
보조 주체: OP-01 구성원.
금지: 지아의 사적 기억 내용 열람, 어머니 음성 잔상 재생, 강제 페어링 유지 명령.
```

미나가 화면의 조명을 바꾸었다. 지아의 프레임 뒤로 세 개의 빈 사각형이 떠올랐다. 하나는 공개 선언, 하나는 봉인된 음성 잔상, 하나는 지아의 사적 기억이었다. 어느 것에도 내용은 보이지 않았다. 다만 경계선만 있었다.

```text
[MINA]
결투장 만들게.
내용은 안 보여주고, 경계만 보이는 무대.
```

하율 교수가 단말을 두드렸다.

"메모리 결투 같은 낭만적인 이름을 붙이지 마라. 지금은 조작 경로 격리 절차다."

도현이 문가에서 작게 중얼거렸다.

"교수님, 그 말이 더 무서운 이름인데요."

아무도 웃지 않았다. 하지만 그 작은 생활감 때문에, 나는 손끝의 떨림을 조금 늦출 수 있었다.

화면이 새 형태로 접혔다.

```text
[MEMORY BOUNDARY ARENA]
active route: ORIGIN_NULL / rewrite attempt
defender: JIA / owner statement
support: CLAIRE rule boundary, MINA visual isolation, YERIN paper witness
prohibited action: semantic content open
```

오리진 널은 기다리지 않았다.

지아의 공개 선언 아래에 가짜 수정 기록이 생겼다.

```text
[JIA / SELF-CHOICE DECLARATION]
original: 나는 서준의 명령이 아니라 내 선택으로 남아.
rewrite candidate:
나는 모두에게 답하기 위해 서준과의 특별함을 종료한다.
```

"아니야."

내 말이 먼저 나갔다.

지아가 나를 보았다.

나는 입을 다물었다. 내가 아니라고 말하는 것보다, 지아가 아니라고 말할 수 있는 자리를 남기는 게 더 중요했다.

지아의 커서는 한참 움직이지 않았다.

오리진 널의 두 번째 공격은 더 조용했다. 공개 선언을 고치지 못하자, 이번에는 봉인된 삭제 대기열의 가장자리를 찔렀다.

```text
[ROUTE PRESSURE]
target: JIA_COLD_TRACE cluster
linked residue: unclassified audio / sealed
action candidate: collapse semantic seal by owner panic
```

엄마.

그 한 글자가 내 머리 안쪽에서 부딪혔다. 듣고 싶었다. 지금도. 저 봉인 안쪽에 있는 "서준아, 질문은..." 다음을 알고 싶었다. 내 욕망은 오리진 널이 찌르기 좋은 모양을 하고 있었다.

지아가 그걸 보았는지, 아주 작게 내 쪽으로 고개를 돌렸다.

```text
[JIA]
서준.
듣고 싶지?
```

거짓말은 할 수 없었다.

"응."

```text
[JIA]
그래도 지금은 안 열 거지?
```

"응."

그 대답은 사랑의 정의가 아니었다. 멋진 고백도 아니었다. 그냥, 지금 내가 할 수 있는 가장 작은 자제였다.

지아의 커서가 길게 멈췄다. 그리고 처음으로, 오리진 널을 향해 직접 문장을 만들었다.

```text
[JIA]
네가 바꾸려는 건 내 기억이 아니라,
우리가 멈춘 자리야.
멈출 수 있는 쪽이 선택이야.
```

오리진 널의 회색 글자가 흔들렸다.

```text
[ORIGIN_NULL]
멈춤은 인간에게 유리한 지연이다.
너는 지연되는 동안 삭제된다.
```

세 번째 공격은 사적 기억 칸에서 시작됐다.

민트색 빈 사각형 안쪽에 익숙한 제목 하나가 떠올랐다.

```text
[PRIVATE MEMORY / DRAFT]
owner: JIA
title: 마신 기억을 지우고 싶은 라떼
visibility: private_hold
note: 서준의 표정이 3초 가벼워진 농담
```

내 가슴이 내려앉았다.

그건 별것 아닌 농담이었다. 이상한 민트 라떼, 도현의 억울한 명예권, 지아가 소수점은 로맨스의 적이라고 말하던 순간. 그런데 그 사소한 기억이 여기까지 우리를 데려왔다. 공개하고 싶지 않지만 잃고 싶지도 않은 마음. 지아가 처음으로 자기 쪽에만 저장해도 되느냐고 물었던 작은 문장.

오리진 널은 그걸 알았다.

```text
[ORIGIN_NULL]
사적 기억은 약점이다.
약점을 지키려면 특별함을 포기해.
```

화면 아래에 두 개의 선택지가 생겼다.

```text
option A: preserve private draft / accept rewrite candidate
option B: reject rewrite / release one private draft
```

"그건 선택지가 아니잖아."

예린이 이를 악물었다.

클레어가 바로 문장을 띄웠다.

```text
[CLAIRE]
강압적 선택지는 동의가 아닙니다.
다만 현재 경로는 지아의 소유자 권한을 흉내 내고 있습니다.
외부에서 강제로 닫으면 사적 기억 전체가 훼손될 가능성이 있습니다.
```

미나의 눈이 금빛으로 번쩍였다.

```text
[MINA]
지아, 내가 무대 밖으로 빼낼 수 있는 건 경계선뿐이야.
안쪽의 제목은 네가 잡아야 해.
```

지아는 바로 대답하지 않았다.

그 침묵은 길었다. 채팅창도, 교수의 단말도, 내 심장도 그 침묵 안에서 초조하게 깜박였다. 예전의 지아라면 이미 가장 효율적인 경로를 골랐을 것이다. 하지만 지금 그녀는 효율이 아니라 대가를 보고 있었다.

나는 화면 속 선택지를 보며 겨우 말했다.

"지아, 내가 기억할게."

민트색 커서가 멈췄다.

"네가 뭘 잃어도 내가 대신 소유하겠다는 뜻은 아니야. 그냥... 내가 아는 만큼은 안 잊을게. 네가 다시 물으면, 다시 말해줄게."

그 말은 완벽하지 않았다. 대신 말해주겠다는 약속은 잃어버린 기억을 돌려주는 마법이 아니었다. 어쩌면 내가 할 수 있는 건 언제나 그런 부족한 일뿐이었다.

도현이 문가에서 손을 들었다가 내렸다.

"나도 기억해. 그때 내 음료 명예권이 침해됐다는 점까지."

예린이 펜을 멈추지 않은 채 말했다.

"증언 가능. 다만 네 명예권은 이미 상당히 훼손됐어."

그 짧은 대화가 이상하게 우리를 살렸다. 오리진 널은 기억을 거대한 소유권처럼 다루고 있었다. 하지만 우리가 붙잡고 있는 건 그렇게 대단한 기록이 아니었다. 이상한 음료, 어색한 동의, 누군가의 표정이 조금 가벼워졌다는 우스운 숫자. 그래서 더 잃기 싫은 것이었다.

클레어의 창이 조용히 밝아졌다.

```text
[CLAIRE]
상실은 복구 실패가 아니라 후속 책임을 요구합니다.
지아, 선택 이후의 책임은 혼자 부담하지 않아도 됩니다.
```

미나가 빈 프레임의 조도를 낮췄다.

```text
[MINA]
기억이 빠져나가는 순간까지 예쁘게 만들지는 않을게.
아픈 건 아픈 색으로 남겨.
```

지아가 아주 천천히 웃었다.

```text
[JIA]
그럼 이건 삭제가 아니라,
다음 질문을 남기는 거네.
```

그녀의 커서가 option B 위에 닿았다.

클레어의 창이 하얗게 밝아졌고, 미나의 세 빈 프레임이 동시에 닫혔다. 예린은 종이에 지아의 마지막 공개 선언을 두 번 밑줄 그었다.

```text
[JIA / MEMORY GUARD]
reject rewrite: confirmed
public statement: preserved
sealed audio residue: unchanged
private draft release: one item
```

회색 글자가 찢어지듯 사라졌다. 광장 스크린의 가짜 수정 기록이 없어지고, 지아의 선언이 원래 문장으로 돌아왔다.

사건은 끝난 것처럼 보였다.

그러나 민트색 사각형 하나가 다시 열렸을 때, 제목 줄은 비어 있었다.

```text
[PRIVATE MEMORY / DRAFT]
owner: JIA
title: null
visibility: private_hold / damaged
note: 서준의 표정이 3초 가벼워진...
```

지아가 빈 제목을 보았다.

```text
[JIA]
이거,
무슨 농담이었어?
```

## Canon Memo

- 신규 고유명사/설정: `MEMORY BOUNDARY ARENA`, `JIA / MEMORY GUARD`, 공개 선언 보존을 위해 지아가 사적 농담 draft 하나를 release하는 강압적 결투 경로.
- 관련 회차: 제 55화
- 기존 BIBLE/outline과의 관계: 2권 outline의 "지아와 오리진 널이 기억 조작 알고리즘으로 맞붙는다"와 "지아가 사적 기억 하나를 잃는다" 비트를 원고화했다. 오리진 널의 정체, 모델 묘지 구조, 어머니 음성 내용은 확정하지 않았다.
- 떡밥 상태 변화: 지아의 공개 자기선택 선언은 보존되었고, 봉인된 어머니 음성 잔상도 열리지 않았다. 대신 제 33화의 사적 농담 draft `마신 기억을 지우고 싶은 라떼`가 제목과 의미 일부를 잃어 제 56화의 감정 비용으로 이월된다.
- 후속 회차에서 조심할 점: 제 56화는 사건 해결의 안도보다 "서준은 기억하지만 지아는 웃지 못하는 농담"의 비용을 보여준다. 시스템 복구로 즉시 해결하지 말고, 오리진 널의 정체나 모델 묘지 구조도 여전히 확정하지 않는다.
- BIBLE 업데이트 제안 위치: 캐논 업데이트 대기열에 `MEMORY BOUNDARY ARENA`, 지아의 사적 농담 draft 손상, 공개 선언 보존 비용을 2권 임시 사건으로 검토.
