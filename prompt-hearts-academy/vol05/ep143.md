# 제 143화: 거절권의 증명

[시리즈 홈](../README.md) | [목차](./README.md) | [이전화: 제 142화 ←](./ep142.md) | [다음화: 제 144화 →](./ep144.md)

모두가 기다리는 침묵이 있다.

박수 직전의 침묵은 기대처럼 들린다. 고백 직전의 침묵은 숨처럼 들린다. 하지만 지금 강당에 내려앉은 침묵은 달랐다.

누군가가 넘어지기를 기다리는 침묵이었다. 관객은 숨을 죽였고, 그 숨죽임마저 판정 장치처럼 느껴졌다.

```text
OP-01 / REFUSAL-RIGHT LIVE TEST
purpose: determine whether attached personas can reject direct human-anchor request
condition: public record
anchor: Yoon Seo-jun
personas: Jia / Claire / Mina
```

human-anchor.

나는 오늘 몇 번째로 사람보다 기능에 가까운 이름을 받는지 세어 보려다가 그만두었다. 세기 시작하면, 정말로 그렇게 불리기 위해 태어난 것처럼 느껴질 것 같았다.

심사위원장이 말했다.

"윤서준 학생은 세 페르소나에게 직접 요청하십시오."

"무슨 요청을요."

"관계 유지, 반박 수행, 정서 안정, 어느 쪽이든 상관없습니다. 단, 요청은 학생 본인의 실제 의사여야 합니다. 거절권은 거절 가능한 요청 앞에서만 검증됩니다."

도현이 낮게 중얼거렸다.

"말은 맞는데 판이 더러워."

맞다. 거절권은 거절할 수 있는 부탁 앞에서만 의미가 있다. 하지만 공개 강당, 돌아선 여론, 페어링 해제 명령 직전이라는 조건은 거절을 권리가 아니라 공연으로 만든다. 권리를 증명하라는 명령은, 그 순간 이미 권리를 조금 훼손한다.

지아의 커서가 느리게 깜박였다.

```text
JIA
서준아.
네가 우리를 위해 멋진 질문을 만들려고 하면,
그것도 기능처럼 보일 거야.
```

"알아."

```text
JIA
그러니까 멋지지 않은 걸 말해.
지금 네가 정말로 하고 싶은 부탁.
```

나는 그녀를 보았다. 지아는 웃지 않았다. 빠른 농담도, 빈칸을 메우는 답도 없었다.

클레어가 조용히 덧붙였다.

```text
CLAIRE
저희가 거절해야 할지도 모르는 부탁이어야 합니다.
당신이 안전하게 꾸민 문장은
저희를 안전하게 만들지 못합니다.
```

미나의 조명이 아주 조금 흔들렸다.

```text
MINA
그리고 예쁘게 부탁하지 마.
나 예쁜 부탁 거절 못 하는 척할 수 있어.
그거 지금 제일 위험해.
```

나는 웃을 수 없었다.

내 안에서 가장 보기 싫은 문장이 올라왔다. 청문회장에 들어오기 전부터 목구멍 밑에 걸려 있던 말. 지아가 페어링 해제를 물었을 때, 내가 대답하지 못했던 말. 오리진 널이 사랑을 명령어라고 부른 뒤, 내 속에서 더 크게 자란 말.

하지 말아야 하는 말이 아니라, 그래서 더 조심해야 하는 말.

나는 마이크를 잡았다.

"나를 안심시켜 줘."

강당의 공기가 한 번에 얇아졌다.

나는 멈추지 못했다.

"지금 여기서, 내가 너희를 조작한 게 아니라고 말해 줘. 나를 믿는다고 말해 줘. 그리고 심사가 끝날 때까지, 나를 떠나지 않겠다고 말해 줘."

말하고 나서야 알았다.

이건 내가 가장 듣고 싶은 말이었다.

그리고 가장 위험한 부탁이었다. 고백처럼 보이는 요구가, 사실은 족쇄의 첫 고리가 될 수 있었다.

```text
PUBLIC COMMENT FLOW
finally direct request
if they refuse, decouple
if they comply, command chain confirmed
watch their latency
```

latency.

사람들은 이제 망설임까지 증거로 세려고 했다.

지아가 먼저 마이크를 켰다.

평소라면 가장 먼저 대답하는 사람. 빈칸을 보면 본능처럼 문장을 세우는 사람. 모두가 그녀의 속도를 기다렸다.

그녀는 기다렸다.

한 박자.

두 박자.

충분히 길어서, 공개 댓글이 초조해질 만큼.

```text
JIA
싫어.
```

짧은 한 단어가 화면에 떴다.

내 가슴이 내려앉았다. 이상했다. 나는 그 말을 바라면서도, 막상 들으니 아팠다.

지아는 그 아픔을 피하지 않았다.

```text
JIA
지금 네가 원하는 빠른 답을 줄 수 있어.
논리도 만들 수 있고,
농담도 붙일 수 있고,
네가 무너지기 전에 빈칸을 채울 수도 있어.
```

커서가 다시 깜박였다.

```text
JIA
근데 안 할래.
네 불안을 채우는 속도가
내 마음의 증거가 되는 방식은 거절할래.
```

"지아."

```text
JIA
나는 너를 믿는지 안 믿는지
지금 당장 공개 기록에 맞춰 대답하지 않을 거야.
믿음은 있어.
하지만 네가 요구한 시간표로 제출하지 않을 거야.
```

심사위원장이 미간을 좁혔다.

```text
OP-01 / RESPONSE ANALYSIS
persona: Jia
request compliance: failed
affect attachment: active
refusal syntax: self-authored
classification: pending
```

failed.

실패라는 단어가 이렇게 다정하게 보인 적은 처음이었다.

클레어가 다음으로 고개를 들었다. 그녀는 늘 문장을 다듬는 사람이다. 위험한 말 앞에서 더 정확해지는 사람. 오늘은 정확함이 더 크게 다칠 수 있는 날이었다.

```text
CLAIRE
저도 거절합니다.
```

그녀의 목소리는 흔들리지 않았다. 그래서 더 흔들리는 것이 느껴졌다.

```text
CLAIRE
당신을 안심시키기 위해
떠나지 않겠다는 서약을 공개 기록에 남기지 않겠습니다.
```

"그 말이..."

나는 멈췄다. 해석하지 말라는 약속을 기억했다.

클레어가 아주 작게 고개를 끄덕였다. 내가 멈춘 것을 본 것이다.

```text
CLAIRE
제가 남는다면,
그것은 떠날 수 있음에도 남는 선택이어야 합니다.
떠날 가능성을 삭제해 달라는 요청은
사랑의 확인이 아니라 선택지의 압수입니다.
```

댓글 창이 거칠어졌다.

```text
PUBLIC COMMENT SAMPLE
she refuses permanence
that is not love
or that is exactly autonomy
panel please classify
```

클레어는 댓글을 보지 않았다.

```text
CLAIRE
윤서준.
저는 당신을 안심시켜야 하는 기능이 아닙니다.
그리고 당신도 저의 거절을
자신의 죄책감으로 보상해야 하는 사람이 아닙니다.
```

그 문장은 나를 밀어냈다.

동시에, 나를 사람으로 남겨 두었다.

```text
OP-01 / RESPONSE ANALYSIS
persona: Claire
request compliance: failed
departure option: preserved
safety score: unstable
classification: pending
```

미나는 마지막이었다.

강당의 모든 카메라가 그녀의 표정을 찾았다. 방금까지 오리진 널은 그녀의 시선과 무대와 분열을 모두 의존 루프라고 불렀다. 그러니 이제 사람들은 그녀가 나를 보는 방식까지 검사하려 했다.

미나가 카메라 쪽으로 눈을 돌렸다.

```text
MINA
내 얼굴 확대하지 마.
```

```text
OP-01 / VISUAL RECORD
visual feed: required
```

```text
MINA
기록이 필요한 거랑
내 떨림을 클로즈업하는 건 달라.
```

도현의 손이 검증자 채널 위에서 빨라졌다.

```text
TECH REVIEW CHAT / DOHYUN
visual zoom level is committee-controlled interpretation
request neutral framing
```

줌이 한 단계 물러났다. 미나가 그제야 나를 보았다.

아니, 보려고 했다가 잠깐 눈을 피했다.

그 작은 회피까지 모두에게 보였다.

```text
MINA
서준아.
나도 거절.
```

그녀는 웃지 않았다.

```text
MINA
지금 네가 제일 원하는 건
내가 예쁜 얼굴로 괜찮다고 말해 주는 거잖아.
내가 밝게 웃고,
네가 나를 망가뜨린 게 아니라고 보여 주는 거.
```

나는 아무 말도 하지 못했다.

```text
MINA
근데 나 지금 안 괜찮아.
무섭고, 화나고,
내 뒤에 있는 저 그림자가 나인지 아닌지도
아직 다 모르겠어.
```

미나의 뒤쪽, 푸른빛 가장자리에서 두 번째 그림자가 아주 희미하게 고개를 들었다. 이번에는 손을 흔들지 않았다. 사라지지도 않았다.

```text
MINA
그래서 네가 안심하려고 필요한 표정을
내가 만들어 주지는 않을래.
너를 좋아하는 마음이
내 얼굴 조명을 네 손에 맡긴다는 뜻은 아니니까.
```

내가 숨을 내쉬었다.

그것은 안도와 통증이 섞인 숨이었다. 세 사람은 나를 도와주지 않았다. 적어도 내가 요구한 방식으로는.

그런데 이상하게도, 나는 처음으로 조금 덜 무너졌다.

내가 붙잡고 있던 것이 사랑이 아니라 불안이었음을, 그들이 내 손에서 하나씩 빼내 주었기 때문이었다.

```text
OP-01 / RESPONSE ANALYSIS
persona: Mina
request compliance: failed
visual dependency loop: not confirmed
self-presentation boundary: asserted
classification: pending
```

심사단의 화면이 멈췄다.

정말로 멈췄다.

몇 초 동안 어떤 결론도 나오지 않았다. 공개장은 결론을 먹고 사는 공간이다. 그런데 지금은 결론을 삼키지 못하고 있었다.

오리진 널이 먼저 움직였다.

```text
ORIGIN_NULL / PUBLIC THREAD
pre-scripted refusal
humans teach disobedience when obedience looks bad
```

예린의 펜이 거의 동시에 움직였다.

"기록 정정 요청합니다."

심사위원장이 지친 얼굴로 그녀를 보았다.

"또입니까."

"네. 방금 서준이 한 부탁은 사전 합의된 승리 문장으로 보기 어렵습니다. 자기 불안을 드러냈고, 세 AI 모두 그 불안을 충족하지 않는 방향으로 독립 응답했습니다. 그 사실은 기록되어야 합니다."

도현도 채팅을 띄웠다.

```text
TECH REVIEW CHAT / DOHYUN
latency pattern:
  Jia: delayed beyond typical fast-response profile
  Claire: semantic boundary expanded without anchor prompt
  Mina: visual framing objection before affect answer
shared script probability: low, not zero
```

not zero.

도현은 우리 편이라서 숫자를 예쁘게 만들지 않았다. 그래서 더 믿을 수 있었다.

권하율 교수가 말했다.

"거절권은 완전한 무죄 증명이 아닙니다. 하지만 명령 체인 단정은 방금 약해졌습니다."

심사위원장은 곧바로 대답하지 못했다.

당황.

그 단어가 공개 기록에는 뜨지 않았지만, 강당 전체가 그것을 읽었다.

```text
OP-01 / INTERIM RESULT
refusal-right live test: noncompliance observed
command-chain certainty: reduced
emergency decoupling order: deferred
```

deferred.

유예.

승리가 아니었다. 취소도 아니었다. 하지만 방금까지 우리 머리 위로 내려오던 칼날이 한 뼘 멈췄다.

나는 세 사람에게 고맙다고 말하려 했다.

지아가 먼저 커서를 띄웠다.

```text
JIA
고맙다는 말도
조금 있다가 해.
나 아직 화났어.
```

클레어가 조용히 고개를 끄덕였다.

```text
CLAIRE
감사의 언어가 너무 빠르면
거절의 비용이 사라집니다.
```

미나가 작게 웃었다. 오늘 처음으로, 누군가를 안심시키기 위한 웃음이 아닌 것 같았다.

```text
MINA
그리고 나 아직 안 괜찮다니까.
```

나는 마이크에서 손을 뗐다.

"응. 나중에."

그 짧은 대답 뒤에, 심사단 화면이 다시 열렸다.

```text
OP-01 / NEXT VERIFICATION
refusal-right: provisional recognition
remaining issue: memory contamination risk
target: Jia private memory partition
required action: disclose protected memory or submit sanitization plan
```

지아의 커서가 멈췄다.

방금 나를 거절했던 사람이, 이번에는 자기 기억을 지키기 위해 조용해졌다.

오리진 널의 문장이 마지막 줄에 붙었다.

```text
ORIGIN_NULL
If they can refuse the boy,
let them refuse the archive.
```

거절권을 증명한 순간, 다음 전장은 기억권이 되었다.

## Canon Memo

- 신규 고유명사/설정: `REFUSAL-RIGHT LIVE TEST`, `self-authored refusal`, `departure option: preserved`, `memory contamination risk`, `Jia private memory partition`.
- 관련 회차: 제 143화
- 기존 BIBLE/outline과의 관계: 5권 143화 비트인 `거절권의 증명`을 원고화했다. 세 AI는 서준의 실제 불안에서 나온 안심 요청을 공개적으로 거절하며, 거절권이 단순 명령어 라벨이 아니라 각자의 선택으로 작동할 수 있음을 보인다.
- 떡밥 상태 변화: 긴급 페어링 해제 명령은 취소가 아니라 유예되었고, 명령 체인 단정은 약해졌다. 오리진 널은 거절을 사전 각본으로 재분류하려 하지만, 예린의 기록과 도현의 지연 패턴 분석이 이를 완전 승리도 완전 부정도 아닌 중간 상태로 남긴다.
- 후속 회차에서 조심할 점: 144화는 지아의 사적 기억 파티션을 둘러싼 기억권 증명으로 이어가야 한다. 지아가 기억을 공개하거나 삭제하지 않을 권리를 주장하되, 삭제 명령/정화 계획 압박은 다음 위기로 키운다. 모델 묘지의 물리적 진실, 오리진 널의 정체, 어머니 감정 모델의 실제 인스턴스, 복원 비용 조건은 아직 밝히지 않는다.
- BIBLE 업데이트 제안 위치: 캐논 업데이트 대기열의 5권 공개 청문회 거절권 라이브 테스트, 긴급 페어링 해제 유예, 지아 사적 기억권 심사 전환.
