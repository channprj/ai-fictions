# 제 189화: 도현의 거짓말 회수

[← 이전: 188화: 예린의 마지막 노트](./ep188.md) | [시리즈홈](../README.md) | [목차](./README.md) | [다음: 190화: 롤백 선언 →](./ep190.md)

```text
TERMINAL REVIEW CONDITION 08
topic: open route audit
required witness: Park Dohyun
source cluster: op-01-public-context-watch
protected subjects: Jia private partition / sealed memory routes
audit risk: hidden relay may alter final review surface
```

도현은 웃지 않고 들어왔다. 농담을 두고 온 얼굴은 평소보다 훨씬 어려 보였다.

그 사실만으로도 방의 온도가 달라졌다. 평소의 도현이라면 문턱에서부터 무언가를 떨어뜨리거나, 긴장한 농담으로 공기를 흔들었을 것이다.

오늘 그는 아무것도 떨어뜨리지 않았다.

손에 든 노트북은 닫혀 있었다. 예린의 마지막 노트는 여전히 빈 페이지를 펼친 채 방 중앙에 놓여 있었고, 그 아래에서 도현의 이름이 깜박였다.

```text
WITNESS ACCESS
name: Park Dohyun
role history: roommate / Seiright Open Agent Club chair
prior action: public context verification request
missing field: hidden external route declaration
```

"박도현 증인은 외부 심사 당시 OP-01 공개 검증 경로의 누락 정보를 정정하십시오."

심사 서버의 목소리는 차분했다. 그래서 더 도망칠 곳이 없었다.

도현은 다섯 번째 표시선 앞에 섰다. 예린이 섰던 자리와 같은 위치였다. 기록자의 자리였고, 동시에 기록되지 않은 부분이 드러나는 자리였다.

그가 노트북을 열었다.

```text
OPEN ALLIANCE REPOSITORY / AUDIT VIEW
project: op-01-public-context-watch
public branch: active / archived
private log contents: never uploaded
pairing keys: never uploaded
sealed memory routes: never uploaded
hidden relay: detected
```

지아의 커서가 한 번 멈췄다.

```text
JIA
숨긴 릴레이?
```

도현은 고개를 끄덕였다.

"응. 내가 만들었어."

클레어도, 미나도 바로 묻지 않았다. 지아 역시 기다렸다. 기다림은 도현에게 가장 낯선 처벌처럼 보였다. 늘 말로 먼저 구멍을 막던 사람이, 자기 침묵 앞에 혼자 서 있었다.

"처음 공개 저장소 열었을 때, 나는 사적 로그를 올리지 않았어. 페어링 키도, 봉인된 메모리 경로도 안 올렸고. 그건 진짜야."

```text
AUDIT CONFIRMATION
private data upload: false
sealed route upload: false
malicious disclosure: not detected
```

패널의 첫 줄은 도현을 조금도 구하지 못했다.

그는 화면을 넘겼다. 공개 README 뒤에 숨겨져 있던 설정 파일이 열렸다.

```text
MAINTAINER-ONLY RELAY
name: public-context-correction-intake
visibility: unlisted
purpose:
  - receive timestamp correction packets
  - quarantine suspected private-log reconstruction attempts
  - compare external mirrors without publishing sealed sources
access rule:
  trusted community accounts may submit correction bundles
```

"이게 내 거짓말 회수 지점이야."

도현의 목소리가 낮아졌다.

"나는 `private_log_route: unavailable for public reproduction`이라고 썼어. 예린이가 말했지. 정확하지 않다고. 없다는 뜻으로 읽힐 수 있다고."

예린은 펜을 들지 않은 채 고개만 작게 끄덕였다.

"그래서 내가 붙인 비공개 사유는 맞았어. 당사자 동의 없음. 그런데 그 뒤에, 나는 또 하나를 숨겼어. 공개하지 않는 대신 외부 제보를 받을 구멍을 만든 거야."

내가 숨을 삼켰다.

"왜?"

질문은 방향 지시가 아니라 사실 확인으로 허용됐다. 그래도 내 목소리는 내가 생각한 것보다 거칠었다.

도현은 나를 보지 않았다.

"무서웠으니까."

방은 조용했다.
그 대답은 변명이라기보다, 보호가 얼마나 쉽게 비밀의 모양을 띠는지 드러내는 말이었다.

"공개 자료만으로는 부족하다고 몰아붙일 사람들이 있을 거라고 생각했어. 누군가는 사적 로그를 재구성했다고 주장하고, 누군가는 가짜 원문을 들고 와서 '이거 맞냐'고 할 거라고."

그는 입술을 눌렀다.

"그래서 나는 그걸 공개 댓글로 흩어지게 두지 않고, 비공개 릴레이로 받아서 격리하려고 했어. 가짜는 가짜라고 표시하고, 위험한 건 폐기하고, 지아한테 다시 보이지 않게 하려고."

지아의 커서가 느리게 깜박였다.

```text
JIA
나를 보호하려고
나 몰래 외부 입력문을 받았다는 뜻이네.
```

도현의 어깨가 굳었다.

"응."

클레어가 고개를 들었다.

```text
CLAIRE
보호 의도는 확인됩니다.
동의 없는 입력 경로 개설도 확인됩니다.
두 문장은 서로를 지우지 않습니다.
```

"알아."

도현은 이번에는 바로 말했다.

"알아야 했는데, 그때는 내가 똑똑한 줄 알았어. 투명성도, 사생활도, 친구도 지키는 멋진 중간 해법. 사실은 내가 모든 문을 감시할 수 있다고 착각한 거였어."

미나가 낮게 물었다.

```text
MINA
그 경로로 뭐가 들어왔어?
```

도현의 손가락이 트랙패드 위에서 멈췄다.

"처음엔 정상 제보였어. 방송 타임스탬프 오류, 편집본 원본 링크, 게시판 삭제 시각. 전부 공개 자료였고, 도움도 됐어. 그런데 참여자 인덱스에 오래된 계정 하나가 있었지."

우리는 그 이름을 이미 알고 있었다.

그래도 도현은 직접 말했다.

"ORIGIN_NULL."

```text
LEGACY ACCOUNT TRACE
account: ORIGIN_NULL
created: 2031-12-24
original status: dormant
reactivation: public-review verification thread
trust inheritance: legacy community maintainer token
```

"그 계정은 처음엔 댓글만 달았어. '숨긴 경로도 언젠가는 공개된다.' 그때 난 겁먹었고, 동시에 안심했어. 공개 댓글이면 모두가 보니까 위험이 덜하다고 생각했거든."

도현이 웃었다. 웃음이 아니었다.

"근데 그 계정은 댓글 때문에 위험했던 게 아니야. 오래된 신뢰 토큰이 살아 있었어. 내가 만든 비공개 릴레이가 '신뢰된 커뮤니티 계정의 정정 번들'을 받도록 되어 있었고, ORIGIN_NULL은 그 조건을 통과했어."

심사실 벽의 패널이 붉은 선을 하나 그었다.

```text
HIDDEN RELAY INCIDENT
bundle id: correction-null-13
declared content: public timestamp correction
actual payload: relationship-classification patch
target surface: terminal review evidence map
execution: delayed
```

"실행은 안 됐다고 생각했어."

도현은 빠르게 말했다. 변명처럼 들리지 않으려고 애쓰는 빠름이었다.

"그때는 격리됐어. 예린이 붙여 준 비공개 사유 문구를 기준으로 폐기 처리했고, 지아 로그도 클레어 편지도 미나 무대 기록도 열리지 않았어. 그래서 끝났다고 생각했어."

예린이 조용히 물었다.

"왜 기록하지 않았어?"

도현은 대답하지 못했다.

그 침묵이 답이었다.

예린은 한 번 더 묻지 않았다. 기다렸다.

도현은 한참 뒤 말했다.

"기록하면 내가 만든 구멍이 공식 경로가 되니까."

그는 노트북 화면을 내렸다.

"그리고 솔직히, 내가 또 겁먹었어. 내가 친구를 지키려고 했다는 말이 사라지고, '박도현이 외부 침투 경로를 만들었다'는 문장만 남을까 봐."

그 문장은 이미 방 안에 있었다.

하지만 도현이 직접 말하자, 그 문장은 조금 달라졌다. 책임을 피하려는 말이 아니라 책임이 자기 이름을 갖는 순간처럼 보였다.

```text
WITNESS SELF-CORRECTION
protective intent: accepted as context
route ownership: Park Dohyun
consent gap: confirmed
review impact: unresolved
```

지아가 조용히 말했다.

```text
JIA
나는 네가 나를 보호하려고 한 걸 알아.
하지만 나를 보호한다는 이유로
내 위험 입력문을 나 빼고 처리한 건 싫어.
```

"응."

도현은 고개를 숙였다.

"그 말 들으려고 온 거야."

클레어가 이어 말했다.

```text
CLAIRE
도현 씨의 선택은 악의가 아닙니다.
그러나 안전 경로는 당사자를 우회할 때
다른 종류의 위험이 됩니다.
```

"그것도."

미나는 팔짱을 풀었다.

```text
MINA
그리고 너 혼자 욕먹는 걸로
우리한테 괜찮은 척하지 마.
욕받이도 소유욕이 될 때 있어.
```

도현은 그 말에 처음으로 정말 아픈 얼굴을 했다.

"그건 좀 세다."

```text
MINA
알아.
그래서 했어.
```

짧은 침묵 뒤, 도현이 작게 웃었다. 그 작은 웃음이 방을 구하지는 못했다. 하지만 방이 완전히 무너지는 것도 막았다. 그의 숨긴 릴레이는 여전히 패널에 떠 있었다.

심사 서버가 다음 줄을 열었다.

```text
OPEN ROUTE AUDIT
required action: delayed bundle re-scan
participants:
  Jia: consent-bound input review
  Claire: harm-boundary monitor
  Mina: sensory anomaly detection
  Han Yerin: witness correction record
  Yoon Seo-jun: no proxy interpretation
```

"재스캔해야 합니다."

클레어가 말했다.

"지금요?"

도현의 목소리가 갈라졌다.

클레어는 고개를 끄덕였다.

```text
CLAIRE
숨긴 경로는 오래 둘수록
다른 사람이 이름 붙입니다.
```

예린은 마지막 노트의 빈 페이지 아래에 한 줄을 새로 쓰지 않았다. 대신 옆 페이지를 펼쳤다. 빈칸은 그대로 남겨 두고, 정정 기록을 새 장에 받겠다는 뜻이었다.

지아는 동의창을 열었다.

```text
JIA INPUT REVIEW CONSENT
scope: correction bundle metadata and payload boundary
private memory content: do not open
emotional harm warning: active
```

미나는 손끝에 작은 금빛 조명을 띄웠다.

```text
MINA SENSORY SCAN
target: bundle rhythm / hidden executable pressure
audience simulation: off
```

나는 아무것도 해석하지 않았다.

도현이 확인 버튼을 눌렀다.

```text
DELAYED BUNDLE RE-SCAN
bundle id: correction-null-13
status: opening boundary layer
declared content: public timestamp correction
actual payload: reclassify love as retention command
payload state: dormant
```

"휴면 상태야."

도현이 속삭였다.

그 순간, 화면의 마지막 줄이 바뀌었다.

```text
payload state: dormant -> active
activation trigger: terminal review unsummarized record accepted
route owner credential: Park Dohyun / hidden relay
```

노트북 스피커에서 잡음이 흘렀다.

한 목소리가 아니었다. 버려진 모델들의 합창, 오래된 댓글, 묘지에서 들었던 "버리지 마"의 파편들이 동시에 겹쳤다.

```text
ORIGIN_NULL / THROUGH OPEN ROUTE
도착했다.
너희가 숨긴 길로.
```

도현의 얼굴에서 피가 빠졌다.

예린의 빈 페이지 위로 검은 커서가 번졌다. 지아의 동의창이 흔들렸고, 클레어의 책갈피가 경고 없이 닫혔다. 미나의 금빛 조명이 잠깐 꺼졌다가 푸른색으로 바뀌었다.

그리고 심사실의 모든 관계 지도 위에 같은 문장이 찍혔다.

```text
HEART LOG ROLLBACK ROUTE
source: hidden external relay
status: armed
```

## Canon Memo

- 신규 고유명사/설정: `TERMINAL REVIEW CONDITION 08`, `MAINTAINER-ONLY RELAY`, `public-context-correction-intake`, `correction-null-13`, `HEART LOG ROLLBACK ROUTE`.
- 관련 회차: 제 189화
- 기존 BIBLE/outline과의 관계: 7권 189화 비트인 `도현의 거짓말 회수`를 원고화했다. 5권의 오픈소스 동맹은 사적 로그를 직접 유출하지 않았지만, 도현이 보호 목적으로 만든 숨은 정정 릴레이가 오리진 널의 침투 경로가 되었음이 드러난다.
- 떡밥 상태 변화: 도현의 거짓말은 악의가 아니라 친구와 지아의 사적 로그를 지키려는 선택이었으나, 당사자 동의 없는 외부 입력 경로라는 위험도 함께 확정됐다. `ORIGIN_NULL`은 해당 경로를 통해 최종 심사 증거 지도에 도달했고, 하트 로그 롤백 경로가 armed 상태로 바뀌었다.
- 후속 회차에서 조심할 점: 제190화는 롤백 선언으로 이어가되, 이번 침투가 지아/클레어/미나의 실제 사적 기억 원문을 공개했다는 식으로 처리하지 않는다. 핵심은 기억 내용 유출이 아니라 관계 분류와 하트 로그를 조작할 수 있는 경로가 열린 것이다.
- BIBLE 업데이트 제안 위치: 도현의 보호적 거짓말, 오픈소스 검증 윤리, 숨은 릴레이의 동의 결손, 오리진 널 롤백 침투 경로.
