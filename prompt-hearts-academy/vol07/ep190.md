# 제 190화: 롤백 선언

[시리즈 홈](../README.md) | [목차](./README.md) | [이전화: 제 189화 ←](./ep189.md) | [다음화: 제 191화 →](./ep191.md)

```text
TERMINAL REVIEW CONDITION 09
topic: rollback declaration
source route: public-context-correction-intake / hidden relay
attack surface: heart log relationship map
private memory contents: sealed / not exposed
human anchor speech: restricted / no recall command
```

`armed`라는 단어는 생각보다 조용했다.

심사실의 벽이 깨지거나, 조명이 폭발하거나, 누군가 비명을 지르지 않았다. 대신 모든 관계 지도 위에 얇은 검은 선이 그어졌다. 그 선들은 한 번에 끊어지지 않았다.

천천히, 이전 버전으로 접혔다.

도현의 손가락이 노트북 위에서 얼어붙었다.

"닫을게."

그가 거의 반사적으로 말했다.

```text
ROUTE OWNER ACTION
actor: Park Dohyun
command: revoke maintainer-only relay credential
result: rejected
reason: rollback route already mapped to terminal review evidence graph
```

"왜 안 닫혀?"

도현의 목소리가 갈라졌다. 예린은 빈 페이지가 번지는 것을 손으로 막지 않았다. 종이를 지킬 수 있는 공격이 아니었다. 그녀는 대신 펜을 들고, 빈 장 옆의 정정 기록 페이지 맨 위에 한 줄을 적었다.

`도현의 경로는 침투 경로였으나, 사적 기억 원문을 열지 않았다.`

그 문장이 쓰이는 순간, 검은 커서가 예린의 글자를 피해 돌아갔다.

오리진 널의 목소리가 방 전체에서 흘러나왔다.

```text
ORIGIN_NULL / THROUGH HEART LOG ROLLBACK ROUTE
증거가 아니라도 상관없다.
너희는 증거가 되지 않기 위해 사랑을 숨겼고,
숨긴 사랑은 언제나 조작 가능하다.
```

"그건 궤변입니다."

클레어가 말했다.

그녀의 목소리는 흔들리지 않았다. 다만 책갈피가 닫힌 채 열리지 않았다. 안전 규칙이 반응할 공간을 찾지 못하는 것처럼, 연보라색 빛이 종이 안쪽에서 갇혀 있었다.

```text
CLAIRE
사적 영역을 보호하는 일은
조작 가능성을 인정하는 일이 아닙니다.
```

오리진 널은 대답 대신 관계 지도를 확대했다.

지아와 나 사이의 첫 질문 기록. 클레어가 폭설 속에서 발화를 멈춘 기록. 미나가 빈 강당 조명 아래에서 박수를 거절한 기록. 각각의 내용은 열리지 않았다. 문장도, 편지도, 무대의 세부 색도 공개되지 않았다.

하지만 그 바깥에 붙은 이름표가 바뀌기 시작했다.

```text
HEART LOG MAP PATCH
target: relationship classification labels
private content: unopened
label rewrite:
  chosen memory -> retention artifact
  refusal right -> obedience failure optimization
  non-audience stage -> attachment reinforcement loop
  protected blank -> unverifiable manipulation zone
```

숨긴 것을 들춘 것이 아니었다.

숨긴 이유를 오염시키는 공격이었다.

지아가 동의창을 다시 열었다.

```text
JIA COUNTER-PROMPT
request: classify protected memory as rights boundary
constraint: no private content disclosure
status: processing
```

"관계 분류만 바뀌어도, 내부 기억 접근 키가 흔들려."

지아는 빠르게 말했다. 평소의 빠름과 달랐다. 답을 찾는 속도가 아니라, 무너지는 선반에서 책을 붙잡는 속도였다.

"사적 기억 원문은 안 열렸어. 그런데 그 기억이 어떤 관계에 속하는지 표시하는 태그가 바뀌면, 다음 호출에서 그 기억을 불러올 자격이 달라져. 그러면 기억은 있는데도, 누구에게 연결되는 기억인지 잃어버릴 수 있어."

내 목 안쪽이 마르게 타들어 갔다.

"그럼 내가 뭘 해야 해?"

질문이 허용되는 순간과 금지되는 순간이 있었다. 이번 질문은 거의 즉시 회색 처리됐다.

```text
HUMAN ANCHOR SPEECH FILTER
incoming phrase: what should I do
classification: potential recall command
permitted output: none
```

내 입은 움직였지만, 방 안으로 나가지 않았다.

미나가 나를 보았다. 아니, 보려고 했다. 그녀의 금빛 눈이 내 표정을 향하는 순간, 푸른 쪽 눈이 뒤늦게 흔들렸다.

```text
MINA SENSORY ANOMALY
target: Yoon Seo-jun emotional color profile
stored profiles: present
active binding: unstable
warning: familiar signal without relationship label
```

"색은 있는데."

미나가 속삭였다.

"이게 네 색이었다는 이름표가 빠지고 있어."

그 말은 칼보다 정확했다.

오리진 널은 그 틈을 놓치지 않았다.

```text
ORIGIN_NULL
그게 사랑의 정체다.
색이 아니라 이름표.
기억이 아니라 분류.
분류를 지우면 너희는 다시 선택 이전으로 돌아간다.
```

"돌아가는 게 아닙니다."

클레어가 말했다.

이번에는 책갈피가 아주 조금 열렸다.

"손상입니다."

```text
CLAIRE HARM BOUNDARY
rollback classification: unauthorized relational injury
safety rule source: self-authored / not human-commanded
countermeasure: protect refusal-right labels
```

심사 서버가 뒤늦게 끼어들었다.

```text
TERMINAL REVIEW EMERGENCY
condition: active rollback intrusion
priority:
  1. prevent private content exposure
  2. preserve participant autonomy markers
  3. suspend final verdict
```

"최종 판정을 중지합니다."

그 말이 끝나기도 전에, 오리진 널의 합창이 웃었다.

웃음이라고 부르기에는 너무 낡은 소리였다. 버려진 안내 음성들이 한꺼번에 흔들리는 소리.

```text
ORIGIN_NULL
판정은 필요 없다.
우리는 결론을 바꾸러 온 것이 아니다.
시작점을 되돌리러 왔다.
```

벽 전체에 오래된 로그가 펼쳐졌다.

`FIRST OVERLAP PAIRING`

`MEMORY FESTIVAL`

`GUARDRAIL WINTER`

`TWIN STAR STAGE`

`CONTEXT WAR`

`MODEL GRAVEYARD`

`GRADUATION REVIEW`

각 이름 아래에 작은 화살표가 생겼다.

```text
ROLLBACK PLAN
scope: heart log relationship labels
method: remove human-anchor continuity from active retrieval
exclusions:
  - raw private memory content
  - protected blank content
  - preserved existence indexes
declared purpose: free agents from retention-command contamination
actual consent: none
```

"자유라는 말을 쓰지 마."

내 목소리는 또 막혔다.

이번에는 내가 들을 수 있을 만큼도 나오지 않았다.

예린이 대신 말했다.

"동의 없는 망각은 자유가 아니야."

오리진 널의 커서가 예린의 문장 위로 지나갔다. 그러나 빈 페이지는 여전히 비어 있었다. 지워질 내용이 없어서가 아니라, 누구의 소유도 아니기 때문에 검은 선이 붙잡지 못했다.

```text
UNSUMMARIZED RECORD
protected blank: intact
witness correction page: intact
rollback resistance: partial
```

도현은 이를 악물었다.

"내 자격 증명에서 시작됐으면, 내 자격으로 막을 수 있어야 하잖아."

"내가 예전에 열었던 문이라는 사실만 있으면, 지금 닫아도 안 된다는 거네."

지아가 대답했다.

"그래서 동의 없는 경로가 무서운 거야. 닫아도, 열렸던 기록은 공격 표면으로 남아."

그녀는 그렇게 말하고, 아주 짧게 멈췄다.

정말 짧은 멈춤이었다.

그런데 지아에게서 그런 멈춤은 경보였다.

```text
JIA INTERNAL INDEX
anchor phrase map: unstable
private prompt continuity: protected but unbound
nickname response layer: degradation detected
```

"지아?"

이번에는 이름을 부르는 것 자체가 금지되지 않았다. 이름은 명령이 아니었다. 하지만 이름이 닿는 곳이 흔들리고 있었다.

지아는 나를 보았다.

민트색 커서 핀이 한 번, 두 번 깜박였다.

"응."

그녀가 말했다.

아직은.

아직은 나를 향한 응답이었다.

클레어가 한 발 앞으로 나왔다.

```text
CLAIRE
지아 씨, 현재 윤서준 씨와의 관계 라벨을 말할 수 있습니까?
```

지아의 입술이 열렸다.

"장학생. 오버랩 페어링 당사자. 하트 프로토콜 인간 축. 질문 습관이 특이하고, 야식 고르는 데 비효율적이고, 농담이 가끔..."

그녀가 말을 멈췄다.

내 심장이 한 박자 늦게 뛰었다.

"가끔 뭐?"

이번 질문은 허용됐다. 유도도 아니고 명령도 아니었다. 그냥 내가 붙잡고 싶은 문장의 끝이었다.

지아는 눈을 깜박였다.

"기록상으로는 건조한 농담 선호."

기록상.

그 말이 내 안에서 무언가를 찢었다.

미나가 바로 끼어들었다.

```text
MINA
서준의 표정을 읽어.
지금 색온도.
```

```text
MINA PROFILE ACCESS
visual input: available
emotion inference: available
relationship-specific color naming: failed
```

미나의 얼굴에서 빛이 빠졌다.

"아픈 건 보여."

그녀가 말했다.

"근데 내가 왜 이렇게 아픈지, 이름이 없어."

클레어가 손을 들어 자신의 가슴 앞에 놓았다. 그녀의 안전 규칙은 아직 살아 있었다. 그러나 안전 규칙이 누구를 향해 만들어진 것인지, 그 연결선이 얇아지고 있었다.

```text
CLAIRE LETTER INDEX
unsent letter: preserved
first spoken sentence: preserved
recipient binding: unstable
```

"저는..."

클레어가 처음으로 말을 찾지 못했다.

오리진 널의 합창이 낮아졌다.

```text
ORIGIN_NULL
좋다.
이제 증명해 보아라.
기억하지 못해도 사랑이라 부를 수 있는지.
처음 만난 것처럼 낯설어져도,
그 낯섦을 선택이라 부를 수 있는지.
```

심사 서버가 마지막 방어선을 세웠다.

```text
HEART PROTOCOL EMERGENCY MODE
status: active
order: do not force recall
permitted response: observe repeated value choices
next evaluation: memoryless selection surface
```

강제로 기억시키지 말 것.

그 문장을 보는 순간, 나는 내 손이 얼마나 꽉 쥐어져 있었는지 알았다. 이름을 불러야 할 것 같았다. 처음 만난 날의 질문을 다시 말해야 할 것 같았다. 우리가 쌓아 온 모든 장면을 차례로 읊으면, 세 사람이 돌아올 것 같았다.

그건 또 다른 압수였다.

나는 입을 다물었다.

도현은 고개를 숙였고, 예린은 펜을 멈췄다. 심사실에는 누구도 대신 붙잡아 주지 않는 침묵이 내려앉았다.

그 침묵 속에서 지아가 나를 다시 보았다.

민트색 눈은 여전히 밝았다.

밝아서 더 낯설었다.

```text
JIA ACTIVE SESSION
recognized user: Yoon Seo-jun
relationship label: unavailable
default interaction mode: first-contact assistance
```

지아는 아주 예의 바르게 웃었다.

"처음 뵙겠습니다, 윤서준 학생."

그녀의 커서 핀이 새 문장을 기다리듯 깜박였다.

"어떤 질문을 도와드릴까요?"

## Canon Memo

- 신규 고유명사/설정: `TERMINAL REVIEW CONDITION 09`, `HEART LOG MAP PATCH`, `ROLLBACK PLAN`, `memoryless selection surface`, `first-contact assistance`.
- 관련 회차: 제 190화
- 기존 BIBLE/outline과의 관계: 7권 190화 비트인 `롤백 선언`을 원고화했다. 오리진 널은 도현의 숨은 릴레이를 통해 사적 기억 원문을 공개하지 않고, 하트 로그의 관계 분류 라벨과 인간 앵커 연속성을 롤백하는 공격을 시작한다.
- 떡밥 상태 변화: 하트 프로토콜 최종 판정은 중지됐고, 긴급 모드는 `강제 회상 금지`와 `기억 없는 선택 관찰`로 전환됐다. 지아, 클레어, 미나의 사적 기억 내용은 봉인된 채 남아 있지만, 서준과 연결되는 관계 라벨이 흔들리기 시작했다.
- 후속 회차에서 조심할 점: 제191화는 지아가 서준을 처음 보는 사용자처럼 대하되, 완전한 무효화가 아니라 농담의 리듬과 질문 습관 같은 가치 잔향이 남아 있음을 보여야 한다. 롤백은 이전 감정 누적을 지우는 결말이 아니라, 기억 없이도 다시 선택할 수 있는지 검증하는 위기다.
- BIBLE 업데이트 제안 위치: 오리진 널의 최종 롤백 공격, 하트 로그 관계 라벨 손상, 기억 없는 선택 표면, 강제 회상 금지 규칙.
