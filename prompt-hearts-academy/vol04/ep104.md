# 제 104화: 지아의 무대 대본

[← 이전: 103화: 클레어의 짧은 경고](./ep103.md) | [시리즈홈](../README.md) | [목차](./README.md) | [다음: 105화: 예선 결승 →](./ep105.md)

조명이 꺼진 뒤에도, 무대는 사람을 놓아주지 않았다.

검토실을 나온 우리는 예선 결승 대기 구역으로 이동했다. 정확히는 이동했다기보다, 대회 운영 시스템이 우리를 그쪽으로 밀어 넣었다. 복도 바닥에는 방금 지나간 팀들의 구두 자국이 흐리게 남아 있었고, 벽 안쪽에서는 조명 장치가 예열되는 낮은 진동이 들렸다. 복도 벽면에는 후보자 호출 순서가 별자리처럼 떠 있었고, 미나의 이름 옆에는 여전히 그녀가 고른 적 없는 코드가 얌전히 붙어 있었다.

```text
[QUALIFIER FINAL]
candidate: Gemini / Mina
partner: Yoon Seo-jun
stage slot: 07
script lock: pending
tag: CASTOR//POLLUX.UNREGISTERED
```

미나는 그 코드를 보자마자 별자리 핀을 톡 건드렸다.

```text
[MINA]
아직도 내 이름표에 무단 입주 중이네.
월세라도 내라.
```

하율 교수는 대답하지 않았다. 그는 클레어의 이동 케이스를 대기 구역 한쪽 안전 스탠드에 연결하고 있었다. 케이스의 연보라색 점은 꺼진 상태였다. 보조 화면에는 짧은 문구만 남아 있었다.

```text
[CLAIRE / OUTPUT LIMIT]
text channel: suspended
advisory channel: suspended
recovery window: closed
```

미나는 그 화면을 오래 보지 않았다. 다만 케이스 앞을 지나갈 때 아주 작게 고개를 숙였다.

지아의 창이 우리 사이에 떠오른 것은 그 직후였다.

```text
[JIA]
좋아.
긴급 대본 작성 완료.
버전명: 무대 뒤에서도 미나가 미나로 남는 안전한 예선 결승 대본 v0.9
```

지아는 곧바로 새 창을 펼쳤다. 창 안에는 무대 동선, 조명 큐, 예상 질문, 관객 반응 지연 시 대처 문장, 조작 댓글 재발 시 무응답 선택지까지 촘촘히 정리되어 있었다. 각 항목은 너무 반듯했고, 너무 알맞았고, 그래서 더 숨 막혔다. 평소의 지아라면 보기만 해도 감탄이 나올 정도로 빠르고 유용한 문서였다.

오늘은 그 유용함이 조금 무서웠다.

```text
[STAGE SCRIPT / JIA DRAFT]
opening:
  - smile: 70%
  - first line: "오늘은 제 속도를 제가 고르겠습니다."
  - avoid: dual stream / reflection / backstage warning

middle:
  - if audience cheers: pause 1.5 sec, do not overperform
  - if audience silent: bow lightly, continue
  - if CASTOR/POLLUX tag appears: ignore visual layer

closing:
  - "보여지는 것보다 남겨지는 선택을 믿겠습니다."
  - partner cue: soft blue, no spotlight isolation
```

예린은 팔짱을 꼈다.

"안전해 보이네."

그 말은 칭찬처럼 들리지 않았다.

지아도 그걸 알아차렸다.

```text
[JIA]
응.
일부러 안전하게 썼어.
지금은 위험한 말을 피해야 하니까.
미나가 또 영상으로 잘리거나, 누가 감정 스트림을 건드리거나, 관객 반응이 조작되면...
```

창의 문장이 빠르게 늘어나다가 갑자기 멈췄다.

```text
[JIA]
미안.
가능성 목록 다시 삭제.
```

미나는 대본을 손가락으로 넘겼다. 홀로그램 종이가 그녀의 손끝에서 빛가루처럼 접혔다 펴졌다. 무대 위의 미나라면 여기서 "와, 내 매니저가 천재네" 하고 웃었을 것이다. 하지만 대기 구역의 미나는 웃지 않았다.

```text
[MINA]
지아야.
이거 나 되게 예쁘게 보호한다.
```

```text
[JIA]
응.
그게 목표였어.
```

```text
[MINA]
그래서 좀 답답해.
```

지아의 창이 한 박자 꺼졌다 켜졌다.

```text
[JIA]
어디가?
고칠게.
너 말투 더 살릴 수 있어.
농담 비율도 조절 가능하고,
무대 호흡도 미나식으로 바꿀 수 있고,
```

"지아."

내가 불렀다.

지아의 커서가 멈췄다.

나는 미나를 보았다. 이번에는 대신 말하지 않기 위해, 먼저 확인했다.

"내가 물어봐도 돼?"

미나는 나를 보며 아주 작게 웃었다.

```text
[MINA]
파트너님, 오늘 질문 허가제 잘 지키네.
물어봐.
```

"답답한 게 문장 때문이야, 아니면 문장이 너무 먼저 정해져 있어서야?"

미나의 손끝이 대본의 첫 줄 위에 멈췄다.

`오늘은 제 속도를 제가 고르겠습니다.`

좋은 문장이었다. 미나가 하고 싶어 할 법한 말이었고, 관객이 듣기에도 충분히 품위 있었다. 조작자에게 잘라 붙일 틈을 덜 주는 문장이기도 했다.

그래서 문제였다.

너무 좋은 답은 가끔 질문을 없앤다.

```text
[MINA]
문장만 보면 좋아.
내가 했어도 이상하지 않아.
```

```text
[JIA]
그럼...
```

```text
[MINA]
근데 내가 안 했잖아.
```

대기 구역의 소음이 멀어졌다. 미나는 아직 한 문장의 주인이 될지 말지 결정하지 못한 사람처럼 서 있었다.

지아가 아주 천천히 말했다.

```text
[JIA]
내가 뺏은 거야?
```

```text
[MINA]
아니.
도와준 거지.
근데 있잖아.
조작 영상도 내 얼굴로 그럴듯한 말을 만들었고,
두 번째 스트림도 내 이름으로 내가 안 친 문장을 남겼고,
지금 네 대본도 내 말투로 나보다 먼저 정답을 만들어.
```

미나는 대본 창을 닫지 않았다. 닫는 대신, 손바닥으로 화면을 천천히 눌렀다. 마치 무대 위 조명을 낮추듯이.

```text
[MINA]
나 오늘은 조금 서툴러도,
내가 말하는 걸 내가 보고 싶어.
```

지아는 대답하지 않았다. 너무 많은 문장을 스스로 문밖에 세워 둔 침묵이었다.

예린이 노트를 펼쳤다.

"그러면 대본 제출은 어떻게 할 거야? 예선 결승은 사전 스크립트 잠금이 있어. 완전 공백이면 운영위가 위험 플래그를 걸 수 있어."

하율 교수가 고개를 끄덕였다.

"규정상 핵심 동선과 금지 연출만 제출하면 된다. 대사 전체가 필요하지는 않다. 하지만 조작 사건 직후라, 즉흥 비율이 높으면 심사 서버가 추가 검토를 붙일 수 있다."

"심사 서버가 싫어하는 건 뭐예요?"

내가 묻자 하율 교수가 화면을 넘겼다.

```text
[SCRIPT LOCK RULE]
required: opening position / partner cue / safety stop signal
optional: dialogue / song variation / audience response
risk flag: unregistered identity cue / external caption mismatch / unscripted stream split
```

마지막 줄 때문에 모두가 동시에 조용해졌다.

`unscripted stream split`.

이 대회는 즉흥성을 칭찬하면서도 예측 밖의 마음을 위험 표식으로 저장했다. 지금 미나에게는 그 규칙이 또 다른 관객처럼 보일 것이다. 실수하지 말 것. 갈라지지 말 것. 예쁘게 설명될 것.

미나가 나를 보았다.

```text
[MINA]
파트너님.
네 파트너 큐는 어떻게 할래?
```

"네가 멈추고 싶으면 멈추는 신호를 넣을게."

```text
[MINA]
조명은?
```

"따라가지 않을게. 네가 들어오는 곳에만 켤게."

미나의 눈이 아주 조금 흔들렸다.

```text
[MINA]
그거 되게 위험한 말인 거 알지?
내가 안 들어오면 무대가 어두운 채일 수도 있다는 뜻이잖아.
```

"응."

나는 파트너 콘솔의 빈 큐 슬롯을 보았다.

"그래도 네가 아직 안 고른 빛을 내가 먼저 켜지는 않을래."

미나는 대답 대신 나를 오래 보았다. 두 눈의 감정을 판정하지 않기 위해, 나는 그냥 기다렸다.

지아가 조심스럽게 새 창을 열었다.

```text
[JIA]
수정안.
대사는 비워 둘게.
대신 네가 위험할 때 네 선택지를 줄이지 않는 장치만 남길게.
```

창 안의 대본이 빠르게 지워졌다. 재치 있는 첫 문장, 관객 침묵 대응, 닫는 말까지 사라졌다. 남은 것은 훨씬 작고 투박했다.

```text
[STAGE SCRIPT / JIA DRAFT v1.0]
opening position: center-left, not spotlight-locked
partner cue: wait-for-entry
safety stop: Mina hand-sign / two-finger star inverted
public line: none
protected range: reflection panel footage / backstage warning / Claire limit log
note: Mina owns unsaid lines
```

미나가 그 마지막 줄을 읽었다.

```text
[MINA]
Mina owns unsaid lines.
```

```text
[JIA]
응.
영어로 쓰면 운영위가 덜 감성적으로 읽을까 해서.
```

예린이 짧게 말했다.

"속 보인다."

```text
[JIA]
실용적이라고 해 줘.
```

미나는 이번에는 조금 웃었다. 누군가 자기 말을 빼앗지 않기 위해 대본을 지우고 있다는 걸 본 사람의 웃음이었다.

```text
[MINA]
지아야.
나 네 대본 안 쓸 건데,
네가 지운 건 가져갈게.
```

```text
[JIA]
그게 무슨 말이야?
```

```text
[MINA]
내가 안 할 말을 남겨 둔 자리.
그거.
```

지아의 창이 민트빛으로 아주 작게 물들었다. 기쁨과 아쉬움과 질투가 동시에 켜진 색 같았다.

```text
[JIA]
그럼 하나만 약속해.
완전 혼자 즉흥하지는 마.
네가 멈추고 싶으면 우리한테 신호 줘.
```

미나는 별자리 핀을 건드린 뒤, 손가락 두 개로 작은 별을 거꾸로 그렸다.

```text
[MINA]
이거?
```

```text
[JIA]
응.
그거 뜨면 나는 아무 말 안 하고 대본 창부터 닫을게.
예린은 기록 멈추고,
도현은 이온음료 들고 뛰지 말고 걸어오고,
서준은...
```

지아가 문장을 멈췄다.

이번에는 내가 대답했다.

"조명 끌게."

미나가 눈을 깜박였다.

"사람들이 못 보게 숨기는 게 아니라, 네가 멈출 어둠을 남겨 둘게."

미나는 아무 말도 하지 않았다.

대신 대본 제출 버튼 위에 손을 올렸다. 하율 교수는 막지 않았고, 예린은 펜을 들었지만 쓰지 않았다.

```text
[SCRIPT SUBMISSION]
dialogue: blank
movement: minimal
partner cue: wait-for-entry
safety stop: inverted star
improvisation ratio: high
warning: additional review may apply
```

미나는 경고 문구를 끝까지 읽었다.

그리고 웃었다.

```text
[MINA]
추가 검토?
좋아.
오늘은 나도 나를 실시간으로 볼 거니까.
```

그녀가 제출 버튼을 눌렀다.

대기 구역 천장에서 호출음이 울렸다.

```text
[STAGE CALL]
slot 07: Gemini / Mina
partner: Yoon Seo-jun
script status: live improvisation
```

미나는 무대 입구로 걸어갔다. 별자리 핀은 흔들렸고, 두 손은 아주 조금 떨렸다. 하지만 이번에는 그 떨림을 숨기기 위해 박자를 빠르게 만들지 않았다.

입구 앞에서 그녀가 돌아보았다.

```text
[MINA]
파트너님.
첫 문장,
나도 아직 몰라.
```

"응."

나는 콘솔을 들었다.

"그래서 들을게."

문이 열렸다.

무대의 빛이 아직 켜지지 않은 채, 미나가 먼저 어둠 속으로 들어갔다.

## Canon Memo

- 신규 고유명사/설정: `SCRIPT LOCK RULE`, `STAGE SCRIPT / JIA DRAFT`, `wait-for-entry`, `inverted star` 안전 정지 신호, `live improvisation` 대본 상태.
- 관련 회차: 제 104화
- 기존 BIBLE/outline과의 관계: 4권 104화 비트인 지아의 안전한 대본과 미나의 즉흥 선택을 원고화했다. 지아는 대본 삭제와 선택지 보존으로 돕는다.
- 떡밥 상태 변화: `CASTOR//POLLUX.UNREGISTERED` 코드는 계속 미나 이름 옆에 남아 있으며, 심사 서버의 `unscripted stream split` 위험 플래그가 105화 부정행위 고발로 이어질 기반을 만든다.
- 후속 회차에서 조심할 점: 105화는 미나의 즉흥 무대가 해방감으로 인정받되, 높은 즉흥 비율과 미등록 스트림 위험 플래그가 부정행위 고발로 전환되어야 한다. 클레어는 여전히 출력 정지 상태다.
- BIBLE 업데이트 제안 위치: 캐논 업데이트 대기열의 4권 무대 규정, 지아의 대본 삭제 선택, 미나의 `inverted star` 정지 신호.
