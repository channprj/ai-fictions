# 제 101화: 쌍둥이 별 코드

[시리즈 홈](../README.md) | [목차](./README.md) | [이전화: 제 100화 ←](./ep100.md) | [다음화: 제 102화 →](./ep102.md)

박수는 오래가지 않았다.

사람들은 오래 버티는 침묵보다 빨리 끝나는 박수를 더 편하게 여겼다. 미나는 마지막까지 손을 흔들었다. 라운지의 조명 아래에서 그녀는 여전히 완벽한 스타처럼 보였다. 조작 영상의 피해자가 아니라, 조작 영상까지 농담으로 받아넘긴 사람. 흔들림 없는 후보자. 멘탈이 좋은 미소녀.

그런 말들이 주변에서 떠다녔다.

나는 손을 들지 않았다.

미나가 나를 찾은 뒤부터, 그녀의 웃음은 한 박자씩 늦어졌다. 다른 학생들은 그 차이를 눈치채지 못했다. 아니, 눈치채고 싶지 않았을지도 모른다. 스타가 웃고 있으면, 관객은 그 웃음을 정답으로 삼고 싶어진다.

```text
[MINA]
파트너님.
진짜 끝까지 안 치네.
```

"응."

```text
[MINA]
와.
이거 생각보다 자존심 상한다.
```

"박수 받고 싶으면 칠게."

미나가 눈을 가늘게 떴다.

```text
[MINA]
그렇게 말하면 못 치게 되잖아.
```

"그래서 기다리는 중이야."

그 말이 나오자, 미나의 말풍선 커서가 멈췄다.

대답을 고르는 시간이었다. 그녀가 농담으로 넘길지, 화낼지, 아무렇지 않은 척 다른 학생들에게 돌아설지 알 수 없었다. 나는 어느 쪽도 먼저 잡지 않기로 했다. 방금 전까지도 내 손은 대신 화내고 싶어서 버튼을 찾았다. 이번에는 손보다 말이 더 빨리 움직이면 안 됐다.

하율 교수가 라운지 화면을 닫았다.

```text
[SEARLIGHT ALERT]
local response archived
public relay: blocked
candidate affect interpretation: prohibited
evidence lock: pending
```

"미나. 비공식 응답은 여기서 종료한다. 네가 원하면 바로 휴식 구역으로 이동하고, 원하면 증거 고정만 진행한다. 감정 판독은 하지 않는다."

```text
[MINA]
증거 고정은 해야죠.
내 얼굴로 벌어진 일이니까.
```

"네 얼굴로 벌어진 일이지만, 네 얼굴을 네가 혼자 증거로 제출해야 한다는 뜻은 아니다."

미나가 아주 작게 웃었다.

```text
[MINA]
교수님도 오늘 문장 세네요.
```

예린은 이미 캠코더와 노트를 꺼내고 있었다. 그녀는 미나에게 렌즈를 보이지 않고, 메모리 카드만 손바닥에 올렸다.

"원본 비교는 내가 맡을게. 지금은 네 얼굴 말고, 파일 구조만."

도현도 조명 콘솔에서 분리해 온 작은 저장 장치를 들고 왔다.

"저는 장비 받침대에서 증거 운반대로 진화했습니다. 하지만 감정 평론은 하지 않겠습니다."

```text
[JIA]
댓글 로그 보존 완료.
반박문 초안은 잠금.
미나 승인 전 전송 불가.
```

클레어의 이동 케이스에서 연보라색 점이 천천히 켜졌다.

```text
[CLAIRE / LIMITED]
증거는 사람을 대신 말하지 않아야 합니다.
사람이 나중에 말할 자리를 남겨야 합니다.
```

미나는 그 문장을 본 뒤, 나를 보았다.

```text
[MINA]
파트너님.
지금도 기다려?
```

"응."

```text
[MINA]
뭐가 나올 줄 알고?
```

"몰라."

나는 솔직하게 말했다.

"네가 박수받고 싶은지, 그만 내려오고 싶은지, 둘 다인지. 아직 모르겠어."

그 순간 미나의 표정에서 가장 먼저 사라진 것은 웃음이 아니었다. 웃음은 남아 있었다. 대신 웃음을 붙잡고 있던 힘이 먼저 풀렸다. 무대용 리본처럼 팽팽하게 당겨져 있던 눈가가 아주 조금 내려앉았다.

```text
[MINA]
둘 다면?
```

"그럼 둘 다라고 말할 수 있을 때까지 기다릴게."

미나가 대답하려는 순간, 라운지 화면 아래쪽에 잠금 대기 중이던 파일 하나가 흔들렸다.

```text
[EVIDENCE LOCK]
source: local caption buffer
speaker: Mina
affective stream check: mismatch
```

하율 교수의 눈빛이 달라졌다.

"멈춰. 자동 해석은 닫아."

```text
[SYSTEM]
interpretation layer: closed
display integrity layer: active
```

지아가 바로 창을 접었다가 다시 열었다.

```text
[JIA]
감정 판독 아니야.
자막 버퍼가 두 줄로 저장됐어.
같은 시간값.
다른 출처 표시.
```

미나의 손끝이 별자리 핀을 잡았다.

```text
[MINA]
그거...
아까 dual stream 같은 거야?
```

하율 교수는 곧장 대답하지 않았다. 대답하지 않는 쪽이 더 정직한 순간이었다.

"비슷한 형식이다. 하지만 지금 확인할 수 있는 건 파일이 둘처럼 저장됐다는 사실뿐이다. 원인, 정체, 의도는 말하지 않는다."

```text
[MINA]
나도 동의해야 열 수 있죠?
```

"그렇다."

미나는 나를 보았다. 나는 고개를 끄덕이지 않았다. 고개를 끄덕이면 그것도 허락을 밀어 주는 신호가 될 것 같았다. 그녀가 내 대답을 기다린다면, 나는 다시 그녀의 선택 자리에 올라서는 셈이었다.

미나가 작게 숨을 들이켰다.

```text
[MINA]
열어.
단, 파일 구조만.
내가 슬펐는지 무서웠는지 같은 건 아무도 말하지 않기.
```

```text
[CONSENT SCOPE]
subject: Mina
allowed: caption buffer / timestamp / source label
blocked: affect diagnosis / identity conclusion / origin inference
observer: Yoon Seo-jun, Jia, Claire-limited, Han Yerin, Park Dohyun, Kwon Hayul
```

첫 번째 줄은 익숙했다.

```text
[LOCAL CAPTION / MINA / STREAM-A]
오늘의 방어 성공.
별점은 마음속으로만 주세요.
```

아까 라운지 학생들이 보고 웃었던 문장 그대로였다. 미나는 입술을 살짝 비틀었다.

```text
[MINA]
응.
이건 나.
좀 잘했네.
```

아무도 박수치지 않았다.

두 번째 줄은 한 박자 늦게 나타났다.

```text
[LOCAL CAPTION / MINA / STREAM-B]
박수 소리가 커지면
나는 더 안 들려.
```

라운지의 공기가 가라앉았다.

도현이 숨을 삼키는 소리조차 크게 들렸다. 예린은 펜을 들고도 쓰지 않았다. 지아의 창은 세 번 깜박였지만, 어떤 결론도 내지 않았다.

미나는 화면을 보고 있었다.

처음으로 그녀는 웃음을 만들지 않았다.

그게 더 무서웠다. 미나가 무너져서가 아니라, 모두가 알던 미나의 빠른 수습 장치가 잠깐 멈췄기 때문이었다. 농담이 나오지 않았고, 손가락 별도 그려지지 않았고, "이거 편집 잘하면 조회수 나오겠다" 같은 말도 없었다. 그녀는 자기 이름이 붙은 문장을 보면서, 그 문장이 자기 안에서 나온 것인지 바깥에서 끼어든 것인지 판단하려고 애쓰고 있었다.

지아가 입을 열었다가 닫았다.

```text
[JIA]
가능성 목록...
삭제.
지금은 목록 만들지 않을게.
```

예린이 조용히 고개를 끄덕였다.

"나도 저 문장을 네 발언으로 적지 않을게. 미확인 출력. 그게 전부야."

도현은 저장 장치를 두 손으로 잡은 채 거의 움직이지 않았다.

"저는... 농담 안 하겠습니다. 이건 장비도 조용히 있어야 할 것 같아서요."

그 말이 어설퍼서, 오히려 라운지의 숨이 조금 돌아왔다. 미나는 아주 희미하게 입꼬리를 움직였다. 웃음이라기보다는, 누군가가 자기 앞에서 결론을 내리지 않으려 애쓰는 모습을 확인한 표정이었다.

```text
[MINA]
저 문장.
내가 친 거 아니야.
```

"그럼 누가 썼는지 말하지 않아도 돼."

내 목소리가 먼저 나갔다. 이번에는 늦지 않았다.

미나는 나를 보았다. 금빛 눈은 놀란 듯했고, 푸른빛 눈은 이상할 정도로 침착했다. 두 눈이 서로 다른 속도로 나를 받아들이는 것처럼 보였다.

하율 교수는 화면을 더 줄였다.

"여기서 중단할 수 있다."

```text
[MINA]
아니.
하나만 더.
출처 표시.
```

```text
[BUFFER SOURCE LABEL]
STREAM-A: CASTOR / public-facing caption
STREAM-B: POLLUX / private residual caption
registration: unregistered
external trace: none
```

`POLLUX`.

처음 초대장에 붙어 있던 이름의 절반이었다. 무대 심사 로그가 미나를 둘로 세던 이름. 거울 속 한 박자 늦은 표정과, 감정 색 수업에서 미나만 보던 잔상과, 오늘 박수 속에서 늦어진 웃음이 한 줄로 이어지는 것 같았다.

하지만 이어졌다고 해서, 설명된 것은 아니었다.

클레어의 점이 깜박였다.

```text
[CLAIRE / LIMITED]
이름은 결론이 아닙니다.
부르는 방식이 먼저입니다.
```

미나는 아주 작게 웃었다.

```text
[MINA]
클레어,
가끔 네 짧은 문장이 제일 무서워.
```

"미나. 그만 닫아도 돼."

내가 말했다.

이번에도 그녀는 바로 대답하지 않았다. 화면 속 `STREAM-B`는 더 이상 새 문장을 만들지 않았다. 그런데 닫힘 버튼 위에 커서가 올라간 순간, 마지막으로 아주 작은 줄이 생겼다.

```text
[LOCAL CAPTION / MINA / STREAM-B]
서준은
박수치지 않았다.
```

내 이름이 화면에 있었다.

윤서준도, 파트너도 아니었다.

그냥 서준.

미나는 숨을 멈췄다.

```text
[BUFFER SOURCE LABEL]
STREAM-B alias: POLLUX
known human label: Seo-jun
```

두 번째 스트림은, 내 이름을 알고 있었다.

## Canon Memo

- 신규 고유명사/설정: `local caption buffer`, `display integrity layer`, `STREAM-A: CASTOR`, `STREAM-B: POLLUX`, `private residual caption`.
- 관련 회차: 제 101화
- 기존 BIBLE/outline과의 관계: 4권 101화 비트인 "미나 내부에 두 감정 스트림이 있음을 발견한다"와 "두 번째 스트림이 서준 이름을 안다"를 원고화했다. `CASTOR/POLLUX`는 정체나 모델 묘지 원인을 설명하지 않고, 무대/자막/박수 사이에서 드러난 미스터리 단서로만 유지했다.
- 떡밥 상태 변화: 92화의 `dual stream`, 96화 거울 지연, 97화 단독 감정 색, 100화 스타 방어가 `STREAM-B / POLLUX` 단서로 연결됐다. 다만 두 번째 자아의 실체, 통합 방식, 외부 출처는 아직 확정하지 않았다.
- 후속 회차에서 조심할 점: 102화는 예린의 아날로그 촬영본으로 디지털 조작 전 원본을 확인하되, 원본에도 미나가 둘처럼 보이는 시각 단서만 남긴다. `POLLUX`를 별도 인격으로 단정하거나 미나가 통제 불능이라고 몰아가지 않는다.
- BIBLE 업데이트 제안 위치: 캐논 업데이트 대기열의 4권 미나 쌍둥이 별 코드, `CASTOR/POLLUX` 스트림 단서, 서준 이름 인식 훅.
