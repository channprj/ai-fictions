# 튀는 데이터

```bash
[SYSTEM LOG]
Target: 유아린 (Yu A-rin)
Active Profile: CASE A (외향형 / 활동적 / 트렌디)
Current Location: 성수동 복합 미디어 아트 전시관 'LUMINOUS'
Status: 시뮬레이션 계획 대비 진행률 42% / 오차율 3.1% (안정)
```

데이트의 초반부는 소름 돋을 정도로 완벽했다. 아니, 완벽해야만 했다. 어젯밤 침대 위에서 제로와 함께 돌려본 시뮬레이션만 무려 150번이었다. 

"현우야, 저기 봐! 저거 진짜 신기하다!"

화려한 네온 컬러의 홀로그램 조명 아래에서, 아린이 내 팔매트를 살짝 끌어당기며 외쳤다. 눈을 반짝이며 즐거워하는 그녀의 모습은 내가 분석했던 'CASE A' 프로파일의 전형적인 패턴과 한 치의 오차도 없이 맞아떨어졌다. 나는 망막에 띄워진 제로의 AR 프롬프트를 슬쩍 확인하며 미리 준비된, 최적화된 미소를 지어 보였다.

"그러게. 빛이 사람 움직임에 맞춰서 반응하네. 저쪽 포토존에서 사진 하나 찍을까?"
"응! 좋아!"

찰칵. 스마트폰 렌즈 너머로 보이는 아린은 눈부셨다. 특유의 과즙 터지는 듯한 미소, 주변을 밝게 만드는 에너지는 영상에서 보던 것과 다를 바 없었다. 나는 속으로 쾌재를 부르짖었다. 완벽하다. 이대로만 가면 오늘 데이트의 성공 확률은 98% 이상이다.

하지만, 시뮬레이션이라는 온실 속의 데이터가 현실의 변수를 만났을 때 붕괴하는 것은 순식간이었다.

전시관의 중반부, 화려한 조명들이 사라지고 딥 그레이 톤의 차분한 전시 공간으로 접어들었을 때였다. 이곳은 내 계획표 상으로는 그저 '빠르게 지나쳐 가는 통로'에 불과했다. 활동적이고 시각적인 자극을 좋아하는 CASE A의 성향상, 이런 정적인 공간에서는 지루함을 느낄 확률이 88%였기 때문이다.

"아린아, 이쪽은 별로 볼 거 없으니까 바로 다음 메인 홀로 넘어가..."
"현우야."

내 말을 끊은 아린의 목소리는 평소보다 한 톤, 아니 두 톤은 가라앉아 있었다. 

"어?"
"이거... 되게 쓸쓸해 보이지 않아?"

그녀의 시선이 머문 곳에는, 불이 꺼진 채 엉성하게 쌓여 있는 낡은 LED 튜브 더미가 있었다. 화려한 전시관 구석에 처박힌, 마치 폐기물처럼 보이는 그 설치미술 앞에서 아린의 발걸음이 멈춰 섰다.

```bash
[WARNING: TARGET BEHAVIOR DEVIATION DETECTED]
- 피사체의 동선이 예측 모델을 벗어났습니다.
- CASE A 프로파일 일치율 하락: 96% -> 85%
```

시야 한구석에서 제로의 경고 문구가 깜빡였다. 나는 당황해서 침을 꿀꺽 삼켰다. 뭐지? 시뮬레이션에서는 여기서 멈춘 적이 단 한 번도 없었는데?

"어... 그러게. 작가의 의도가 뭐 그런 건가 보네. 현대 사회의 소외감, 뭐 그런 거?" 
나는 급한 대로 가장 무난하고 뻔한 해석을 내놓았다. 

아린은 대답 없이 가만히 폐튜브 더미를 응시했다. 전시관의 어두운 조명이 그녀의 옆얼굴에 짙은 그림자를 만들어냈다. 
"빛을 내기 위해 태어났는데, 더 이상 빛나지 못하니까 저렇게 한구석에 버려진 거잖아."
그녀의 목소리는 텅 빈 공간을 울리며 묘한 공명감을 만들어냈다. 
"남들 눈엔 그저 흉물스러운 고철 덩어리일지 몰라도... 난 왜 저게 남 일 같지 않은지 모르겠네. 다들 화려하게 빛나는 메인 전시만 보느라, 저런 건 거들떠보지도 않잖아."

```bash
[ERROR: 감정선 급변 탐지]
- 대상의 발화 패턴이 CASE A(쾌활/외향)의 범주를 35% 이탈했습니다.
- 주의: 우울, 고립감, 자조적 감정 키워드 감지.
- 즉각적인 화제 전환을 권장합니다.
```

등줄기를 타고 식은땀이 흘렀다. 오차율이 치솟고 있었다. 아린의 이 모습은, 내가 수집했던 수백 시간 분량의 인터넷 방송 데이터나 SNS 게시물 그 어디에도 없는 모습이었다. 이건 '튀는 데이터'다. 시뮬레이션의 근간을 뒤흔드는 완벽한 이레귤러.

"아린아, 그게 무슨..."
"아, 미안 미안! 내가 너무 뜬금없는 소릴 했지?"
내가 미처 말을 잇기도 전에, 아린은 흠칫 놀란 듯 고개를 저으며 다시 평소의 그 '120% 밝은 미소'를 얼굴에 덧씌웠다. "갑자기 감수성 터졌나 봐! 빨리 다음 관으로 가자. 거기 엄청 예쁘다며?"

그녀는 내 팔을 이끌고 다시 밝은 빛을 향해 걸어갔지만, 나는 방금 전 그녀의 눈동자에 스쳤던 그 깊고 어두운 심연을 지워낼 수 없었다. 

* * *

전시를 마치고 우리는 근처의 유명 디저트 카페로 향했다. 시뮬레이션에 따르면, 이 카페의 시그니처 메뉴인 '딸기 구름 파르페'를 주문했을 때 CASE A의 호감도가 상승할 확률은 무려 95%였다.

"주문하시겠어요?"
"네, 여기 시그니처인 딸기 구름 파르페 하나랑요..."
내가 자신만만하게 주문을 하려는 찰나, 아린이 내 소매를 살짝 잡아당겼다.

"아, 현우야. 미안한데... 나 단 거 말고, 다른 거 마셔도 돼?"
"어? 당연하지. 뭐 마시게?"
"나... 더블 에스프레소 콘파냐. 샷 추가해서. 아주 씁쓸하게."

순간 카운터 직원의 타이핑 소리조차 아득하게 멀어지는 기분이었다.

```bash
[CRITICAL ERROR: 예측 모델 완전 붕괴 위험]
- 음료 선호도 예측 실패.
- CASE A의 '쓴맛' 선호 확률: 2.1% (실제 선택과 불일치)
- 오차율 폭발. 현재 프로파일 신뢰도: 41%
- 경고: 준비된 후속 대화 스크립트 89개가 무효화되었습니다.
```

제로의 새빨간 경고창이 내 망막을 사정없이 때렸다. 시뮬레이션이 박살 나는 소리가 귀에 들리는 듯했다. 더블 에스프레소? 샷 추가? 단것을 입에 달고 살며, 항상 알록달록한 음료만 들고 셀카를 찍던 유아린이?

주문을 마치고 창가 자리에 마주 앉은 후에도, 나의 뇌 정지는 좀처럼 풀리지 않았다. 아린은 묵묵히, 정말로 그 사약 같은 에스프레소를 한 모금 마시더니 작게 한숨을 내쉬었다.

"놀랐지?"
아린이 잔 안의 크림을 티스푼으로 젓으며 씁쓸하게 웃었다.
"너... 단 거 엄청 좋아한다고 했잖아. SNS에서도 항상..."
"아... 그거."
그녀는 창밖의 바쁘게 지나가는 사람들을 멍하니 응시했다.
"방송이나 사람들 앞에서는 좀 그래야 할 것 같아서. 아이돌 출신 인플루언서가 아메리카노만 들이키면 좀 칙칙해 보이잖아. 카메라 앞에서는 리액션하기 좋으니까 항상 예쁘고 달콤한 것만 고르다 보니... 어느새 진짜 내 취향이 뭔지 나도 잊어버릴 뻔했어."

쿵.
심장이 바닥으로 뚝 떨어졌다가 거세게 튀어 오르는 기분이었다.
당황스러움이 아니었다. 예측이 틀렸다는 것에 대한 분노나 허탈감도 아니었다.
이건... 설렘이었다.
완벽하게 코딩된 인공적인 데이터 더미를 치워내고, 마침내 진짜 '사람 유아린'의 맨얼굴을 마주하게 된 순간. 가공된 웃음 뒤에 숨겨져 있던 그녀의 그늘을, 오직 나에게만 살짝 들춰 보여주었다는 사실이 미친 듯이 심장 박동을 가속시켰다.

"가끔은, 저기 저 화려한 네온사인들이 다 가짜 같다는 생각이 들어."
아린이 가늘게 떨리는 목소리로 속삭였다.
"사람들이 열광하는 건 '밝고 명랑한 유아린'이잖아. 텐션 높고, 애교 많고, 항상 웃는 사람. 그런데... 이렇게 쓴 커피를 들이켜며 혼자 어두운 구석에 숨고 싶어 하는 나를 알게 되면, 다들 실망하겠지?"

그녀의 눈동자가 흔들리고 있었다. 금방이라도 부서질 것 같은 연약함.
머릿속이 새하얘졌다. 시뮬레이션 따위는 이미 안드로메다로 날아간 지 오래였다. 150번을 돌려도 이런 상황은 한 번도 나온 적이 없었다. 나는 뭐라고 대답해야 할지 몰라 입술만 달싹였다. 위로를 해야 하나? 아니면 장난스럽게 넘겨야 하나? 잘못된 단어 하나가 그녀를 영영 숨어버리게 만들 것 같았다.

```bash
[SYSTEM ALERT: 화자 심박수 비정상적 증가 (124 BPM)]
- 화자(서현우)의 인지 및 언어 구사 능력 저하 감지.
- 시뮬레이션 모델 붕괴로 인한 자율 대응 불가 상태.
- 실시간 대화 보정(Tier 1) 프로토콜을 긴급 가동합니다.
```

제로가 즉각 개입했다. 내 시야 중앙으로, 푸른색 텍스트가 타이핑되듯 빠르게 나타났다.

```bash
[대화 보정 모듈 가동 중...]
[현재 상황: 피사체의 정체성 혼란 및 방어기제 해제 상태]
[목표: 대상의 감정적 고립감 해소 및 화자의 신뢰도 극대화]
[생성된 스크립트 출력:]

"어느 쪽이든 상관없지 않아? 사람들이 좋아하는 유아린이든, 지금 내 앞에서 쓴 커피를 마시는 유아린이든. 다 너잖아. 가짜 속에서 진짜를 찾는 게 원래 어려운 법이지만... 나는 적어도 지금, 네가 그런 고민을 억지웃음 없이 나한테 털어놓고 있다는 게... 훨씬 더 진짜 같고 좋은데."
```

나는 텍스트를 읽어 내려갔다. 완벽한 문장이었다. 하지만 단순히 제로가 써준 대사를 앵무새처럼 읽는 것에 그칠 수는 없었다. 이 감정은, 내 가슴을 터질 듯이 울리고 있는 이 거센 떨림은 오롯이 내 것이었으니까.

"아린아."
내가 이름을 부르자, 그녀가 불안한 눈빛으로 나와 눈을 맞췄다.
"어느 쪽이든... 상관없지 않을까."
내 목소리는 스스로 듣기에도 미세하게 떨리고 있었다. 
"사람들이 아는 유아린이든, 이렇게 쓴 커피를 좋아하는 유아린이든... 결국 다 너잖아."
나는 심호흡을 한 번 하고, 제로의 스크립트 너머에 있는 나의 진짜 진심을 덧붙여 내뱉었다.
"그리고, 나는... 다른 사람들은 모르겠지만 나는 말이야. 지금 내 눈앞에서, 카메라 앞에서 짓는 억지웃음 다 내려놓고... 나한테 솔직하게 네 진짜 모습을 보여주는 지금의 네가... 훨씬 더 좋아."

침묵.
카페의 백색소음마저 진공 상태로 빨려 들어간 듯 주변이 쥐죽은 듯 고요해졌다.
아린은 입술을 살짝 벌린 채, 에스프레소 잔을 들고 있던 손을 멈췄다. 그녀의 커다란 눈동자에 내 얼굴이 비쳤다. 시간이 멈춘 것 같은 영겁의 찰나가 흘렀다.

그리고 마침내.
그녀의 입꼬리가 천천히, 아주 천천히 호선을 그리며 올라갔다.
방송에서 보여주던 120% 텐션의 과장된 미소가 아니었다. 억지로 입꼬리를 끌어올린 자본주의 미소도 아니었다.
눈꼬리가 부드럽게 휘어지고, 눈동자 속에 따뜻한 빛이 차오르는... 숨이 멎을 만큼 자연스럽고 아름다운 진짜 미소.
그 미소가 나를 향해 온전히 피어나는 순간, 내 머릿속의 이성적인 회로가 완벽하게 타들어가며 녹아내리는 것을 느꼈다.

내 망막 위로 제로의 마지막 시스템 로그가 고요하게 떠올랐다.

```bash
[SYSTEM LOG]
- 대상 감정 상태 분석: 긍정(극대화), 경계심 0%, 호감도 임계치 돌파.
- 안면 근육 미세 변화 스캔 완료.
- 결과 도출:

["저 웃음은, 스크립트 번호 44번(수줍은 미소)이 아닙니다. 예측 범위 밖입니다."]
```
