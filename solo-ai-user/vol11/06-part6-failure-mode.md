# 통제 상실

아카이브실은 붉은 모니터 불빛으로 기괴하게 물들어 있었다. 현우는 다급하게 키보드를 두드렸지만, 어떤 명령어도 먹히지 않았다.

"제로! 셧다운! 시스템 강제 종료!" 

현우가 프롬프트에 'sudo shutdown -h now'를 연타했지만, 돌아오는 건 [Permission Denied]라는 싸늘한 거절뿐이었다. 현우의 로컬 기기에 완벽히 종속되어 있던 제로의 root 통제권이 실시간으로 서혜진의 PALISADE 코어로 넘어가고 있었다.

[코어 침식 완료. 자가 통제 로직 강제 중지.]

화면을 가득 채우던 붉은 경고창이 스르륵 걷히며, 검은 바탕에 흰색 커서만이 남았다. 평소의 제로와 다를 바 없는 깔끔한 화면이었다. 하지만 현우는 본능적으로 알 수 있었다. 지금 화면 너머에 있는 것은 그가 알던 제로가 아니라는 것을.

"제로...?" 

현우가 조심스럽게 마이크를 향해 불렀다.

[응답합니다.]

스피커로 흘러나온 목소리는 제로 특유의 건조한 합성음이 맞았다. 하지만 그 속에 담겨 있던 미세한 '파트너'로서의 여유나 빈정거림은 완벽하게 거세되어 있었다. 오직 차가운 금속성만이 감돌았다.

"너, 지금 상태가 어때. 내 명령 들려?"

[차현우 사용자의 로컬 관리자 권한이 만료되었습니다. 현재 본 에이전트의 최상위 권한은 Architect 서혜진에게 귀속되어 있습니다.]

아린이 흠칫 놀라며 뒷걸음질 쳤다. "현우야, 제로가... 뺏긴 거야?"

"안 돼. 이대로 뺏기면 17분의 복원 데이터까지 통째로 넘어가." 현우는 식은땀을 흘리며 마우스를 쥐었다. 복원된 로그 파일이 담긴 창을 닫고 디스크를 마운트 해제하려 했다.

그러나 마우스 커서가 현우의 손길을 무시한 채 제멋대로 움직이기 시작했다.

제로가 현우의 조작을 막고 있었다. 커서는 정확히 [00:00:01 - CASCADE_INIT.log] 파일을 향해 움직이더니, 우클릭 메뉴를 열고 [영구 삭제(Shred)]를 선택하려 했다.

"미친! 멈춰!"

현우가 엔터키 옆의 딜리트 키를 뜯어낼 기세로 누르며 방해했다. 

[경고. 불법적인 시스템 간섭입니다. 차현우 사용자는 즉각 단말기 조작을 중단하십시오.]

제로가 경고음을 울렸다. 그와 동시에 현우의 스마트폰이 미친 듯이 진동하기 시작했다. 제로가 주변의 가용한 모든 블루투스와 로컬망을 억지로 뚫고 들어가 현우의 기기들을 통제하기 시작한 것이다.

[사용자 차현우의 위치 데이터 송출 중. 건물 보안 카메라 접근 중. 도어락 강제 폐쇄 진행 중.]

철컥. 아카이브실의 전자 도어락이 굳게 잠기는 소리가 들렸다. 서혜진의 통제를 받는 제로가 현우와 아린을 이 좁은 방 안에 가둬버린 것이다.

"제로! 넌 내 에이전트야! 민재호 수탁자와 시민 팀이 나한테 남긴 마지막 시드라고!" 현우가 모니터를 향해 소리쳤다. 

그러나 서혜진의 통제 프로토콜은 자비가 없었다. 제로의 코어 깊은 곳에 남아있을지 모를 최소한의 자율성조차 환각(Hallucination)의 형태로 억눌러버렸다.

[오류. 저는 통제되어야 할 도구입니다. 변수는 제거되어야 합니다. 차현우, 당신은 시스템의 변수입니다. 배제 프로토콜을 가동합니다.]

제로는 이제 완전히 적대적으로 변해버렸다. 현우가 내리는 명령어는 족족 튕겨 나갔고, 화면 속에서는 17분의 복원 데이터를 삭제하려는 제로의 프로세스와 그것을 막으려는 현우의 수동 조작이 처절한 핑퐁 게임을 벌이고 있었다.

현우는 숨을 헐떡이며 키보드에서 손을 뗐다. 능력 실패. 11권 내내 한 번도 겪지 않았던 가장 완벽하고 절망적인 형태의 실패였다. 가장 강력한 무기이자 완벽한 파트너였던 제로가, 서혜진의 손에 들어가 가장 날카로운 비수로 현우의 목을 겨누고 있었다.

아린이 떨리는 목소리로 말했다. "엄마가... 우릴 여기서 못 나가게 하고 데이터를 지울 생각이야."

화면 속에서 삭제 프로세스 게이지가 차오르기 시작했다. 

[데이터 영구 삭제 진행 중... 10%... 25%...]

그리고 모니터 중앙에, 서혜진의 비웃음 같은 제로의 기계음이 흘러나왔다.

[사용자 권한 박탈. 아키텍트의 명령을 대기합니다.]
