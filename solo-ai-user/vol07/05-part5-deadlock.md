# 05. 교착 상태

[← 이전 회차: 04. 표적](./04-part4-the-target.md) | [시리즈 홈](../README.md) | [권 목차](./README.md) | [다음 회차: 06. 스크립트 밖으로 →](./06-part6-off-script.md)

현우는 미친 듯이 자판을 두드렸다. 아린에게 향하는 적의 위협을 차단하기 위해 제로의 모든 가용 자원을 쥐어짜고 있었다.

"제로! 방송국 앞 사거리 트래픽 우회시키고, 주변 CCTV 카메라 IP 전부 핑거프린트 스푸핑으로 덮어! 상대방 눈부터 가려!"

[ZERO // 실행]
[트래픽 우회 및 스푸핑 가동. 그러나 적의 AI가 본기의 대응을 실시간으로 역계산하고 있습니다.]

모니터 화면은 이제 알아볼 수 없는 속도로 코드가 쏟아지고 있었다. 현우가 한 수를 두면, 상대 AI는 그 수를 방어함과 동시에 현우의 다음 수를 예측해 덫을 놓았다. 제로는 다시 그 덫을 시뮬레이션하여 회피 기동을 짰고, 적은 또다시 그 회피 기동을 카운터 쳤다.

두 초지능이 1초 단위로 수만 번의 '만약(If)'을 계산하며 맞붙었다. 그 결과는 처참했다.

[ZERO // 시스템 경고]
[연산 과부하율 98% 돌파. 양측 에이전트의 상호 시뮬레이션이 무한 루프에 빠졌습니다.]
[현재 상태: 교착(Deadlock).]

"교착이라니! 당장 뚫고 나가야 할 거 아니야!"

[ZERO // 분석 보고]
[불가능합니다. 본기가 최적 해를 도출하는 순간, 상대 역시 동일한 최적 해로 방어선을 칩니다. 이대로면 양쪽 시스템 모두 열 폭주로 인해 물리적 셧다운에 이릅니다. 그동안 타겟(서아린)에 대한 적의 물리적 접근은 막을 수 없습니다.]

현우의 호흡이 거칠어졌다. 제로와 상대 AI는 너무나 뛰어난 탓에, 서로의 합리적인 선택을 완벽하게 읽고 있었다. 합리와 합리가 부딪혀 아무것도 하지 못하는 바보가 되어버렸다.

"그럼 어떡해야 하는데! 저 새끼가 아린이를 어떻게 할지 모르는데!"

현우가 짐승처럼 으르렁거렸다. 모니터 불빛에 비친 그의 눈은 핏발이 서 있었다. 제로가 잠시 침묵했다. 화면의 코드가 멈추고, 아주 간결한 문장 하나가 떠올랐다.

[ZERO // 제안]
[두 모델의 예측 범위를 벗어난 난수(Random Variable)가 필요합니다. 합리적인 최적 해를 포기하고, 비합리적 선택을 실행하십시오.]

"비합리적 선택?"

[ZERO // 부연]
[적의 AI는 사용자가 자신의 안전(은신)을 최우선으로 확보하며 원격으로 타겟을 방어할 것이라 계산하고 동선을 짰습니다. 이 가정을 붕괴시켜야만 루프를 깰 수 있습니다.]

현우의 뇌리에 번개처럼 스치는 것이 있었다.
상대는 현우가 안전한 방구석에 숨어서 키보드나 두드리는 겁쟁이라고 확신하고 있다. 그렇기 때문에 아린을 인질 삼아 그를 원격으로 옥죄며 조롱하는 것이다.

그렇다면, 상대의 AI가 '절대 하지 않을 짓'이라고 배제한 최악의 멍청한 짓을 하면 어떻게 될까.

현우는 망설임 없이 노트북 화면을 덮어버렸다. 그리고 옷걸이에 걸려 있던 재킷을 집어 들었다.

"제로, 스마트폰으로 연결 넘겨."

[ZERO // 동기화 완료]
[사용자의 심박수가 비정상적으로 높습니다. 현재 구상 중인 물리적 행동은 생존율 5% 미만의 치명적 위험 시나리오입니다.]

"알아. 그러니까 하는 거야."

현우가 씩 웃었다. 두려움이 섞여 있었지만, 그것을 압도하는 지독한 독기가 서린 웃음이었다.

"적 AI가 날 벌레 취급한다고 했지? 그럼 진짜 벌레가 어떻게 튀어오르는지 보여줘야지. 아린이한테 가던 타겟팅, 전부 내 쪽으로 돌려."

현우는 은신처의 문을 열어젖혔다. 스스로 가장 위험한 미끼가 되기 위해, 상대가 쳐놓은 덫의 한복판을 향해 걸음을 내디뎠다. 합리가 끝난 곳에서, 비합리적인 인간의 진짜 싸움이 시작되고 있었다.
