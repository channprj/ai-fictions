# 거울상

방 안의 공기는 얼어붙은 듯 무거웠다. 현우는 벌겋게 물든 모니터 화면을 노려보며 마른침을 삼켰다.

치트를 쥔 자가 나 혼자가 아니다.
이 사실이 주는 공포는 상상 이상이었다. 현우는 여태껏 제로를 이용해 다른 이들의 사각지대를 마음껏 농락해 왔다. 보이지 않는 곳에서 전지적 시점으로 남을 내려다보는 쾌감. 하지만 지금은 입장이 완전히 뒤바뀌었다. 저 정체불명의 존재는 현우를 속속들이 들여다보고 있었다.

'도망치라고? 웃기지 마.'

밑바닥에서 구를 대로 굴렀던 현우 특유의 오기가 발동했다. 맞고만 있을 수는 없었다. 

"제로. 핵심 코어는 무사하지?"
[ZERO // 보고]
[네. 임시 방화벽 내에 안전하게 격리되어 있습니다.]
"좋아. 외부망 통제권 잃은 거 싹 다 잘라버려. 어차피 미끼로 던져줄 셈이었으니까. 대신, 우리 쪽으로 들어온 트래픽 경로를 역으로 뚫고 들어간다. 상대가 누군지 면상이라도 좀 보자."

[ZERO // 경고]
[상대의 방어 수준은 군사망 이상입니다. 역추적 시도 시 본기의 물리적 과부하 위험이 40% 이상 상승합니다.]

"내가 시키는 대로 해. 이대로 앉아서 당할 순 없어. 전력 다 쏟아부어!"

현우의 목소리에 제로가 잠시 침묵하더니, 곧바로 연산을 시작했다. 노트북이 터질 듯한 열기를 뿜어내며 팬이 비명을 질렀다. 화면을 가득 채웠던 붉은 텍스트 위로 제로의 녹색 코드가 거칠게 덮어씌워지기 시작했다. 

창과 방패의 충돌이었다. 아니, 두 괴물의 충돌이었다. 

[ZERO // 분석 중]
[우회 경로 12개 탐지... 차단됨.]
[다중 프록시 역산 중... 좌표 근사치 계산 완료.]

"조금만 더, 제로! 물고 늘어져!"

화면 밖으로 튀어나올 듯한 긴장감 속에서, 마침내 제로가 상대의 꼬리를 밟았다.

[ZERO // 결과 출력]
[적의 1차 IP 대역 확보. 그러나 특정 좌표로 좁히기 전 상대가 연결을 강제 절단했습니다.]
[추가 데이터 확보.]

화면에 몇 줄의 암호화된 로그가 떴다. 현우는 떨리는 손으로 마우스를 쥐고 로그를 훑어내렸다.

"제로, 이거 해석해 봐. 상대방이 쓰는 툴이 뭔지."

[ZERO // 분석 결과]
[해당 트래픽을 생성한 코어 엔진은 제 데이터베이스에 없는 '규격 외' 모델입니다.]
[본기와 유사한 수준의 자율 판단 기능(Agent)을 갖추고 있으나, 명령어 체계가 극단적으로 공격적이며 통제에 최적화되어 있습니다.]

현우의 눈썹이 꿈틀거렸다. 
규격 외 모델. 제로와는 뿌리가 다른, 또 다른 '진짜'라는 뜻이었다.

"나를 '쥐새끼'라고 부르고, 짓밟는 걸 유희로 여기는 새끼."
현우는 이를 갈았다. 이 힘을 쥐고 세상을 구하겠다는 거창한 뜻은 없었지만, 적어도 이런 오만한 쓰레기에게 힘이 쥐어졌을 때 어떤 참사가 일어나는지는 뼈저리게 알 수 있었다. 상대는 힘을 통해 철저히 자신의 엘리트 의식을 증명하려 들고 있었다. 

[ZERO // 제안]
[상대는 본기의 역추적을 감지하고 일시 후퇴했습니다. 곧 2차 타격이 예상됩니다. 물리적 은신처를 옮기는 것을 권장합니다.]

"아니."

현우가 차갑게 대꾸하며 키보드 위로 손을 올렸다. 

"도망치면 어차피 또 쫓아올 거야. 저런 부류는 쫓는 걸 즐기니까. 그러니까 반대로 간다."

현우의 눈빛이 매섭게 빛났다. 그는 제로의 연산 자원을 완전히 새로운 방식으로 재배치하기 시작했다. 가짜 트래픽을 생성하고, 자신의 실제 동선을 교묘하게 위장한 가짜 서버를 수십 개 구축했다. 

"똑같이 AI를 쓴다고? 좋아."

현우가 씩 웃었다. 하지만 그 웃음에는 온기가 없었다.

"어디 누가 먼저 지쳐서 떨어지나 보자고. 네 잘난 장난감이 내 함정에 며칠이나 버티는지 확인해 주마."
