# 03. 역산

[← 이전 회차: 02. 거울상](./02-part2-mirror-image.md) | [시리즈 홈](../README.md) | [권 목차](./README.md) | [다음 회차: 04. 표적 →](./04-part4-the-target.md)

현우가 구축한 함정은 완벽에 가까웠다.
제로의 Tier 3 시뮬레이션을 한계까지 돌려, 상대가 침투할 수밖에 없는 가짜 취약점들을 정교하게 설계했다. 적이 미끼를 무는 순간, 트래픽은 수십 개의 위장 서버를 맴돌며 상대의 연산 자원을 고갈시키고 최종적으로 적의 본체 좌표를 역산출하도록 짜여 있었다.

"들어온다."

새벽 2시. 모니터에 알람이 점등됐다. 현우는 숨을 죽인 채 화면을 주시했다. 상대의 트래픽이 현우가 열어둔 첫 번째 가짜 서버로 진입하고 있었다.

"좋아, 제로. 계획대로 플랜 A 가동해."

[ZERO // 승인]
[위장 서버망 개방. 적의 트래픽을 루프(Loop)로 유도합니다.]

현우의 입꼬리가 살짝 올라갔다. 치트를 쥔 상대라 해도 결국 함정의 논리 구조 안으로 들어오면 빠져나갈 수 없었다. 제로의 시뮬레이션은 단 한 번도 틀린 적이 없었다.

하지만, 상황은 5초도 지나지 않아 기괴하게 비틀어지기 시작했다.

[ZERO // 오류 보고]
[적의 트래픽이 루프에 진입하지 않았습니다. 위장 서버 14곳이 동시에 무력화되었습니다.]

"뭐? 어떻게? 경로가 막힌 것도 아닌데!"

[ZERO // 분석 중]
[상대는 본기의 함정 구조를 이미 시뮬레이션하여 파악했습니다. 가짜 취약점을 역이용해, 본기의 코어 방화벽을 향한 직접적인 백도어를 생성 중입니다.]

현우의 머릿속이 하얗게 섰다.
'내 시뮬레이션을... 시뮬레이션 했다고?'

그것은 바둑에서 내가 둘 십여 수를 미리 내다보고 완벽한 덫을 놨는데, 상대가 그 십여 수의 덫마저 계산에 넣고 정확히 내 목을 치러 들어오는 것과 같았다.

"플랜 B! 당장 플랜 B로 전환해! 더미 데이터 쏟아부어!"

[ZERO // 경고]
[플랜 B 전개 실패. 상대가 더미 데이터 배포 타이밍을 예측하고 사전에 차단 프로토콜을 가동했습니다. 방화벽 2선 붕괴 위험.]

현우의 손가락이 키보드 위에서 굳어버렸다.
완벽하게 읽히고 있었다. 자신이 제로를 통해 세운 모든 전략, 모든 변수, 모든 함정이 상대방의 괴물 같은 AI에게 실시간으로 카운터 당하고 있었다.

[ZERO // 치명적 경고]
[적의 역산이 완료되었습니다. 본기의 물리적 접속 위치가 노출되었습니다.]
[좌표: 은신처 반경 50미터 이내 특정 완료.]

"미친...!"

현우가 자리에서 벌떡 일어났다. 오만했던 자신감이 산산조각 났다. 치트를 치트로 억누르는 싸움. 여기서 그는 패배했다.

그 순간, 책상 위에 놓여 있던 현우의 스마트폰이 기분 나쁜 진동을 토해내며 울리기 시작했다.
발신자 표시 제한. 현우는 떨리는 손으로 전화를 받았다.

"여보세요."

수화기 너머로 짧은 실소가 들려왔다. 아주 여유롭고, 지독하게 오만한 웃음소리였다.

— "함정 수준 하고는. 뒷골목 쓰레기가 머리를 굴려봤자 거기서 거기지."

현우의 심장이 쿵 내려앉았다. 상대의 목소리는 너무나도 평온해서 오히려 소름이 끼쳤다.

"너... 누구야."

— "나? 네 분수에 넘치는 장난감을 빼앗으러 온 사람."

전화 너머의 남자가 나른하게 말을 이었다.

— "너 같은 버그 새끼가 설쳐대는 걸 보는 건 여기까지야. 네가 얼마나 무력한지, 똑똑히 가르쳐주마. 쓰레기는 쓰레기통에 있어야지."

통화가 툭 끊어졌다. 현우는 식은땀으로 흠뻑 젖은 채, 숨을 헐떡이며 모니터를 노려보았다. 모니터에는 적의 트래픽이 현우의 남은 방어망을 유린하는 붉은 궤적이 선명하게 남아 있었다.
