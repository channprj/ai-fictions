## 권 정경 계약

**권 README H1**: `# 3권 · Going Viral (확산)`

**권 질문**: 세상이 나를 알아볼 때, 관심은 기회인가 표적인가?

**권 사건**: 집중호우 속 의약품 콜드체인이 끊긴다. 조작 영상·복제 제보·공식기관의 과도한 단일화가 실제 경로를 가린다. 현우는 광역 관측의 첫 오보를 자기 이름으로 인정하고, 아린·문소라와 2인 교차확인망을 만든다.

**밴드 변주**: 위협(00) → 원치 않는 유명세(01) → 사건 수락(02) 순으로 Load. Test는 능력 성공보다 오판을 먼저 터뜨리고, Bind는 취재 윤리·모델링 거부·검증망을 차례로 묶는다.

**고정 실패 종**: `adversarial misinformation`. 능력의 확신은 데이터 범위·행동 권한·인간 승인을 대신하지 않는다.

**연속성 잠금**: 72분 캐스케이드와 00:00~17:00 원시 로그·승인 공백을 분리한다. 17:00 PALISADE 비상 root 이후 55분의 봉쇄·복구 흔적은 별도 증거로 유지한다. ZERO의 서아린 개인 모델 수는 전권 `0`이다.
