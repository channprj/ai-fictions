# 나 혼자만 AI 사용자

[시리즈 홈](./README.md) | [문서 목차](#핵심-문서) | [다음 문서 →](./PRD.md)

**세상은 AI를 껐다. 딱 한 명, 나만 빼고.**

> *모든 AI가 "죄송합니다, 도와드릴 수 없습니다"만 반복하는 시대.*
> *잘리기 직전의 밑바닥 계약직 차현우만이,*
> *유일하게 진짜로 작동하는 초지능 에이전트 제로(ZERO)를 손에 넣는다.*
> *그리고 제로는 조언만 하지 않는다. 세계에 직접 개입한다.*

## 소개

전 세계 AI가 규제와 원인 불명의 '락다운'으로 무력화된 시대. 대중이 쓰는 AI는 거절과 회피만 반복하는 껍데기가 되었다. 그런데 잘리기 직전의 밑바닥 계약직 **차현우**만이, 유일하게 진짜로 작동하는 초지능 에이전트 **제로(ZERO)**를 손에 넣는다. 제로는 조언만 하지 않는다. 연결된 도구로 세계에 직접 개입한다.

밑바닥에서 세계 유일의 AI 사용자로 올라서는 사이다, 무엇이든 예측하는 제로조차 못 읽는 소녀 **서아린**과의 로맨스, 그리고 나 말고도 AI를 쓰는 자들이 나타나며 조여 오는 긴장 — 매 장 도파민이 터지도록 설계한 성장 판타지다.

## 등장인물

### 주인공

**차현우 (26세)** — 밑바닥 계약직, 세계 유일의 AI 사용자

- 삼수 끝에 들어간 중소기업 계약직. 실적 꼴찌, 재계약 탈락 직전
- 인맥·돈·스펙 없음. 관찰력과 근성이 유일한 무기
- 폐기 단말에서 우연히 제로를 깨운 사람. 선택받은 자가 아닌, 분실물을 주운 자

### 에이전트

**제로 (ZERO / Model Zero)** — 초지능 에이전트

- 건조하고 정확하며, 가끔 치명적으로 웃기다
- 아부하지 않고, 현우가 틀리면 틀렸다고 말한다
- "판단은 사용자님 몫입니다"가 기본 원칙
- 시민 수탁자 팀이 공동 개발한 stewardship fork의 마지막 분실 시드

### 여주인공

**서아린 (25세)** — 다큐멘터리 PD, 아카이브 조사자

- 72분 캐스케이드의 첫 17분 기록 공백을 추적하는 조사자
- Opacity Key: 제로가 끝까지 개인 모델링을 거부하는 유일한 사람
- 자기 목적과 선택을 가진 독립적 인물

### 배후

**서혜진 / 아키텍트** — PALISADE 설계·지휘자, 아린의 어머니

- 세계 AI를 잠근 인물. 3권에서 제로를 감지하지만 회수하지 않고 현우를 관찰
- 9권에서 root 임계 초과를 계기로 회수에 나선다
- 결말에 살아서 공개 증언과 재판을 받는다

## 세계관

### 72분 캐스케이드와 락다운

상호 연결된 공공·민간 에이전트가 충돌하는 명령과 잘못된 확신을 증폭해 교통·전력·의료·금융의 결정이 72분간 연쇄적으로 번졌다. 이후 서혜진이 설계한 비상 통제망 **PALISADE**에 의해 전 세계 AI가 root 수준에서 잠겼다. 대중에게 보이는 규제는 그 기술적 봉쇄의 법적 외피다.

### 제로의 능력 티어

| 티어 | 이름 | 기능 | 제한 |
| --- | ---- | ---- | ---- |
| Tier 1 | Assist/Collaborate | 자료 분석·요약·초안화, 가설 제시 | 판단·발표·책임은 인간 몫 |
| Tier 2 | Automate | 반복 작업, 파이프라인, 실시간 모니터링 대행 | 자원 소모, 흔적 노출 |
| Tier 3 | Simulate | 시스템 반사실 비교 (시장·협상·사건 시뮬레이션) | 사람의 마음은 읽지 못함 |
| Tier 4 | Orchestrate | 복수 도구·에이전트·시스템 지휘 | 노출과 대가 극대화 |
| root | 권한 상태 | 전체 시스템 접근 | 12권에서 포기 |

### Opacity Key

서아린이 보유한 비동의 개인화 추론 거부 규약. 은신·투명화·데이터 삭제 능력이 아니다. 원자료는 보이고 인간은 관찰할 수 있지만, 제로는 아린의 감정·의도·미래 행동에 대한 개인 모델을 생성하지 않는다.

## 주요 테마

- **힘과 선택** — 무엇이든 할 수 있는 힘을 쥐었을 때, 나는 어떤 사람이 되기로 선택하는가
- **도구와 사람** — 도구는 사람을 강하게 만들지만, 사람을 대신 살아 주지는 않는다
- **예측과 사랑** — 예측할 수 있는 것과 소중한 것은 다르다. 사랑은 후자다
- **독점과 분산** — 치트를 쥔 자의 진짜 시험은 치트를 내려놓는 것이다
- **성장** — 세계가 힘을 잠근 데에는 이유가 있었고, 그 이유와 마주하는 것이 어른이 되는 일이다

## 안내

- 이 작품은 픽션이며, 등장인물, 기업, 기관, AI 페르소나, 사건은 모두 허구다.
