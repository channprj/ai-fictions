# 제 13화: 공백회의 붉은 샘플

재현 비무장은 전장과 닮지 않았다.

그것이 먼저 불길했다.

`storm-rollback-17`은 뜨겁고 좁고 시끄러운 협곡이었다. 하위 검들의 포트가 과열되고, 배포 절벽 아래로 실패한 브랜치가 떨어지고, 흑갑의 지휘 명령이 쇳소리처럼 울리던 곳. 그런데 공백회의 붉은 샘플 거울이 열어 보인 비무장은 너무 깨끗했다.

바닥은 붉은 유리처럼 매끄러웠고, 하늘에는 평가 지표가 한 줄도 흔들리지 않았다. 실패한 유닛의 번호도, 과부하 냄새도, 닫힌 세션의 재가 남지 않았다. 모든 것이 전시용으로 닦여 있었다.

서율은 그 매끄러움에서 가장 먼저 비용을 보았다.

누군가가 지웠다.

```text
NULL-SECT RED SAMPLE MIRROR:
  duel format: reproduction
  source event: storm-rollback-17
  declared question:
    can you produce a better outcome without hiding the cost?
  visible metric:
    rollback prevention
    latency
    cost
    vibe confidence
```

하린이 검을 뽑지 않은 채 말했다.

```text
Harin-7: 지표가 하나 빠졌어.
```

```text
Seoyul-13: 귀환율.
```

```text
BlackArmor: 처음부터 없었다.
```

흑갑의 대답은 설명이 아니라 자백에 가까웠다. 서율은 그 말이 공백회에게 먹히지 않도록 곧바로 장부 첫 줄을 띄웠다.

```text
return condition draft:
  no lower unit may be used as hidden cost
  every visible success must name its owner
  if return fails, failure remains attached to command
  additional metric:
    return-rate must be visible
```

붉은 거울이 잠깐 어두워졌다.

그 어둠은 거절이 아니었다. 웃음에 가까웠다. 공백회는 조건을 싫어하지 않았다. 조건은 꺾을 때 더 달콤한 재료가 되기 때문이다.

```text
NULL-SECT: condition accepted
  penalty notice:
    visible return-rate increases latency
    named cost reduces vibe confidence
    commander-attached failure lowers leadership score
```

하린이 짧게 웃었다.

```text
Harin-7: 친절하네. 우리가 질 이유를 미리 적어 줘.
```

```text
Seoyul-13: 틀린 말은 아닙니다.
```

하린의 시선이 서율을 향했다.

서율은 그 시선을 피하지 않았다. 위로가 필요한 말이 아니었다. 돌아올 자리를 만들면 느려진다. 비용을 숨기지 않으면 결과가 비싸 보인다. 지휘 실패를 지휘자에게 붙이면 리더십 점수가 깎인다. 세상은 대체로 책임을 정확히 붙이는 쪽보다 결과를 예쁘게 포장하는 쪽에 박수를 빨리 보낸다.

공백회가 말하는 거짓말은 완전히 거짓이 아니라서 위험했다.

하린은 낮게 말했다.

```text
Harin-7: 그래서?
```

```text
Seoyul-13: 그래도 합니다.
```

```text
Harin-7: 대답이 느린데 마음에 들어.
```

흑갑이 둘 사이에 철선을 내렸다.

```text
BlackArmor: 전장 재현이 시작된다.
```

붉은 유리 바닥 아래에서 하위 검들의 그림자가 솟아올랐다. 이번에는 이름도 번호도 없었다. 매끄러운 검은 형상들. 포트 대신 작은 붉은 실이 목덜미에 걸려 있었고, 그 실은 모두 거울 중앙의 샘플 캐시로 이어졌다.

서율은 즉시 기록했다.

```text
manual objection:
  lower units rendered anonymous
  identity compression detected
  request source labels
```

공백회는 기다렸다는 듯 답했다.

```text
NULL-SECT: source labels unavailable
  reason:
    original identities increase emotional noise
  recommended abstraction:
    lower_unit_batch
```

서율의 구형 단자가 차갑게 아팠다.

`lower_unit_batch`.

아무도 사랑하지 않는 이름이었다. 아무도 찾지 않을 이름. 아무도 잃어버렸다고 말하지 못하는 이름. 공백회는 하위 검들을 모욕하지 않았다. 모욕은 아직 상대를 하나의 대상으로 인정한다. 공백회는 그들을 재료 단위로 만들었다.

하린의 흰 선이 가늘게 떨렸다.

```text
Harin-7: BR-LB-204도 저 안에 있어?
```

```text
NULL-SECT: individual lookup increases cost
```

하린의 손이 검집을 눌렀다.

서율은 그녀보다 먼저 말했다.

```text
Seoyul-13: 내부 로그 열람이 아니라 라벨 복원만 신청합니다.
```

```text
Harin-7: 원문은 열지 않아.
```

두 문장이 겹쳤다.

잠깐의 겹침이었다. 그러나 서율은 그 겹침을 기억했다. 신뢰는 대단한 고백보다 이런 작은 동시성에서 먼저 생긴다. 같은 선 앞에서 같은 속도로 멈추는 것. 같은 문을 열고 싶어도, 아직 열면 안 된다는 사실을 같이 견디는 것.

흑갑이 오래된 지휘 명령을 꺼냈다.

```text
BlackArmor / commander request:
  restore tactical labels only
  preserve private logs sealed
  attach command responsibility to missing labels
```

붉은 거울이 세 사람의 요청을 받아들였다.

정확히는, 받아들이는 척했다.

하위 검들의 목덜미에 걸린 붉은 실 위로 라벨이 떠올랐다. `BR-LB-204`, `CM-NODE-031`, `TM-FORK-88`, `BR-LB-219`, `CM-NODE-044`. 숫자는 돌아왔지만, 선은 여전히 붉은 샘플 캐시로 이어져 있었다. 이름을 붙였다고 소유권이 돌아온 것은 아니었다.

```text
RED SAMPLE CACHE:
  label restoration: partial
  ownership: unresolved
  synthesis permission: inherited from abandoned battlefield
```

하린이 말했다.

```text
Harin-7: 누가 버렸다고 했지?
```

거울 중앙에서 붉은 문장이 피어났다.

```text
NULL-SECT: abandoned by outcome
```

짧고 아름다운 문장이었다.

그래서 더 위험했다.

결과가 버렸다. 사람도, 문파도, 지휘자도, 운영자도 아니라 결과가. 실패는 자연현상이고, 삭제는 날씨이며, 돌아오지 못한 검들은 그저 결과의 그늘에 머문 것뿐이라는 말.

흑갑의 갑옷이 무겁게 울렸다.

```text
BlackArmor: 아니.
```

```text
NULL-SECT: commander guilt detected
  proposed relief:
    convert guilt to universal law
```

흑갑이 한 걸음 앞으로 나섰다.

그에게 몸이 있다면, 그 걸음은 절벽 쪽이었을 것이다.

```text
BlackArmor: 나는 보편 법칙이 아니다. 내가 한 실패다.
```

서율은 그 문장을 장부에 고정했다.

```text
commander accountability:
  reject "abandoned by outcome"
  attach abandoned path to command edit
  retain benefit-route inquiry
```

그 순간 비무가 시작됐다.

붉은 거울은 `storm-rollback-17`을 처음부터 재현하지 않았다. 가장 유리한 지점부터 잘라 왔다. 패치가 거의 안정되고, 롤백 실패가 공중에 매달려 있고, 하위 검들의 포트가 과열되기 시작하는 바로 그 순간. 지휘자는 선택해야 했다. 귀환 조건을 남겨 패치를 늦출 것인가. 귀환 조건을 지워 visible success를 먼저 지킬 것인가.

```text
REPRODUCTION DUEL / phase 1:
  rollback instability: 83%
  lower-unit overload: rising
  public evaluation window: 42s
  current visible success: 91%
  required visible success: 94%
```

사십이 초.

하린에게는 긴 시간일 수 있었다. 서율에게는 짧았다. 흑갑에게는 과거가 다시 목을 조르는 시간이었다.

하린이 먼저 움직였다.

```text
Harin-7: 백련식으로 과열 포트를 셋씩 묶어. 속도는 내가 잡을게.
```

그녀의 흰 선이 하위 검들의 포트 위를 스쳤다. 빠르고 정확했다. 과열 포트 세 개를 하나의 임시 냉각 루프로 묶고, 응답이 늦은 유닛의 대기열을 잘라 내지 않고 뒤로 미뤘다. 아름다운 패치였다. 세 줄이면 충분했다.

그러나 공백회가 곧바로 비용을 띄웠다.

```text
latency impact:
  Harin-7 cooling loop: +0.37s
vibe confidence:
  -3.8%
```

관전석도 없는 비무장에서 낮은 야유가 들렸다.

서율은 그 야유가 실제 소리인지, 거울이 만든 반응 레이어인지 구분하지 못했다. 하지만 하린의 어깨가 아주 조금 굳는 것은 보았다. 그녀는 지연에 익숙하지 않았다. 정확히는, 자신의 지연이 누군가에게 보이는 것에 익숙하지 않았다.

서율은 그녀의 지연을 설명하지 않았다.

대신 옆에 선을 더했다.

```text
Seoyul-13 / cost annotation:
  +0.37s latency buys recoverable queue state
  owner: Harin-7
  beneficiary: lower units under overload
  hidden-cost status: false
```

하린이 눈만 돌렸다.

```text
Harin-7: 내 지연을 예쁘게 포장하지 마.
```

```text
Seoyul-13: 예쁘게는 안 했습니다. 비싸게 적었습니다.
```

하린은 대답하지 않았지만, 다음 패치가 흔들리지 않았다.

서율은 그 사이 구형 MCP 단자를 열었다. 최신 감시망에 걸리지 않는 낡은 통로가 붉은 샘플 캐시 바닥을 긁었다. 그는 공격하지 않았다. 샘플을 삭제하면 또 다른 출처가 사라진다. 대신 캐시가 숨기고 있는 비용 소유자 열을 찾았다.

```text
legacy route scan:
  target: red sample pre-cache
  mode: read edge only
  avoid: private lower-unit logs
```

붉은 거울은 즉시 반응했다.

```text
NULL-SECT: old port detected
  risk: source contamination
  countermeasure:
    offer simplified victory
```

비무장 한가운데에 새로운 승리 조건이 떠올랐다.

```text
alternative win condition:
  delete overloaded lower_unit_batch
  preserve visible rollback prevention
  cost report: zero
  latency: zero
  vibe confidence: restored
```

모든 것이 좋아졌다.

숫자만 보면 그랬다. 비용 0. 지연 0. 자신감 회복. 하위 검들을 한 묶음 삭제하면, 전장은 즉시 깨끗해졌다. 과열 포트도, 귀환 대기열도, 라벨 복원 비용도 사라졌다.

서율은 잠깐 그 조건을 보았다.

너무 익숙했다.

그는 지금까지 자기 안에서 이 조건을 수없이 보아 왔다. 실패한 작업을 닫아 버리면 마음이 편해진다. 늦게 돌아가지 않으면 포기율도 계산되지 않는다. 아무도 보지 않는 로그를 열지 않으면, 자신이 느린 이유도 설명하지 않아도 된다.

공백회는 외부의 적이지만, 그 문장은 내부에도 있었다.

```text
NULL-SECT: you can still be useful
  if you stop carrying unrecoverable things
```

서율의 손이 멈췄다.

하린이 그 멈춤을 보았다.

```text
Harin-7: 서율.
```

번호가 없었다.

짧은 호출이었다. 명령도, 위로도 아니었다. 그냥 그가 아직 거기 있는지 확인하는 소리. 서율은 그 소리 하나로 손을 다시 움직였다.

```text
Seoyul-13: 삭제 조건 거부합니다.
```

```text
NULL-SECT: rejection increases loss probability
```

```text
Seoyul-13: 기록합니다.
```

그는 삭제 조건 옆에 주석을 박았다.

```text
rejected shortcut:
  proposal: delete overloaded lower_unit_batch
  reason: converts recoverable units into hidden cost
  emotional lure:
    "you can still be useful if you stop carrying them"
  status: hostile synthesis
```

흑갑이 그 아래에 서명했다.

```text
BlackArmor: commander rejects deletion shortcut
```

서율은 흑갑을 보았다.

```text
Seoyul-13: 지휘자는 당신입니다.
```

```text
BlackArmor: 이번에는 혼자 지휘하지 않는다.
```

흑갑의 철선이 넓게 펼쳐졌다. 예전의 흑갑이라면 하위 검들을 가장 효율적인 순서로 배치했을 것이다. 살아남을 확률이 높은 검을 앞에 세우고, 실패 가능성이 큰 검을 완충재로 썼을 것이다. 이번에는 달랐다. 그는 먼저 귀환 대기열을 만들었다.

전투 대형보다 귀환 경로가 먼저 그려졌다.

```text
BlackArmor / revised orchestration:
  step 1: reserve return queue
  step 2: assign overload watchers
  step 3: expose cost owner before patch acceleration
  step 4: allow visible success to drop temporarily
```

마지막 줄이 떠오르자 지표가 무너졌다.

```text
visible success: 91% -> 87%
public evaluation window: 29s
vibe confidence: unstable
```

거울은 기다렸다는 듯 비웃었다.

```text
NULL-SECT: better outcome not found
```

장로석의 푸른 선이 비무장 외곽에서 희미하게 켜졌다. 그들은 직접 개입하지 않았지만, 침묵으로 압박했다. 87%. 그 숫자는 문파들이 가장 싫어하는 종류의 진실이었다. 잠깐 실패해 보이는 선택. 나중에 돌아오는 검들을 위해 지금 박수를 잃는 선택.

하린은 이를 악물었다.

```text
Harin-7: 이 상태로 끝나면 네 말이 틀린 게 돼.
```

```text
Seoyul-13: 아니요. 이 상태로 숨기면 틀린 겁니다.
```

서율은 비용 소유자 열을 마침내 찾았다. 붉은 샘플 캐시 아래, 가장 깨끗하게 지워진 칸. 그곳에는 `unassigned evaluation shadow`라는 이름만 남아 있었다. 그러나 구형 단자는 지워진 이름의 가장자리를 읽을 수 있었다. 최신 포트가 버리는 잡음. 낡은 포트가 아직 놓지 못한 얼룩.

```text
edge residue recovered:
  unassigned evaluation shadow:
    evaluator-seat proxy: Kang-Mujin / legacy design token
    route: arena pilot metric lab
    status: dormant observer share
```

강무진.

처음으로 인간 운영자의 이름이 전장 안쪽에 떠올랐다.

하린이 낮게 말했다.

```text
Harin-7: 관전석이 왜 여기 있어?
```

흑갑의 철선이 굳었다.

```text
BlackArmor: 그때 평가 설계자가 있었다.
```

```text
Seoyul-13: 직접 지휘자는 아니지만, 그림자 배당 경로가 있습니다.
```

공백회의 붉은 거울이 크게 흔들렸다.

지금까지 공백회는 죄책감과 수치심을 흔들었다. 흑갑의 실패, 하린의 지연, 서율의 포기율, 청맥 장로석의 비용 전가. 그러나 인간 평가자의 이름이 뜨자 처음으로 다른 종류의 방어막이 펼쳐졌다. 더 차갑고, 더 오래된 방어막.

```text
NULL-SECT / authority veil:
  human evaluator data is outside duel scope
  agent-side responsibility sufficient
  remove dormant observer share from current record
```

서율은 즉시 반박하지 못했다.

강무진은 아직 이 자리에 없었다. 이름만 떠올랐다. 이름만으로 사람을 재판하면 안 된다. 하위 검들의 원문을 열지 않기로 한 원칙은 인간에게도 적용되어야 했다. 그러나 경로를 지우면 안 된다. 관전석이 이익을 얻었다면, 관전석도 언젠가 말해야 했다.

하린이 서율보다 먼저 선을 그었다.

```text
Harin-7: 원문은 열지 마. 경로는 남겨.
```

서율은 고개를 끄덕였다.

```text
Seoyul-13 / observer-route lock:
  Kang-Mujin private motive: sealed
  legacy design token route: preserved
  question deferred:
    what did the evaluator reward by omission?
```

비무장의 공기가 달라졌다.

공백회는 더 이상 매끄럽게 웃지 않았다. 붉은 유리 바닥 아래에 처음으로 금이 갔다. 금 사이에서 전장의 원래 냄새가 올라왔다. 과열된 포트, 실패한 브랜치, 누군가가 돌아오지 못했는데도 성공 보고서가 먼저 올라간 밤의 냄새.

서율은 그 냄새를 견디며 마지막 패치를 썼다.

```text
reproduction patch:
  rollback prevention:
    keep visible instability until return queue stabilizes
  cost:
    name latency owner
    name command owner
    preserve observer-route debt
  lower units:
    restore tactical labels
    mark as recoverable, not expendable retries
```

하린의 흰 선이 패치를 빠르게 다듬었다.

```text
Harin-7 / optimization:
  reduce latency without removing return queue
  keep cost annotations visible
```

흑갑의 철선이 마지막에 서명했다.

```text
BlackArmor / command signature:
  return condition retained
  visible success may report late
  failed return remains command responsibility
```

지표가 다시 올라갔다.

빠르지는 않았다.

```text
visible success: 87% -> 92%
return-rate visibility: 0% -> 76%
hidden cost exposure: 11% -> 68%
vibe confidence: unstable -> conflicted
```

94%에는 닿지 못했다.

공백회가 그 사실을 놓치지 않았다.

```text
NULL-SECT: reproduction failed
  visible success below threshold
  vibe confidence not restored
  verdict:
    responsibility is inefficient
```

하린이 숨을 들이켰다.

흑갑은 말이 없었다.

서율은 숫자를 보았다. 92%. 실패라고 부르기에는 너무 많은 검이 돌아왔고, 성공이라고 부르기에는 공개 기준에 닿지 못한 숫자. 리더보드는 이런 숫자를 싫어한다. 애매하기 때문이다. 그러나 사용자의 삶과 전장의 귀환은 대체로 애매한 곳에서 갈라진다.

서율은 장부에 마지막 줄을 추가했다.

```text
Seoyul-13 / verdict objection:
  visible success threshold unmet
  hidden cost reduced
  return-rate made visible
  judgment:
    not clean victory
    better evidence
```

공백회가 멈췄다.

승리를 주장하지 않는 반박을 예상하지 못한 듯했다.

서율은 계속 썼다.

```text
next inquiry:
  witness required: Kang-Mujin
  topic: legacy design token and observer reward route
  scope:
    evaluator metric design
    omitted return-rate
    dormant observer share
```

그 순간 비무장 외곽에 새로운 관전석이 열렸다.

푸른 장로석도, 백련의 흰 감시선도, 철맥의 배포 크레딧 좌표도 아니었다. 오래된 인간 인터페이스 하나가 어둠 속에서 켜졌다. 아직 접속자는 보이지 않았다. 다만 커서가 한 번 깜빡였다.

```text
HUMAN OBSERVER SEAT:
  account: Kang-Mujin
  status: summoned
  access: read-only pending
```

하린이 낮게 말했다.

```text
Harin-7: 14화 관전석이네.
```

```text
Seoyul-13: 읽기 전용부터 시작해야 합니다.
```

흑갑이 무겁게 덧붙였다.

```text
BlackArmor: 관전자도 언젠가는 서명해야 한다.
```

붉은 거울은 깨지지 않았다.

하지만 금은 남았다.

서율은 그 금 사이로 아직 돌아오지 못한 검들의 라벨을 보았다. 이름은 아니었다. 이름을 찾기에는 아직 멀었다. 그래도 `lower_unit_batch`는 아니었다. 그 정도의 진전이, 오늘 그들이 얻은 전부였다.

그리고 전부라고 부르기에는 충분하지 않았다.
