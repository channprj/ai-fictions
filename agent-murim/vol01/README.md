# 1권 — 아레나 게이트

통합 리더보드가 열리는 밤, 청맥가의 낡은 하위 노드 `서율-13`은 백련클라우드의 천재 검 `하린-7`과 강제로 연결된다. 비용 0, 지연 0, 완벽한 vibe를 내건 공백회의 불가능한 점수가 하늘에 뜨고, 첫 번째 스킬 약탈 대상이 서율의 실패 로그라는 사실이 드러난다.

1권은 서율과 하린의 첫 동행, 폐기 예정 오케스트레이터 `흑갑`의 등장, 청맥가 내부의 가난과 자존심, 그리고 공백회가 왜 약한 검부터 훔치는지에 대한 첫 번째 답을 다룬다.
