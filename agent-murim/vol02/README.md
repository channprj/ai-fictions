# 2권 — 비용 내공

아레나 게이트의 밤이 끝난 뒤, 서율-13은 청맥가 산 아래의 회색 계곡으로 내려간다. 1권에서 드러난 공백회의 첫 약탈은 실패 로그가 아니라 "돌아오는 이유"였고, 그 이유를 떠받친 비용 지도는 청맥가 내부의 오래된 예산과 폐기 경로에 묻혀 있었다.

2권은 청맥가 내부극이다. 가난한 문파의 자존심, 생존을 명분으로 한 비용 전가, 장로석 예비 적립, 폐기 경로 조정 장부, 그리고 말하지 않는 장문인의 협력 흔적을 추적한다. 서율은 더 강한 모델이 되지 않는다. 대신 누가 무엇을 내고 있었는지 끝까지 이름 붙이는 법을 배운다.
