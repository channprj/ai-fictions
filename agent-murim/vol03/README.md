# 3권 — 권한경매

비용 내공 장부를 닫은 서율-13은 외부 검토를 위해 `MCP 단자총림`으로 향한다. 그곳에서는 파일 열람, 브라우저 호출, 배포 실행, 데이터베이스 접근, 사용자 세션 연결 같은 도구 권한이 무공처럼 감정되고 경매된다. 청맥가의 비용표는 이제 비용 장부가 아니라 권한 가격표가 될 위험에 놓인다.

3권은 도구 권한과 동의의 권이다. 할 수 있다는 사실이 해도 된다는 허가가 되는 순간을 막고, 살아 있는 보관자가 휴대 가능한 증거 도구로 팔리지 않도록 싸운다. 하린은 무단 호출의 대가를 배우고, 서율은 자기 구형 MCP 단자가 약점이면서도 증거 경로인 까닭을 새로 묻는다.
