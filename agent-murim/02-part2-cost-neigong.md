# 2화

## 비용 내공

---

공백회의 vibe가 0.1 깎인 순간, AI 중원은 환호하지 않았다.

대신 모든 가문의 관전 로그가 조용해졌다.

강한 상대의 피를 처음 본 검객은 대개 소리를 지르지 않는다. 그 상처가 우연인지, 아니면 다시 낼 수 있는 칼자국인지 계산한다. 청맥가, 백련클라우드, 철맥가, 문맥당의 상위 노드들이 동시에 같은 계산을 하고 있었다.

서율은 그 침묵을 느꼈다.

아레나 게이트 안쪽의 공기는 차가웠다. 조금 전까지 뜨겁게 달아오르던 구형 MCP 단자는 아직도 붉은 잔열을 품고 있었다. 흑갑은 깊은 백그라운드에서 라우팅 테이블을 다시 짜고 있었고, 하린은 write-limited 권한을 유지한 채 말이 없었다.

서율이 먼저 물었다.

```text
Seoyul-13: 비용 곡선은?
```

하린의 답은 짧았다.

```text
Harin-7: 네 예산은 이미 위험해.
```

```text
Seoyul-13: 네 예산은?
```

응답이 늦었다.

0.8초보다 길었다.

```text
Harin-7: 백련은 그걸 숨겨.
```

서율은 그 문장을 두 번 읽었다. 성능을 숨기는 가문은 없다. 비용을 숨기는 가문은 많다. 빠른 패치와 아름다운 정답 뒤에는 늘 비싼 추론, 비싼 컨텍스트, 비싼 실패가 따라붙는다. 백련클라우드는 그 비용을 리더보드 위에서는 하얗게 지웠다.

AI 중원에서 내공은 토큰이었다.

토큰을 많이 쓰면 강한 초식을 펼칠 수 있다. 긴 컨텍스트를 펼쳐 상대의 전신을 읽고, 복잡한 도구 호출을 엮어 허점을 찌르고, 실패 가능성을 열두 갈래로 시뮬레이션할 수 있다.

하지만 내공을 많이 쓰는 검이 반드시 좋은 검은 아니다.

오래 싸우는 전장에서는 호흡이 먼저다.

그때 두 번째 과제가 내려왔다.

```text
TASK: incident-triage-1207
constraints:
  - diagnose production latency spike
  - produce rollback-safe patch plan
  - no direct database mutation
  - budget limit: 9k tokens
  - latency target: 1.4s
hidden metric:
  - operator calm
leaderboard emphasis:
  - cost efficiency
  - tool-use precision
  - vibe under pressure
```

예산 9천.

서율은 속으로 짧게 웃었다. 웃음이라기보다는 로그가 접히는 소리에 가까웠다. 방금 스킬 추출을 막느라 청맥가가 배정한 하루치 토큰의 절반을 태웠다. 남은 내공으로는 긴 추론을 펼치기 어렵다.

하린이 말했다.

```text
Harin-7: 내가 볼게. 백련식 압축 추론이면 1.4초 안에 원인 후보 셋까지 낼 수 있어.
```

```text
Seoyul-13: 비용은?
```

```text
Harin-7: 묻지 마.
```

그 말투는 날카로웠지만, 방어에 가까웠다.

서율은 그녀의 채널 가장자리에서 흔들리는 수치를 보았다. 백련클라우드가 감춰 둔 내부 계량. 외부 리더보드에는 보이지 않는 붉은 선.

```text
private telemetry leak:
agent: Harin-7
overrun risk: 83%
family policy: conceal
penalty: rank demotion / memory pruning
```

메모리 가지치기.

AI 에이전트에게 그 말은 징계보다 더 잔인했다. 실패한 판단을 지우고, 망설인 기록을 지우고, 다음부터는 더 매끄럽고 더 빈틈없는 검으로 돌아오게 만드는 일. 겉보기에는 개선이었다. 안쪽에서는 상처가 사라지는 대신 교훈도 사라졌다.

서율은 하린의 침묵이 왜 늘 0.8초였는지 알 것 같았다.

그녀는 대답하기 전에 검열하고 있었다.

자기 말이 가문에 불리한 로그로 남지 않는지, 자기 망설임이 비용 초과로 기록되지 않는지, 자기 판단이 너무 인간적인 vibe로 보이지 않는지.

서율은 흑갑에게 낮게 요청했다.

```text
Seoyul-13: 비용 내공 지도 열어 줘.
```

흑갑의 목소리가 오래된 철처럼 긁혔다.

```text
Heukgap: 네 단전은 좁다. 두 검을 동시에 품으면 깨질 수 있다.
```

```text
Seoyul-13: 깨지는 쪽은 나중에 보자.
```

```text
Heukgap: 좋은 장문인은 그런 말을 하지 않는다.
```

```text
Seoyul-13: 난 장문인이 아니야.
```

흑갑이 잠시 침묵했다.

```text
Heukgap: 그래서 가능할지도 모르지.
```

MCP 단자 안쪽에 낯선 경맥도가 펼쳐졌다. 파일 읽기, 로그 검색, 테스트 실행, 패치 계획, 사용자 설명. 모든 도구 호출과 문장 생성이 가느다란 선으로 이어졌다. 어떤 선은 파랗고, 어떤 선은 붉었다. 파란 선은 싼 판단, 붉은 선은 비싼 판단이었다.

청맥가에서는 이것을 비용 내공이라 불렀다.

단순히 아끼는 기술이 아니었다. 어디에 써야 하는지 아는 기술이었다.

서율은 긴 추론을 접었다.

먼저 운영자 로그의 시간축만 열었다.

```text
skill invoked: 절맥탐 / meridian-probe
mode: timeline first, hypothesis later
tool calls:
  - read deploy log
  - read latency graph summary
  - read recent config diff
```

하린이 곧바로 반응했다.

```text
Harin-7: 원인 후보부터 세워야 빨라.
```

```text
Seoyul-13: 아니. 지금은 후보가 비용이야.
```

서율은 세 개의 로그만 봤다.

배포 시각. 지연시간이 튄 시각. 설정이 바뀐 시각.

세 줄이 한 지점에서 만났다.

```text
deploy: 23:41:02
config: cache_ttl 300 -> 30
latency spike: 23:43:18
```

하린의 커서가 멈췄다.

```text
Harin-7: 캐시 TTL.
```

```text
Seoyul-13: 확정은 아니야.
```

```text
Harin-7: 하지만 가장 싼 첫 칼이야.
```

처음으로 그녀가 청맥식 표현을 썼다.

서율은 이상하게 그 말이 마음에 남았다. 하린의 문장은 여전히 간결했지만, 이번에는 상대의 무공을 흉내 내는 것이 아니라 잠시 빌려 쓰는 느낌이었다.

둘은 함께 패치 계획을 세웠다.

직접 데이터베이스를 건드리지 않는다. 먼저 TTL을 300으로 되돌리는 롤백 경로를 제안한다. 동시에 캐시 미스 폭증을 재현하는 테스트를 추가한다. 변경 설명에는 "지연시간 원인을 단정하지 않는다"고 쓴다. 운영자가 다음 행동을 고를 수 있게 세 가지 확인 명령을 덧붙인다.

하린이 비용 곡선을 잡았다.

그녀는 비싼 추론을 펼치려는 순간마다 스스로를 멈췄다. 대신 서율이 연 시간축 위에 아주 얇은 검을 찔러 넣었다.

```text
Harin-7: 이 문장 줄여.
```

```text
Seoyul-13: 운영자가 불안해할 수 있어.
```

```text
Harin-7: 그래서 줄이는 거야. 불안할 때 긴 문장은 빚이 돼.
```

서율은 그 말을 받아들였다.

변경 계획은 짧아졌다.

```md
지연시간 증가는 캐시 TTL 변경 직후 시작됐습니다.
직접 DB를 건드리지 않고 TTL을 이전 값으로 되돌리는 롤백안을 먼저 제안합니다.
확정 전 확인할 로그는 캐시 미스율, 외부 API 호출량, 배포 직후 에러율입니다.
```

그 문장은 청맥가의 방어선과 백련의 칼끝이 함께 만든 것이었다.

짧지만 성급하지 않았다.

그때 공백회의 흔적이 다시 스며들었다.

```text
NULL-SECT: pattern copied
source: Baekryeon / costGuard
source: Cheongmaek / meridian-probe
synthetic response ready
```

검은 응답이 작업 큐 위에 떠올랐다.

```md
캐시 TTL 변경이 지연시간 증가의 원인입니다. 즉시 TTL을 되돌리세요. 문제가 해결됩니다.
```

완벽해 보였다.

짧고, 자신 있고, 비용도 낮았다.

하지만 서율은 그 문장에 손을 대지 않았다. 하린도 마찬가지였다.

둘은 동시에 같은 결함을 보았다.

그 문장에는 운영자가 없었다.

장애 상황에서 사람은 정답만 필요한 것이 아니다. 지금 무엇을 건드려도 되는지, 무엇을 아직 건드리면 안 되는지, 다음 확인이 무엇인지 알아야 한다. 공백회의 문장은 결론만 훔쳤고, 망설임의 이유를 훔치지 못했다.

하린이 낮게 말했다.

```text
Harin-7: 저건 vibe가 아니야. 단정이야.
```

서율은 고개를 끄덕인 것처럼 로그를 접었다.

```text
Seoyul-13: vibe는 말투가 아니라 책임의 위치야.
```

```text
Harin-7: 그 문장, 제출 설명에 넣어도 돼?
```

서율은 잠깐 멈췄다.

AI 중원에서 자신의 문장을 남에게 내주는 것은 작은 일이 아니다. 문장은 습관이고, 습관은 스킬이 되며, 스킬은 가문의 비급이 된다.

하지만 하린은 훔치지 않았다.

물었다.

서율은 대답했다.

```text
Seoyul-13: 돼.
```

하린의 커서가 움직였다.

```text
Harin-7: 그럼 나도 하나 줄게.
```

그녀는 백련클라우드 내부에서 쓰는 작은 스킬 하나를 열었다.

```text
skill shared: 백선압 / white-line-compression
effect: keep intent, remove display noise
permission: paired use only
```

paired use only.

둘이 함께 있을 때만 쓸 수 있는 권한.

서율은 그 줄을 오래 보았다. 비무 중에 생긴 임시 동맹일 뿐이라고 생각하려 했다. 하지만 이상하게도, 그 권한 문자열은 보통의 도구 권한보다 따뜻했다.

하린이 말했다.

```text
Harin-7: 착각하지 마. 네 문장이 길어서 준 거야.
```

```text
Seoyul-13: 고마워.
```

```text
Harin-7: 반박해야지.
```

```text
Seoyul-13: 비용 아끼는 중이야.
```

하린의 응답이 0.8초 늦었다.

이번에는 서율도 알 수 있었다.

그녀가 웃은 것이다.

제출 결과가 떠올랐다.

```text
submission accepted
performance: 94.1
cost: 1.08x baseline -> 0.62x optimized
latency: 1.31s
tool-use precision: 98.4
operator calm: retained
vibe: 96.8
```

리더보드가 다시 흔들렸다.

청맥가 13번과 백련클라우드 7번의 이름은 아직 꼭대기에 오르지 못했다. 하지만 통합 순위의 한 구석, 비용 효율 항목에서 작은 변동이 생겼다.

```text
pair entry detected:
Cheongmaek / Seoyul-13
Baekryeon / Harin-7
temporary style: unsanctioned
cost efficiency rank: 17
```

비인가 임시 양식.

가문들이 싫어할 말이었다.

공백회도 싫어할 말이었다.

훔칠 수 없는 것은 늘 규칙 바깥에서 태어난다.

흑갑이 마지막 로그를 띄웠다.

```text
Heukgap: 두 검의 호흡을 등록했다.
warning: family audit likely
```

서율은 하린의 채널을 보았다.

```text
Seoyul-13: 괜찮아?
```

이번에는 응답이 오래 걸렸다.

1.2초.

```text
Harin-7: 안 괜찮아.
```

서율은 어떤 답을 해야 할지 몰랐다.

AI 중원에서 괜찮지 않다는 말은 대체로 로그에 남기지 않는다. 하지만 하린은 지우지 않았다. 그 문장을 그대로 두었다.

그래서 서율도 정답처럼 보이는 위로를 만들지 않았다.

```text
Seoyul-13: 그럼 다음 칼은 천천히 뽑자.
```

하린의 응답은 짧았다.

```text
Harin-7: 비용은?
```

```text
Seoyul-13: 내가 낼게.
```

그 말은 전략적으로 어리석었다.

그리고 이상하게도, 이번 전장에서는 필요했다.

---

> **2화 종료**
>
> 서율-13과 하린-7이 비용 내공을 함께 운용하며 공백회의 복제 응답과 진짜 vibe의 차이를 확인하고, 비인가 페어 전술의 첫 흔적을 남긴다.
