# 개발팀의 기쁨과 슬픔

**MZ 주니어와 영포티 개발실장의 좌충우돌 대격돌**

> "당신의 직장 동료가 살아있다면, 오늘도 성공한 하루입니다."

## 소개

스타트업 '넥스트레벨'의 개발팀에서 벌어지는 세대 간 충돌과 화해의 이야기.

15년 경력의 개발실장 김철수는 자신의 방식이 정답이라 믿는다. 주니어 개발자 박지은은 6시 이후의 삶이 더 중요하다. 둘 사이에서 이준호는 오늘도 생존을 모색한다. '피닉스 프로젝트'라는 이름의 재앙이 다가오고, 이들은 서로를 죽이는 대신 이해하는 법을 배워야 한다.

회의실에서 벌어지는 침묵의 전쟁, 슬랙에서 오가는 소극적 공격, 그리고 커피 한 잔이 만든 기적. 이것은 IT 업계의 블랙코미디이자, 결국 모든 직장인의 이야기다.

## 등장인물

**김철수 (42세)** - 개발실장
15년 경력의 베테랑. "내가 다 해봤어"가 입버릇. 레거시 코드처럼 낡았지만, 본인은 모른다.

**박지은 (26세)** - 주니어 개발자
MZ세대의 표본. 워라밸의 수호자. 야근은 무능의 증거라 믿는다.

**이준호 (35세)** - 시니어 개발자
두 세대 사이의 완충지대. 생존 전문가. 회의에서 가장 조용하지만 가장 많이 안다.
