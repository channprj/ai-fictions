# PART 7

## 커피 한 잔: 처음 나누는 대화

> *"커피의 온도가 식기 전에, 마음의 온도를 나눠야 한다."*

---

### 7-1. 커피챗인가 취조인가: 어색한 1on1 미팅의 시작

바닥을 친 다음 날.

김철수 실장이 슬랙 메시지를 보냈다.

**@김철수 → @박지은**
> 지은 씨, 시간 있으면 커피 한 잔 할까요?

박지은은 그 메시지를 보고 멈칫했다. 예전 같았으면 경계부터 했을 것이다. *'뭐지? 또 무슨 일이야?'*

하지만 오늘은 달랐다. 어젯밤 나눈 대화가 떠올랐다.

**@박지은**
> 네, 좋아요. 어디서요?

**@김철수**
> 1층 카페에서요. 지금 가능해요?

**@박지은**
> 네!

박지은은 자리에서 일어나며 이준호를 봤다. 이준호가 눈짓으로 물었다. *괜찮아?*

박지은이 살짝 끄덕였다. *괜찮아요.*

---

### 7-2. 1층 카페: 어색하지만 다른 시작

오후 3시, 1층 카페.

두 사람이 마주 앉았다. 어색한 침묵이 흘렀다. 하지만 예전과는 다른 어색함이었다. 경계의 어색함이 아니라, 새로움의 어색함.

"아메리카노요?"

"네."

"저도요."

주문한 커피가 나왔다. 각자 컵을 들고 한 모금씩 마셨다.

김철수 실장이 먼저 입을 열었다.

"지은 씨, 어젯밤에 한 얘기 있잖아요."

"네."

"진심이었어요."

"저도요."

"그래서... 어떻게 하면 좋을지 같이 얘기해보고 싶어서요."

박지은이 놀랐다. 예전의 김철수 실장이었다면 '내가 정했으니까 따라와'라고 했을 것이다.

"좋아요. 근데 솔직히요."

"응?"

"아직 좀 어색해요."

"나도."

두 사람이 동시에 웃었다. 처음으로 같이 웃었다.

---

"실장님, 질문 해도 돼요?"

"응."

"왜 저한테 그렇게 하셨어요? 항상 틀렸다고 하시고, 제 의견 무시하시고."

김철수 실장이 잠시 생각했다.

"... 무서웠어."

"뭐가요?"

"지은 씨가. 아니, 지은 씨 같은 사람들이."

"저요?"

"응. 새로운 기술 잘 알고, 트렌드 따라가고, 자기 주장 확실하고. 그게 무서웠어."

"왜요?"

"내가 뒤처지는 것 같아서."

박지은은 처음 듣는 이야기였다.

"실장님도 그런 걸 느끼세요?"

"당연하지. 15년 경력이라고 해도, 새 기술은 매일 나오잖아. 따라가기 힘들어."

"그래서 제 의견을 무시하신 거예요?"

"무시한 게 아니라... 인정하면 내가 무능해 보일 것 같았어."

"..."

"근데 그게 더 무능한 거였네. 결국."

---

박지은도 말했다.

"저도요, 솔직히."

"응?"

"실장님한테 반항한 거, 다 이유가 있었던 건 아니에요."

"무슨 뜻이야?"

"그냥... 반항하고 싶었어요. 권위에."

"권위?"

"네. 학교 다닐 때부터 그랬어요. '해라' 하면 '왜요?' 먼저 물었어요."

"그건 좋은 거 아니야?"

"근데 가끔은 그냥 싫어서 반대한 것도 있었어요. 실장님 말이 맞는데도."

김철수 실장이 놀랐다.

"진짜?"

"네. TypeScript 도입할 때도요. 실장님이 반대하시니까 더 하고 싶었어요. 기술적 판단보다 감정적으로."

"그랬구나."

"죄송해요."

"아니야. 나도 미안해. 반대할 때 제대로 설명 안 했으니까."

두 사람이 서로를 바라봤다. 처음으로, 방어벽 없이.

---

### 7-3. 세대가 아니라 언어가 달랐다

커피챗은 예상보다 길어졌다. 1시간이 지나고 있었다.

"근데 지은 씨."

"네?"

"우리가 왜 이렇게 싸웠을까?"

"세대 차이 아니에요?"

"그럴까?"

박지은이 잠시 생각했다.

"... 아닌 것 같기도 해요."

"어떤 게?"

"언어가 달랐던 것 같아요."

"언어?"

"네. 실장님이 '효율'이라고 하면, 저는 '구식'으로 들었고. 제가 '성장'이라고 하면, 실장님은 '건방지다'로 들으셨잖아요."

김철수 실장이 웃었다. 쓴웃음이었지만, 진심이 담긴 웃음.

"맞아. 같은 말을 하면서 다른 걸 들었어."

박지은이 노션 앱을 열어 뭔가를 정리하기 시작했다.

```
[용어 번역 테이블]

| 실장님 용어 | 실장님 의도 | 내가 들은 것 |
|------------|-----------|------------|
| "효율적으로" | 검증된 방법으로 | 옛날 방식으로 |
| "표준을 지켜야지" | 일관성 유지 | 내 방식 강요 |
| "내가 해봤는데" | 경험 공유 | 네 의견은 틀렸어 |
| "급하니까" | 실제로 급함 | 네 의견은 안 중요해 |

| 내 용어 | 내 의도 | 실장님이 들은 것 |
|--------|--------|----------------|
| "새로운 기술" | 더 나은 방법 | 유행 따라가기 |
| "제 방식" | 제 전문성 | 반항심 |
| "성장하고 싶어요" | 배우고 싶음 | 현재가 불만족스럽다 |
| "이건 비효율적" | 개선 제안 | 실장님 방식 비판 |
```

김철수 실장이 화면을 들여다봤다.

"이거... 맞네."

"맞죠?"

"근데 이걸 어떻게 고치지?"

"번역기가 필요한 것 같아요."

"번역기?"

"네. 서로 말할 때, 상대방이 어떻게 들을지 먼저 생각하는 거요."

---

### 7-4. 감정 로그를 남기면 버그가 줄어든다

"실장님, 제안이 있어요."

"뭔데?"

"앞으로 솔직하게 말해요. 평소에도."

"지금도 솔직하잖아."

"아니요. 지금처럼 말고. 업무할 때도요."

김철수 실장이 고개를 갸웃했다.

"무슨 뜻이야?"

"저희가 싸웠던 이유 있잖아요."

"응."

"감정을 숨겼기 때문이에요."

"감정?"

"네. 코드에 버그 있으면 로그 남기잖아요. 에러 메시지 출력하고."

"그러지."

박지은이 냅킨에 뭔가를 적기 시작했다.

```javascript
// 버그 있는 코드
function 반응(input) {
  try {
    const 겉모습 = "괜찮아요";
    const 실제감정 = "힘들어요";
    // 감정 로그 없음 - 디버깅 불가!
    return 괜찮은척();
  } catch (e) {
    // 에러 처리 없음
  }
}

// 수정된 코드
function 반응_v2(input) {
  try {
    const 실제감정 = "힘들어요";
    console.log(`[LOG] 현재 감정: ${실제감정}`);
    return 솔직하게말하기();
  } catch (e) {
    console.error(`문제 발생: ${e.message}`);
    return 도움요청();
  }
}
```

김철수 실장이 웃었다.

"개발자답네."

"개발자니까요."

"그래서 제안이 뭐야?"

"주간 1on1 해요."

"1on1?"

"네. 일주일에 한 번, 30분씩. 오늘처럼 솔직하게 얘기하는 시간."

"그게 될까?"

"모르죠. 근데 안 하는 것보단 나을 것 같아요."

김철수 실장이 고개를 끄덕였다.

"해보자."

"진짜요?"

"응. 시도는 해봐야지."

---

### 7-5. 이준호도 함께

"한 가지만 더요."

"뭔데?"

"이거 준호 선임도 같이 하면 안 돼요?"

"준호?"

"네. 선임도 힘들어하시는 것 같아서요. 저희 둘 사이에 끼어서."

김철수 실장이 고개를 끄덕였다.

"좋아. 준호도 부르자."

---

**그날 저녁, 이준호 선임과의 대화:**

장소: 옥상.
시간: 오후 6시.

김철수 실장이 먼저 말했다.

"준호야, 그동안 고생 많았어."

"... 네?"

"너 힘들었잖아. 나랑 지은이 사이에서."

이준호 선임은 놀랐다.

"솔직히요."

"응."

"진짜 힘들었어요."

"미안해."

"아니에요. 제가 말을 안 해서..."

"말 못 하게 만든 것도 내 잘못이야."

이준호 선임은 잠시 생각했다. 그리고 말했다.

"실장님, 저도 솔직하게 말해도 돼요?"

"당연하지."

"저요, 사실 어느 쪽 편도 아니었어요."

"응?"

"실장님 말씀도 맞고, 지은이 말도 맞았거든요. 근데 둘 다 안 들으시니까... 저만 이상한 사람 된 것 같았어요."

"..."

"회의할 때마다 '이거 이상하다'고 생각했는데, 말을 못 했어요."

김철수 실장이 고개를 숙였다.

"미안해, 준호야. 그런 줄 몰랐어."

"괜찮아요. 지금 말하는 것만으로도 시원해요."

"앞으로는 말해줘. 이상하다 싶으면."

"... 네."

"그리고 너한테 부탁이 있어."

"뭐요?"

"나랑 지은이 사이에서 통역 좀 해줘."

"통역이요?"

"응. 우리가 같은 말을 해도 다르게 듣잖아. 그때 중간에서 해석해주면 좋겠어."

이준호 선임이 웃었다.

"그건 제가 잘하는 거네요."

"알아. 그동안 고마웠어. 근데 이제는 혼자 끙끙 앓지 말고, 같이 해결하자."

"... 네. 감사합니다, 실장님."

---

### 7-6. 첫 번째 진짜 대화

세 사람이 옥상에 모였다. 해가 지고 있었다.

"자, 어젯밤에 한 약속 있잖아요."

김철수 실장이 말했다.

"한 달 동안 제대로 해보자는 거요?"

"응. 근데 '제대로'가 뭔지 구체적으로 정하자."

박지은이 노션을 열었다.

"제가 정리해볼게요."

```
[한 달 프로젝트: 팀 리부트]

목표: 서로 믿고 소통하는 팀 만들기

규칙:
1. 주간 1on1 (화요일 오후)
2. 솔직하게 말하기 (감정 로그)
3. 상대방 의견 먼저 듣기
4. "왜?"를 물어보기 (비난 아님)
5. 실수해도 괜찮다고 인정하기

금지 사항:
- "내가 다 해봤어" (벌금 1만원)
- "제 방식이 효율적이에요" (벌금 1만원)
- "그건 니 일이야" (벌금 1만원)

벌금 용도: 팀 회식
```

이준호 선임이 웃었다.

"벌금 모이면 꽤 되겠는데요?"

"그러게. 실장님 많이 내실 듯."

김철수 실장도 웃었다.

"인정. 근데 그만큼 고치겠다는 거지."

세 사람이 손을 포갰다.

"한 달."

"한 달."

"한 달."

커피가 식었다. 하지만 대화는 따뜻해지고 있었다.

---

### 7-7. 작은 변화의 시작

다음 날 아침. 데일리 스탠드업.

분위기가 달랐다.

"자, 오늘 데일리 시작합니다."

김철수 실장이 말했다. 여기까진 평소와 같았다.

"근데 오늘부터 방식을 바꿔볼게요."

팀원들이 놀랐다.

"어떻게요?"

"예전처럼 '어제 한 일, 오늘 할 일' 말고, '어려운 점'도 말해요."

"어려운 점이요?"

"응. 막히는 거, 모르는 거, 힘든 거. 다 말해."

"근데 그러면 무능해 보이지 않을까요?"

김철수 실장이 웃었다.

"나도 그렇게 생각했어. 근데 어려운 점을 숨기면 나중에 더 큰 문제가 되더라고."

박지은이 먼저 손을 들었다.

"저 먼저 할게요. 어제 API 연동하다가 막혔어요. 문서가 이상해서."

"어떤 부분이?"

"응답 포맷이 문서랑 달라요."

김철수 실장이 말했다.

"그거 내가 전에 겪은 적 있어. 나중에 같이 볼까?"

"네!"

예전 같았으면 "그것도 못 해?"라고 했을 것이다. 하지만 지금은 달랐다.

이준호 선임도 말했다.

"저도요. DB 쿼리 최적화하는데 막혔어요."

"어떤 부분?"

"인덱스를 어떻게 잡아야 할지..."

박지은이 끼어들었다.

"저 인덱스 공부 좀 했는데, 같이 봐드릴까요?"

"오, 진짜? 고마워."

대화가 달라졌다. '누가 뭘 못 했는지'가 아니라, '어떻게 같이 해결할지'로.

---

작은 변화였다. 하지만 시작이었다.

커피 한 잔으로 시작된 대화. 그것이 팀을 바꾸기 시작했다.

물론, 아직 갈 길은 멀었다. 한 달 동안 무슨 일이 일어날지는 아무도 몰랐다.

하지만 적어도, 방향은 맞았다.

*함께.*
