# 너드 개발자, 시스템 관리자 권한으로 이세계를 제패합니다

> `root@lumen-territory:~#`
> 영창이 3분 걸리는 이유? 그거 그냥 `sleep(3000)`이야.

## 소개

판교 8년차 서버 개발자가 무중단 배포에 성공한 그날 밤, 과로사로 쓰러진다. 눈을 떠 보니 몰락한 하급 귀족가의 장남 '아르시안 폰 루멘' — 마나 친화력 0으로 '가문의 수치'라 불리던 폐급 소년의 몸이었다.

그러나 상태창 하단에 미친 듯이 스크롤되는 시스템 에러 로그를 발견한 순간, 그는 깨닫는다. 이 세계는 마법도 신비도 아닌 **버그투성이 레거시 소프트웨어**라는 것을. 그리고 `sudo su` 한 줄로 세계 시스템의 **root 권한**까지 손에 넣는다.

비효율적인 영창은 리팩토링하고, 몬스터 스탯은 쿼리로 갈아엎고, 막힌 마나 회로는 디버깅한다. 무능한 창조신 'Gemini'가 짜놓은 똥코드 세계를, 판교에서 단련된 너드 개발자가 처음부터 끝까지 갈아엎는 IT밈 사이다 판타지.

## 등장인물

**아르시안 폰 루멘** — 주인공. 전생은 판교 8년차 서버 개발자. 몰락 남작가 장남이자 세계 시스템의 유일한 root. 비효율을 못 견디는 [안티-레거시 체질]. 시니컬한 실용주의자.

**마리** — 주근깨 메이드. 충직하고 호들갑스럽다. 아르시안의 만행에 매번 기겁하면서도 누구보다 그를 챙긴다.

**세리아** — 기사단장. 멸망한 왕국 출신의 은발 여기사. 마나 회로의 레거시 병목을 아르시안이 디버깅해주며 황금 오러로 각성, 가장 충성스러운 물리 딜러가 된다.

**루멘 남작** — 아르시안의 아버지. 빚과 흉작에 짓눌린 몰락 영지의 영주. 아들의 각성에 감격 반 공포 반.

**Gemini** — 세계 `World System Kernel v2.0`을 짜놓고 손 떼버린 무능한 창조신. 시리즈를 관통하는 떡밥.
