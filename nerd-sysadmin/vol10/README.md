# 10권 — 월드 리부트 v2.0

**세계 코어, 새 마계 파티션, 리부트 이후 루멘 영지**

> "장애는 신성하지 않다. 재현 가능하면 고칠 수 있고, 고칠 수 있으면 배포하면 된다."

## 소개

아르시안은 세계 커널을 전면 재작성하고, 마계를 클린 설치하며, 모두가 마법을 배울 수 있는 오픈 API 세계를 배포한다. 리부트 이후 그는 황제가 아니라 진정한 퇴근을 선택한다.

이 권은 전 권의 사건을 이어받아 더 큰 시스템 계층으로 올라간다. 아르시안은 적을 쓰러뜨리는 데서 멈추지 않고, 왜 그런 적이 반복해서 생기는지 구조를 바꾼다. 전투는 화려하지만 핵심은 늘 같다. 권한을 확인하고, 로그를 남기고, 레거시를 걷어내고, 다음 장애가 사람의 희생으로 막히지 않게 만든다.

## 등장인물

**아르시안 폰 루멘** — 세계 최고 관리자. 마지막으로 세계를 리부트하고 퇴근하려 한다.

**엘리스** — 새 마법 교육 체계의 설계자.

**세리아** — 리부트 중 보안과 이후 세계 방위를 맡는다.

**마리** — 아르시안이 신이 아니라 사람으로 남도록 붙잡아 주는 중심.

**대륙의 사람들** — 더미 데이터가 아니라 새 세계의 실제 사용자.
