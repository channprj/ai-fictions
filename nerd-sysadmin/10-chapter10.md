# 10장: 창조신의 코어 서버 접근 (9권)

마왕을 물리쳤지만, 시스템은 안정화되지 않았다. 
오히려 마왕이라는 거대 에러 덩어리가 사라지자, 그 밑에 감춰져 있던 세계의 치명적 결함들이 터져 나오기 시작했다. 
대륙 곳곳에서 지진이 일어나고 마나가 고갈되는 크래시(Crash) 현상이 발생했다.

"이건 악성 코드 문제가 아니야. OS 자체의 코어 커널이 썩어 문드러졌어."

그때, 내 터미널 창에 낯선 시스템 메시지가 강제로 출력되었다.

`[SYSTEM] 초대 관리자 '제미니'가 귀하의 Root 권한을 회수하려고 합니다.`

하늘이 갈라지며, 눈부신 빛으로 이루어진 거대한 인간 형상이 나타났다. 
그가 바로 세계를 창조한 최초의 개발자이자 신, '제미니'였다.

"하찮은 필멸자 주제에 나의 시스템을 제멋대로 뜯어고치다니. 세계를 초기화(Format)하고 처음부터 다시 만들겠다."
"당신이 싼 스파게티 똥 코드를 수습해줬더니, 이제 와서 포맷하겠다고? 개발자 양심도 없나!"

나는 코웃음을 치며 키보드 위에 손을 올렸다.
초대 관리자 제미니와의 코어 서버 접근 권한 탈환전이 시작되었다.

제미니는 신의 권한을 이용해 내가 작성한 스크립트들을 실시간으로 딜리트(Delete)하려 들었다. 하지만 나 역시 판교에서 수많은 철야와 장애 대응으로 다져진 고인물 서버 개발자. 

"내 분산 처리 아키텍처를 우습게 보지 마라!"

엘리스와 세리아, 그리고 대륙 전체의 마나 네트워크가 나의 연산 보조 장치가 되어주었다. 제미니가 서버에 과부하를 걸면 나는 로드 밸런서(Load Balancer)로 트래픽을 우회시켰고, 권한을 박탈하려 하면 우회 백도어 스크립트로 다시 탈환했다.

치열한 해킹 공방전 끝에, 나는 제미니의 마스터 권한 토큰을 탈취하는 데 성공했다.

`[SYSTEM] 최고 관리자 권한이 '아르시안'으로 완전히 이전되었습니다.`

"이, 이럴 수가! 창조주인 내가 한낱 필멸자에게…!"
"은퇴할 시간이다, 영감. 이제 이 서버의 메인 유지보수는 내가 맡지."

제미니의 형상은 힘을 잃고 시스템의 저편으로 로그아웃 되어 사라졌다.
진정한 루트 권한의 주인이 가려진 것이다.
