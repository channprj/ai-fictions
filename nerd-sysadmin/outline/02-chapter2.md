# 2장: 레거시 코드 리팩토링과 첫 번째 빌드

화면에 출력된 `ignite.mag`의 코드는 가관이었다. 
전생의 나였다면 당장 코드 리뷰에서 "이거 짠 사람 당장 제 자리로 오세요"라고 샤우팅을 날렸을 수준이었다.

```
function cast_ignite() {
    allocate_mana(100);
    wait(3000); // Gather energy from nature
    
    for (int i = 0; i < 10; i++) {
        check_temperature(i);
        increase_heat(5);
    }
    
    if (mana >= 100) {
        release_fire();
    } else {
        fail_magic();
    }
}
```

"이게... 전 대륙의 마법사들이 수십 년간 외우고 다니는 파이어볼의 기본 원리라고?"

나는 어처구니가 없어서 헛웃음을 흘렸다. 
불필요한 마나 할당은 그렇다 치자. 가장 경악스러운 부분은 바로 `wait(3000);`이었다. 
자연에서 기운을 모은다는 명목으로, 캐스팅 시 무조건 3초를 대기하도록 하드코딩되어 있었던 것이다. 마법사들이 영창을 외우며 폼을 잡는 시간이, 사실은 그냥 시스템이 멈춰있는 'sleep' 상태였다니.

게다가 무의미하게 온도를 체크하며 반복하는 for 루프는 CPU, 아니 이 세계 기준으로는 마나 회로의 자원만 갉아먹는 쓰레기 코드였다.

"미치겠네. 이런 레거시 코드로 돌아가는 세계라니. 당장 리팩토링이 시급하다."

손가락이 근질거렸다. 
전생에 더러운 스파게티 코드를 보면 참지 못하고 밤을 새워가며 뜯어고치던 버릇이 도진 것이다. 

`root@lumen-territory:~# vi /sys/magic/fire/ignite.mag`

가상의 터미널에서 vi 에디터를 열었다. 
경고문이 떴다.

`[WARNING] Read-Only file system. You need Root privileges to modify core magic logic.`

"이미 루트 권한은 내 손에 있다고."

나는 가볍게 무시하고 코드를 지워나가기 시작했다. 
먼저 쓸데없는 3초 대기 시간을 날려버렸다. 마나 할당량도 100에서 10으로 대폭 줄였다. 온도를 서서히 올리는 for 루프 역시 필요 없었다. 한 번에 임계점까지 온도를 끌어올리도록 로직을 비동기(Asynchronous) 처리로 바꿨다.

```
async function cast_ignite_v2() {
    allocate_mana(10);
    await instant_ignition(1000); // 1000도까지 즉시 가열
    release_fire();
}
```

"좋아, 코드가 훨씬 깔끔해졌군."

에디터를 저장하고 빠져나왔다. 
수정된 파일의 권한을 확인한 뒤, 나는 침대에서 일어나 창문을 열었다.
바깥은 루멘 영지의 낡고 초라한 뒷마당이었다. 테스트하기엔 딱 좋은 장소였다.

나는 터미널 창에 조심스럽게 명령어를 입력했다.

`root@lumen-territory:~# sudo ./ignite.mag`

명령어를 치고 엔터를 누르는 순간.
영창도, 대기 시간도 없었다. 

"……어?"

**콰아아아아아아앙-!!!**

내 손끝에서 시작된 아주 작은 불씨가, 눈을 깜빡일 새도 없이 집채만 한 거대한 화염 폭풍으로 변해 뒷마당을 휩쓸었다.
비동기 처리로 인해 딜레이 제로(0)로 발동된 데다, 쓸데없는 마나 누수 로직을 잡아버리니 10의 마나만으로도 기존 파이어볼의 수십 배에 달하는 화력이 뿜어져 나온 것이다.

"미친…."

훈련용 허수아비는 물론이고, 뒷마당의 담장 절반이 시커멓게 숯더미로 변해버렸다.
지독한 매연이 피어오르고, 놀란 새들이 하늘로 푸드덕 날아올랐다.

"도, 도련님!! 무슨 일입니까!!"

방금 전 나갔던 메이드 마리가 비명을 지르며 방문을 박차고 들어왔다. 창밖의 처참한 광경을 본 그녀의 다리가 풀리며 주저앉았다.

"적의 습격인가요?! 아니면 마법 폭주?!"
"아, 아니. 그게 아니라…."

나는 머리를 긁적였다.
"그냥… 빌드 테스트를 좀 해봤어."

"빌드… 테… 뭐요?"
"신경 쓰지 마. 버그는 없는 것 같으니."

마리는 공포에 질린 눈으로 나와 창밖의 화염을 번갈아 보았다. 평소 마력 친화력이 제로에 가까워 '마법계의 폐급'이라 불리던 아르시안이, 영창 한 마디 없이 영지 뒷마당을 날려버렸으니 놀랄 만도 했다.

"그보다 마리. 우리 영지에 조만간 몬스터 웨이브가 온다는 소문이 있던데, 사실이야?"

내 물음에 마리는 아직도 덜덜 떨리는 목소리로 대답했다.

"네, 넵! 그래서 영주님께서 용병단을 고용하려 하셨지만… 영지 재정이 바닥나서 아무도 오지 않으려 한다고…."

"재정이 바닥나? 그럼 기사단은?"
"기, 기사단은 세리아 기사단장님을 제외하곤 전부 야반도주했습니다…."

세리아. 전생의 지식과 융합된 아르시안의 기억 속에서 그녀의 얼굴이 떠올랐다. 멸망한 왕국의 기사 출신으로, 갈 곳 없는 그녀를 아버지가 거두어 기사단장으로 임명했었다. 의리는 있지만, 혼자서 몬스터 웨이브를 막는 건 불가능에 가깝다.

'몬스터 웨이브라….'

나는 허공에 떠 있는 터미널 창을 내려다보았다.
루트 권한. 그리고 최적화된 매직 스크립트. 
방금 전의 화력 테스트로 내 가설은 완벽하게 증명되었다. 이 세계의 마법과 법칙은 코드로 이루어져 있고, 나는 그걸 마음대로 뜯어고칠 수 있다.

"마리, 아버님께 전해. 용병단은 필요 없다고."
"네? 하지만 몬스터들이 몰려오면…!"
"걱정 마."

나는 입꼬리를 올려 웃으며, 터미널 창에 새로운 디렉토리를 생성했다.

`root@lumen-territory:~# mkdir /sys/magic/mass_destruction`

"서버 랙(영지)을 방어하는 건, 관리자의 기본 소양이거든."
