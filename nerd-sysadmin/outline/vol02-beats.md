# 2권 비트시트 — 아카데미 스파게티 코드 디버깅 (25화: ep026~050)

> 회차당 약 5,000자. 1인칭 아르시안 시점. 캐논 시드: `outline/04-vol2-academy.md`.
> 1권 결말과 연속(아르시안+마리는 이그니스 아카데미로, 세리아는 영지 수호자로 잔류). 메인 떡밥: 1권의 '비인가 접근(Unauthorized Access)' 세력 = 아카데미 지하 던전 해킹의 배후 = 창조신 Gemini 라인으로 수렴.

## 1막 — 수도 입성 (ep026~030)

- **ep026 · 마차는 느리다**: 1권 후 안정된 영지를 떠나는 길. 세리아의 과잉 충성 배웅, 마리 동행. 수도로 가는 여정, 화려하지만 아르시안 눈엔 엉킨 네트워크 패킷으로 보이는 수도 입성. 끝: 아카데미 정문 도착.
- **ep027 · iptables -A INPUT -j DROP**: 정문 앞 귀족 자제의 텃세. `nmap`으로 취약점 정찰 → `iptables ... -j DROP`으로 마나 회로 차단, 불씨도 못 만드는 청년. "버그나 잡고 와라, 애송아." (캐논 1장)
- **ep028 · 노드 등록**: 입학 등록·기숙사 배정·신입생 환경. 수도/아카데미의 중앙 마나 네트워크 구조를 처음 스캔하며 흥미를 느낌(영지=로컬 서버, 여기=중앙 노드). 마리와 정착 코미디.
- **ep029 · auto_aim.sh**: 입학 실기 시험('마나 표적 맞추기') 안내. 다른 신입생들의 장황한 영창과 대조. 아르시안은 미리 `auto_aim.sh` 작성. 끝: 시험 직전.
- **ep030 · 만점 수석**: `./auto_aim.sh --target crystal_ball --skill magic_missile --mana 5` → 손가락 튕김(Click), 핑-콰앙, 측정기 ERROR. 마나 5·영창 0초·적중 100%. 만점 수석 통과로 일약 유명세. (캐논 2장)

## 2막 — 원칙주의자 엘리스 (ep031~035)

- **ep031 · 수석의 무게**: 수석 타이틀로 쏟아지는 관심·시기. 아카데미 적응 IT밈 코미디. 엘리스(차석)의 따가운 시선 스침(본격 충돌 예고).
- **ep032 · 표준 규격**: 도서관에서 엘리스(금발 트윈테일, 이그니스 공작가 영애, 원칙주의 모범생) 본격 등장. "영창도 술식 궤적도 없는 건 협회 표준 규격 위반!" vs "그 효율 똥망 스파게티 코드?" 논쟁. (캐논 3장 전반)
- **ep033 · kill -9 PID_ELICE_FIREBALL**: 격분한 엘리스가 불꽃을 피우자 `kill -9`로 프로세스 강제 종료, 불꽃 소멸. "구조가 무거우니 죽이기도 쉬워. 리팩토링부터 배워 와." 신념이 흔들리는 엘리스. (캐논 3장 후반)
- **ep034 · 132절 영창**: 엘리스 캐릭터 심화 — 공작가 영애의 압박, 완벽주의 콤플렉스, 132절 영창에 바친 자부심. 아르시안과의 티키타카, 그녀의 분한 호기심. 관계의 씨앗.
- **ep035 · 이상 트래픽**: 권위적 마법학계·보수파 소개(하인즈 교수 빌드업). 아르시안이 중앙 마나 네트워크를 들여다보다 미세한 이상 트래픽/비인가 접근 흔적의 잔향을 감지(1권 떡밥과 연결, 확정 X·의심만). 끝: 다음 날 하인즈 수업.

## 3막 — 교수 참교육 (ep036~040)

- **ep036 · 메모리 누수입니다만**: 하인즈 교수의 마법 기초학 수업, 재앙급 술식. 아르시안이 손 들고 "4번째 줄 마나 할당에서 메모리 누수(Mana Leak) 나지 않습니까?" 교수 분노. (캐논 4장 전반)
- **ep037 · optimize_mana()**: 칠판으로 나가 3줄짜리 `optimize_mana()`로 재작성 → 강의실을 밝히는 빛. 100줄 똥코드 대비 10배 속도·1/10 마나. 하인즈 주저앉음. 공개 코드 리뷰 참교육. (캐논 4장 후반)
- **ep038 · 풀 리퀘스트**: 참교육 여파 — 명성 폭발 + 보수파 교수·귀족 적대 형성. 학생들의 디버깅 의뢰 쇄도(아르시안의 영향력 확대). 엘리스의 동요 심화.
- **ep039 · 사이드카**: 엘리스가 자존심 누르고 몰래 관찰/접근, 아르시안의 '코드'를 곁눈질로 배우려 함. 둘의 관계 진전(라이벌→호기심→끌림). 코믹+달달.
- **ep040 · Unauthorized Access**: 아르시안이 아카데미 시스템을 깊이 파다가 지하 던전 봉인 시스템의 취약점·비인가 접근 흔적을 발견(1권의 그 수법과 동일 시그니처). 불길한 전조. 끝: 그날 밤 비상 사이렌.

## 4막 — 지하 던전 비인가 접근 (ep041~045)

- **ep041 · 비상 사이렌**: `[경고] 아카데미 지하 던전 봉인 해제! 비인가 접근 발생!` 통제 구역의 고대 마수들. 아카데미 대혼란. (캐논 5장 전반)
- **ep042 · DDoS**: 아르시안 로그 분석 — `[WARN] Multiple requests from unknown IP targeting Seal_Gate_01`, DDoS로 봉인 서버 다운. "또 너냐." 1권 비인가 접근 세력과 동일 수법임을 직감(Gemini 라인 떡밥 강화).
- **ep043 · 고립**: 던전에서 마수 탈출, 아카데미 위기. 순찰 중이던 엘리스가 지하에 고립됐다는 소식. 아르시안이 기숙사를 박차고 출동.
- **ep044 · 케르베로스**: 지하 던전. 엘리스가 홀로 거대 보스 케르베로스(머리 셋)와 대치. 마법 저항 극악으로 정통 마법이 안 통해 위기. 아르시안 도착. (캐논 5장 후반)
- **ep045 · MAX 스탯**: "비켜, 원칙주의자." 케르베로스의 방어·마법저항 MAX. 단, 그 스펙이 비정상적으로 외부 주입된 값임을 아르시안이 간파(1권 보스와 동일 패턴 — 배후 동일범 단서). 대치 긴장.

## 5막 — 원펀치 디버깅과 함락 (ep046~050)

- **ep046 · UPDATE monsters SET defense=0**: "방어력 MAX? 프론트엔드 표기 오류야. 백엔드 DB를 직접 건드리면 그만." `UPDATE monsters SET defense=0, magic_resist=0 WHERE name='Cerberus';` → 원펀치로 케르베로스 소멸. 압도적 사이다. (캐논 6장)
- **ep047 · 함락**: 넋 나간 엘리스 "너… 대체 정체가 뭐야?" "그냥 흔한 서버 관리자." 엘리스가 코드·존재 자체에 매료되어 조수(QA 테스터) 자청. 히로인 합류. (캐논 6장)
- **ep048 · 같은 네트워크**: 던전 해킹의 흔적 추적 → 배후가 수도/아카데미/황실, 그리고 그 위 'Creator-level'과 한 라인으로 이어진다는 정황(1권 ep025 떡밥 발전). 적 세력(암흑 마법사/수도 권력)의 윤곽.
- **ep049 · 입지**: 여파 정리 — 아카데미 내 입지 확립, 엘리스+마리 히로인 조합 코미디, 하인즈·보수파와의 냉전. 세리아(영지)와의 원격 연락 한 컷. 3권(영지전·빌드 자동화) 빌드업.
- **ep050 · 2권 결말**: 새 사건/소환(영지 관련 위협 or 더 큰 음모)으로 3권 예고. Gemini 메타플롯 한 단계 진전(또 한 번의 직접 접선 or 결정적 단서). 강한 훅 + "(2권 끝)".
