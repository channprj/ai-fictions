# 1권 비트시트 — 튜토리얼 및 권한 획득 (25화)

> 회차당 약 5,000자. 1인칭 아르시안 시점. 캐논 시드는 `outline/00~03`, `outline/seed-episodes/vol01-ep00*`.
> 1권은 기존 6장 압축본을 25화로 확장한다. 캐논 사건·결말은 반드시 보존하고, 사이/전후로 장면·대사·세계관·코믹/사이다 비트를 추가한다.

## 1막 — 전생과 root 권한 (ep001~005)

- **ep001 · Exception in thread "main": OverworkError**
  판교 야근→무중단 배포 성공→심장마비. 아르시안 폰 루멘으로 깨어남. 메이드 마리, 낡은 영주성. 빙의 정보 유입. "상태창" → 형편없는 스탯 + 하단에 스크롤되는 에러 로그(NullPointerException 등) 발견. *시드: `00-prologue.md` 전반.* 끝: 로그 영역을 누르려는 순간.

- **ep002 · sudo su**
  로그 영역 더블 탭 → CLI 진입(`Welcome to World System Kernel v2.0 - Creator: Gemini`). `whoami`(arsian, user). `sudo su` → 초기 비밀번호 추리(admin/1234/password/gemini 실패 → `root` 성공). 프롬프트 `$`→`#`. *시드: `00-prologue.md` 후반.* 끝: "이거, 사고 친 것 같은데."

- **ep003 · ls -al /sys/magic/**
  몰락 남작가의 현실·아르시안의 과거 기억 정리, [안티-레거시 체질]의 의미 자각. 마리와의 첫 제대로 된 대화(영지 사정 일부). `ls /sys/magic/` 으로 세계가 코드/디렉토리로 구성됨을 확인. *확장(신규 장면).* 끝: ignite.mag를 열어보기로.

- **ep004 · 레거시 코드 리뷰**
  `cat /sys/magic/fire/ignite.mag` → 끔찍한 스파게티 코드(불필요 `sleep(3000)`, 의미 없는 for 루프, 과도한 마나 할당) 발견. `vi`로 3줄 async 리팩토링. *시드: `01-chapter1.md`, `02-chapter2.md` 코드 부분.* 끝: 테스트 결심.

- **ep005 · 첫 번째 빌드**
  뒷마당에서 `sudo ./ignite.mag` 실행 → 콰앙, 담장 초토화. 마리 기겁. 루멘 남작·기사장 난입, '무영창 마법'에 경악(마나 친화력 0이던 아들이). 아르시안 "깨달음을 얻었다"로 둘러댐. *시드: `01-chapter1.md` 후반.* 끝: 가족의 충격 + 자신의 능력 확신.

## 2막 — 영지의 현실과 시스템 장악 (ep006~011)

- **ep006 · 전부 똥코드잖아**
  다른 마법들도 디버깅(힐링 마법의 마나 누수=메모리 릭, 보호막의 `sleep` 등) 발견. 안티-레거시 체질의 정체(비효율 연산 거부) 확정. 아르시안의 '리팩토링 본능' 캐릭터 확립. *확장.* 끝: 영지 전체 시스템이 궁금해짐.

- **ep007 · 적자 영지의 영주**
  루멘 남작과의 부자 대화 — 가문의 빚, 흉작, 떠나는 영지민. 아버지의 무력감과 아들에 대한 미안함. 아르시안이 영지 인프라(`/sys/territory/`)를 들여다봄. *확장(드라마: 아버지 심화).* 끝: 영지를 직접 고치기로.

- **ep008 · cron으로 굴러가는 영지**
  관개·우물 마법의 버그 수정, 농업/수자원 자동화(`cron` 같은 반복 마법 스크립트). 영지민들의 공포 반 경이 반 반응. 첫 '치적'. *확장.* 끝: 그때 들려오는 몬스터 웨이브 소문.

- **ep009 · 몬스터 웨이브가 온다**
  마리에게서 상황 청취: 용병단은 재정 파탄으로 거절, 기사단은 세리아만 남고 야반도주. 아르시안 "용병 필요 없다" 선언으로 마리·남작 당황. *시드: `02-chapter2.md` 후반.* 끝: `mkdir /sys/magic/mass_destruction`.

- **ep010 · mass_destruction/**
  방어 스크립트 설계 착수. 영지 하인·주민들과의 코믹 상호작용(개발자 밈으로 동문서답). 세리아 단장에 대한 소문/예고. *확장.* 끝: 세리아를 호출.

- **ep011 · 소드마스터의 스탯**
  세리아 등장 — 은발 여기사, 멸망 왕국 출신, 비장한 충성. `cat seria_stats.dat` → 물리 스탯 최상위인데 **마나 응집 효율 15% 병목** 발견. *시드: `03-vol1-climax.md` 3장.* 끝: "내가 디버깅 좀 해줄게."

## 3막 — 세리아 디버깅과 웨이브 전조 (ep012~017)

- **ep012 · 병목 현상**
  오러가 끊기는 원인 = 출신 왕국의 구식 하드코딩 로직 병목임을 진단. 세리아의 사연(멸망 왕국, 갈 곳 없던 그녀를 남작이 거둠) 백스토리. 반신반의하는 세리아. *시드: 3장 + 확장.* 끝: 매크로 적용 직전.

- **ep013 · 펌웨어 업데이트**
  `./apply_buff_macro.sh --target seria --optimize-aura true --infinite-loop on` → 황금빛 각성, 오러 블레이드가 성벽을 관통. 마력 누수율 0%. 세리아 무릎 꿇고 충성 맹세. *시드: `03-vol1-climax.md` 4장.* 끝: 최강 물리 딜러 획득.

- **ep014 · 무한 루프 오러**
  각성한 세리아의 시연·코믹한 과잉 충성. 두 사람의 유대 형성. 아르시안의 (드물게) 뿌듯함. *확장.* 끝: 시스템 경보 알림이 뜸.

- **ep015 · 스폰 레이트 급증**
  모니터링 로그에 `[WARN] 몬스터 스폰 레이트 급증` 경고. 어둠의 숲 던전 이상. 원격 로그 분석으로 위협 규모 파악(수천 마리). *확장 + 떡밥(누군가의 개입 암시).* 끝: 방어 시스템을 짜기로.

- **ep016 · 오토 터렛 디펜스 설계**
  `auto_turret_defense.sh` 작성 — 비동기 파이어볼 2.0 기관총, 마법진 좌표 계산. 세리아·마리와 작전 회의, 영지민 대피 계획. *확장(클라이맥스 빌드업).* 끝: 배치 완료.

- **ep017 · 폭풍 전야**
  웨이브 직전 밤의 긴장·인간 드라마: 마리의 걱정, 남작의 불안, 세리아의 각오. 아르시안은 캠핑 의자를 세팅하는 여유(코미디). *확장.* 끝: 지평선 너머 몬스터 울음소리.

## 4막 — 몬스터 웨이브 방어전 클라이맥스 (ep018~023)

- **ep018 · 웨이브 개전**
  수천 마리 오크·고블린·트롤 쇄도. 세리아 황금 오러로 돌진, 전장을 휩쓺. 압도적 액션. *시드: `03-vol1-climax.md` 5장.* 끝: 아르시안 "나도 구경만 할 순 없지."

- **ep019 · 오토 터렛 가동**
  수백 개 마법진 → 파이어볼 2.0이 비처럼 쏟아짐(약점 정밀 타격). 아르시안 캠핑 의자에서 모니터링하는 사이다. *시드: 5장.* 끝: 전세 압도 중 변수 발생.

- **ep020 · 핫픽스**
  보스급 몬스터(트롤 군주/변종) 등장 → 터렛 방어 일부 돌파, 위기 고조. 실시간 디버깅/스크립트 핫픽스로 대응. *확장(클라이맥스 갈등).* 끝: 적의 정체에 대한 의문.

- **ep021 · 비인가 접근**
  웨이브가 자연발생이 아님 — 로그에 던전 봉인에 대한 **비인가 접근(Unauthorized Access)** 흔적. 배후 인간(적 마법사/세력)의 존재 암시. *확장 떡밥(후속 권 빌런 라인).* 끝: 분노한 아르시안이 직접 나섬.

- **ep022 · root의 분노**
  아르시안이 직접 전선에 개입 — root 권한으로 몬스터 속성 무력화(`UPDATE monsters SET defense=0`, `kill -9`) + 광역 스크립트. 세리아와의 콤비 피니시로 보스 처치. *확장 + 시드 톤.* 끝: 전장 정리.

- **ep023 · BUILD SUCCESS**
  새벽의 승리. `[SYSTEM] 몬스터 웨이브 방어 성공 / 관리자 권한이 루멘 영지 전체로 확장`. 무사상자 전멸, 영지민 환호. *시드: `03-vol1-climax.md` 6장 전반.* 끝: 영지가 '내 사유 서버'가 됨.

## 5막 — 권한 확장과 결말 (ep024~025)

- **ep024 · Private Server**
  여파와 영지의 변화. 루멘 영지 = 아르시안의 Private Server. 아버지·세리아·마리·영지민의 달라진 시선. 소문이 수도로 확산. 캐릭터 정리·잔잔한 마무리. *시드: 6장 + 확장.* 끝: 수도에서 온 편지 한 통.

- **ep025 · 입학 권유장 (1권 결말)**
  이그니스 아카데미 입학 권유장 도착. 아르시안 "거긴 또 얼마나 더러운 스파게티 코드가 굴러다닐까. 디버깅할 맛 나겠군." 창조신 Gemini에 대한 더 큰 비밀 암시(메타플롯 떡밥). 2권 예고. *시드: 6장 결말.* 끝: 2권으로 잇는 강한 훅.
