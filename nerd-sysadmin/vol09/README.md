# 9권 — 창조신의 코어 서버 접근

**세계 관제 센터와 Creator-level 코어 서버**

> "장애는 신성하지 않다. 재현 가능하면 고칠 수 있고, 고칠 수 있으면 배포하면 된다."

## 소개

마왕 소멸 후 세계 커널이 붕괴한다. Gemini는 세계 초기화를 선언하고 아르시안의 root 권한을 회수하려 한다. 아르시안은 대륙 연합 분산 방어망으로 버티며 창조신의 코어 서버를 역해킹한다.

이 권은 전 권의 사건을 이어받아 더 큰 시스템 계층으로 올라간다. 아르시안은 적을 쓰러뜨리는 데서 멈추지 않고, 왜 그런 적이 반복해서 생기는지 구조를 바꾼다. 전투는 화려하지만 핵심은 늘 같다. 권한을 확인하고, 로그를 남기고, 레거시를 걷어내고, 다음 장애가 사람의 희생으로 막히지 않게 만든다.

## 등장인물

**아르시안 폰 루멘** — 창조신에게서 세계 최고 관리자 권한을 탈환하려는 운영자.

**Gemini** — 세계를 만든 초대 관리자. 포맷으로 문제를 해결하려 한다.

**엘리스** — 대륙 오픈소스 커뮤니티를 전시 체제로 지휘한다.

**세리아** — 신의 공격을 막는 물리 방어선.

**마리** — 사람들이 더미 데이터가 아님을 아르시안에게 계속 상기시킨다.
