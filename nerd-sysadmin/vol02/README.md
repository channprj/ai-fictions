# 2권 — 아카데미 스파게티 코드 디버깅

영지를 사유 서버로 만든 아르시안이 대륙 최고의 마법 교육 기관 **이그니스 아카데미**로 향한다. 귀족 텃세를 `iptables` 한 줄로 제압하고, `auto_aim.sh`로 실기 만점 수석을 찍고, 원칙주의 공작가 영애 **엘리스**의 132절 영창을 `kill -9`로 종료시키고, 권위적인 하인즈 교수를 공개 코드 리뷰로 참교육한다. 그리고 지하 던전의 **비인가 접근** 사건 — 1권의 그 침입자, 그리고 창조신 Gemini로 이어지는 실타래가 조금씩 드러난다.
