# 7권 — 다크웹 방어전

**대륙 관제 센터와 차원의 틈새**

> "장애는 신성하지 않다. 재현 가능하면 고칠 수 있고, 고칠 수 있으면 배포하면 된다."

## 소개

암흑 마법사 집단 다크웹이 대륙 네트워크를 랜섬웨어로 공격한다. 엘리스가 화이트 해커 커뮤니티를 이끌고, 아르시안은 차원의 틈새에 숨은 메인 서버를 역추적해 궤멸시킨다.

이 권은 전 권의 사건을 이어받아 더 큰 시스템 계층으로 올라간다. 아르시안은 적을 쓰러뜨리는 데서 멈추지 않고, 왜 그런 적이 반복해서 생기는지 구조를 바꾼다. 전투는 화려하지만 핵심은 늘 같다. 권한을 확인하고, 로그를 남기고, 레거시를 걷어내고, 다음 장애가 사람의 희생으로 막히지 않게 만든다.

## 등장인물

**아르시안 폰 루멘** — 대륙 보안 인시던트 총괄. 방어보다 근원지 제거를 선택한다.

**엘리스** — 화이트 해커 네트워크의 지휘자. 대륙 방화벽을 구축한다.

**세리아** — 차원의 틈새 침투를 맡는 물리 서버 철거 담당.

**암흑 관리자 모르그** — 고대 관리자 후예이자 다크웹의 핵심 운영자.

**마리** — 공포에 질린 사용자들을 안정시키며 안내문을 고친다.
