# 1장: 스파게티 코드로 빚어진 세계

"도련님! 이제 좀 괜찮으십니까? 의원님을 모셔 올까요?"

침대 옆에서 호들갑을 떠는 주근깨 메이드, 마리. 그녀의 목소리가 귓전을 때렸지만 내 신경은 온통 허공에 떠 있는 검은 터미널 창에 쏠려 있었다. 

`root@lumen-territory:~#`

가슴이 벅차올랐다. 서버 개발자로 구르면서 가장 짜릿한 순간은 트래픽을 견뎌냈을 때도, 보너스가 들어왔을 때도 아니었다. 바로 건드려서는 안 될 시스템의 가장 깊숙한 곳, 루트(Root) 권한을 손에 쥐었을 때였다.

"마리라고 했지? 난 괜찮아. 며칠 누워 있었더니 머리가 좀 멍한 것뿐이니까. 혼자 쉬게 나가봐."
"하지만 영주님께서 도련님이 깨어나시면 바로 보고하라고..."
"아버님껜 내가 나중에 직접 갈 테니 걱정 말고. 어서."

마리가 못내 아쉬운 표정으로 방을 나갔다. 문이 닫히는 소리를 확인하자마자, 나는 곧바로 터미널 창에 명령어들을 때려 넣기 시작했다.
이 세계의 구조를 파악하는 것이 우선이었다.

`root@lumen-territory:~# ls -al /sys/magic/`

엔터를 치자마자 수백 줄의 디렉토리와 파일 목록이 주르륵 쏟아졌다.
`fire/` `water/` `earth/` `wind/` `light/` `dark/`...
기본적인 원소 마법들이 폴더별로 정리되어 있었다.

"호오. 제법 객체지향적으로 짜놨는데?"

나는 가장 만만해 보이는 화염 마법 디렉토리로 이동해 파일을 열어보았다.

`root@lumen-territory:~# cat /sys/magic/fire/ignite.mag`

그러자 화면 가득 알 수 없는 고대어와 수학 기호가 뒤섞인 코드가 출력되었다. 아르시안의 뇌에 희미하게 남아있는 지식을 빌리자면, 이것은 이세계 마법사들이 수년에 걸쳐 외우고 수련하는 '주문 영창'의 원리였다.
그런데...

```
function cast_Ignite(caster) {
    if (caster.mana < 10) return FAILED;
    
    // 마나 주변의 불의 정령을 모은다
    gather_spirits("fire", 1);
    gather_spirits("fire", 1);
    gather_spirits("fire", 1);
    gather_spirits("fire", 1);
    gather_spirits("fire", 1);
    // (중략 - 동일한 코드 50줄 반복)
    
    // 온도를 올린다
    let temp = 20;
    temp = temp + 1;
    temp = temp + 1;
    // (중략 - 300번 반복)
    
    create_flame(temp);
    caster.mana -= 10;
    return SUCCESS;
}
```

"...돌겠네, 진짜."

나는 나도 모르게 이마를 짚었다.
반복문(For loop) 하나면 끝날 것을, 불의 정령을 모은다는 함수를 일일이 수십 번 복사해서 붙여넣고 있었다. 온도 올리는 로직은 더 가관이었다. 1도씩 300번을 더하고 있다니. 

"어떤 미친놈이 코드를 이따위로 짠 거야?"

이러니 이세계 마법사들이 불꽃 하나 만들어내는데 영창을 3분씩이나 웅얼거리고 자빠졌지. 이건 마법이 아니라 그냥 컴파일러를 학대하는 수준의 스파게티 코드, 아니, 레거시 똥덩어리였다.

이뿐만이 아니었다. 다른 마법 파일들도 열어보니 가관이었다.
쓸데없는 딜레이 펑션(`sleep(5000)`)이 걸려 있는 보호막 마법, 마나를 회수하지 않아 심각한 메모리 누수(Memory Leak)를 일으키는 힐링 마법 등...
이 세계 시스템의 근간은 당장 서버가 터져도 이상하지 않을 만큼 아슬아슬하게 돌아가고 있었다.

"이게... 세계의 진실?"

아르시안, 아니 내가 빙의한 이 소년이 왜 마나 친화력이 제로인지도 알 것 같았다. 
아르시안의 신체 정보 로그를 까보니, 기본적으로 시스템의 불필요한 연산을 본능적으로 거부하는 [안티-레거시 체질]이 패시브로 걸려 있었다. 마법사들이 똥 덩어리 코드를 영창할 때, 그의 몸은 "이딴 비효율적인 연산은 처리할 수 없어!"라며 마나 섭취를 뱉어내고 있었던 것이다.

나는 씩 웃었다.
"비효율적인 마법 따위, 내가 전부 리팩토링(Refactoring) 해주마."

나는 터미널에서 가장 익숙한 텍스트 에디터 명령어를 쳤다.

`root@lumen-territory:~# vi /sys/magic/fire/ignite.mag`

구시대의 쓰레기 같은 하드코딩을 전부 지워버렸다. 
그리고 단 세 줄로, 세상에서 가장 완벽하고 효율적인 '파이어볼' 로직을 새로 짰다.

```
function cast_Ignite_optimized(caster) {
    if (caster.mana < 2) return FAILED;
    create_flame(target_temp: 500, scale: "MAX");
    caster.mana -= 2;
    return SUCCESS;
}
```

쓸데없는 정령 소환 애니메이션 생략, 온도 상승 딜레이 생략. 마나 소모량은 5분의 1로 줄이면서 온도는 하드코딩으로 최대치까지 올려버렸다.
저장 후 닫기. `:wq!`

"자, 그럼 테스트를 해볼까."

침대에서 일어난 나는 방 한구석에 있는 낡은 철제 화로를 향해 손을 뻗었다.
본래대로라면 눈을 감고, 두 손을 모아, 대지의 정령에게 불을 구걸하는 3분짜리 영창을 해야겠지만...
나는 허공에 뜬 터미널 창에 간단한 실행 명령어를 입력했다.

`root@lumen-territory:~# ./sys/magic/fire/ignite.mag`

**퍼엉-!!**

가벼운 펑 소리가 아니라, 폭탄이 터지는 듯한 굉음이 방 안을 울렸다.
눈앞에서 시퍼런 불기둥이 치솟더니 철제 화로를 순식간에 녹여버리고, 돌로 된 벽면까지 시커멓게 태워버렸다. 방 안의 온도가 급격히 상승해 숨이 턱 막힐 지경이었다.

"미친, 스케일을 너무 크게 잡았나?"

마나 소모량 '2'. 레벨 1짜리인 내 전체 마나가 '1'이니까, 시스템 오버플로우가 나거나 마나가 모자라 캐스팅이 실패해야 정상이지만, 나는 Root였다. 마나가 모자라면 마이너스 통장처럼 시스템 코어에서 마나를 강제로 끌어다 써버린 것이다.

우당탕탕!
방 밖에서 다급한 발소리가 들리더니 벌컥 문이 열렸다.

"도, 도련님!! 무슨 일입니까! 폭발음이... 히익!"

뛰어 들어온 영지의 기사장과 마리, 그리고 아버지인 루멘 남작이 녹아내린 화로와 불탄 벽면을 보고 경악을 금치 못했다.

"아, 아르시안! 대체 이게 무슨 짓이냐! 설마... 네가 마법을 쓴 것이냐?!"

루멘 남작의 눈이 튀어나올 듯 커졌다. 마나 부적응자로 낙인찍혔던 아들이, 고위 마법사나 쓸 법한 위력의 마법을 영창도 없이 무영창으로 쏴버렸으니 기절할 노릇일 것이다.

나는 등 뒤로 슬쩍 터미널 창을 숨기며 태연하게 대답했다.
"아... 네. 앓아눕고 났더니, 갑자기 깨달음을 좀 얻어서요."

이 세계의 무능한 개발자(창조신)가 짜놓은 스파게티 코드를, 내가 완벽하게 최적화해주겠다는 깨달음 말이다.
너드 개발자의 이세계 물리치료, 아니 코드 치료가 시작되는 순간이었다.
