# 5권 — 제국 네트워크 병합 및 분산 처리

**북방 국경, 신성 제국 수도, 황실 서버룸**

> "장애는 신성하지 않다. 재현 가능하면 고칠 수 있고, 고칠 수 있으면 배포하면 된다."

## 소개

신성 제국이 왕국의 오픈 API를 이단으로 규정하고 십자군을 파병한다. 아르시안은 Holy DB의 패턴을 분석하고 기도 트래픽을 가로채 제국 네트워크를 병합한다.

이 권은 전 권의 사건을 이어받아 더 큰 시스템 계층으로 올라간다. 아르시안은 적을 쓰러뜨리는 데서 멈추지 않고, 왜 그런 적이 반복해서 생기는지 구조를 바꾼다. 전투는 화려하지만 핵심은 늘 같다. 권한을 확인하고, 로그를 남기고, 레거시를 걷어내고, 다음 장애가 사람의 희생으로 막히지 않게 만든다.

## 등장인물

**아르시안 폰 루멘** — 왕국 클라우드의 운영자. 신성 제국의 폐쇄 네트워크를 병합하려 한다.

**엘리스** — 패킷 분석과 협상 문안을 동시에 처리하는 수석 조수.

**세리아** — 국경 방어와 황궁 돌입을 맡는 최전방 딜러.

**대주교 베르단** — 신성 제국 십자군의 지휘자. 신앙 트래픽을 무기로 쓴다.

**황제 카이로스** — 신성 DB를 사유화한 제국의 지배자.
