# 6권 — 오픈소스와 마법 생물 패치

**드워프 지하 도시 아이언 포지와 엘프의 세계수 숲**

> "장애는 신성하지 않다. 재현 가능하면 고칠 수 있고, 고칠 수 있으면 배포하면 된다."

## 소개

통일 네트워크가 이종족 시스템과 충돌한다. 아르시안은 드워프 마도구 펌웨어를 갱신하고, 랜섬웨어에 감염된 세계수 DB를 오픈소스로 전환해 대륙 개발자 커뮤니티와 함께 구한다.

이 권은 전 권의 사건을 이어받아 더 큰 시스템 계층으로 올라간다. 아르시안은 적을 쓰러뜨리는 데서 멈추지 않고, 왜 그런 적이 반복해서 생기는지 구조를 바꾼다. 전투는 화려하지만 핵심은 늘 같다. 권한을 확인하고, 로그를 남기고, 레거시를 걷어내고, 다음 장애가 사람의 희생으로 막히지 않게 만든다.

## 등장인물

**아르시안 폰 루멘** — 통합 대륙 네트워크의 관리자. 서드파티 호환성 문제를 처리한다.

**엘리스** — 오픈소스 프로젝트의 메인 메인테이너로 성장한다.

**세리아** — 기술 현장 보호와 엘프 숲 보안 담당.

**드워프 족장 그롬** — 아이언 포지의 공장과 마도구를 지켜달라고 요청한다.

**엘프 특사 리아나** — 세계수의 시듦을 보고한 숲의 대표.
