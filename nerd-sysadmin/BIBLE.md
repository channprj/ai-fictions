# 시리즈 바이블 — 너드 개발자, 시스템 관리자 권한으로 이세계를 제패합니다

> 집필 일관성을 위한 마스터 문서. 모든 회차는 이 바이블을 기준으로 쓴다.

## 로그라인

판교 8년차 서버 개발자가 과로사 후 이세계로 전생, 세계 시스템의 **root 권한**을 손에 넣고 버그투성이 세계를 리팩토링하며 최강의 관리자로 군림하는 IT밈 사이다 판타지.

## 톤 & 스타일

- **1인칭 주인공(아르시안) 시점.** 빠른 전개, 사이다·먼치킨.
- IT/개발자 밈을 마법·세계 법칙에 직접 매핑(아래 카탈로그).
- 코믹하고 가벼운 입담. 가벼운 욕설("시발", "미친" 등) 허용 — **성인 웹소설 등급**.
- **회차당 약 5,000자(±10%).** 매 화 끝에 훅(떡밥/사이다)으로 다음 화를 당긴다.
- 시스템 출력은 터미널/CLI 톤으로, 반드시 **코드블록(```)** 안에 넣는다. 예:
  ```
  root@lumen-territory:~# whoami
  arsian
  [SYSTEM] 관리자(Root) 권한이 확장되었습니다.
  ```
  (인라인 백틱 안에 `$`/`#`를 섞으면 렌더가 깨지므로 코드블록을 쓸 것.)

## 세계관 / 시스템 규칙

- 세계 = `World System Kernel v2.0`, **Creator: Gemini**(세계를 부실하게 짠 무능한 창조신 = 메타 빌런, 10권까지 관통하는 떡밥).
- **마법 = 코드.** 영창 = 비효율 레거시(불필요한 `sleep`/반복문/메모리 누수). 마나 = 연산 자원.
- 상태창 = UI 패널. 그 하단의 작은 로그 영역을 더블 탭하면 → CLI(터미널) 진입.
- 아르시안만 **root**. 일반인은 user 계정(영창 = 느리고 비효율적인 호출). root는 마법 로직·몬스터 스탯·세계 법칙을 직접 수정(`vi`, `UPDATE`, `kill -9`, `iptables`, `cron` 등).
- **긴장 유지용 제약:** 과도한 권한 사용 시 마나 오버플로우(코어에서 강제 차용 → 신체 부담/리스크), 일부 시스템 영역은 추가 인증 필요, 창조신 레벨 로직은 아직 접근 불가(후속 권 떡밥).

## IT밈 카탈로그 (마법 매핑)

- 스파게티 코드/레거시 → 기존 마법 영창
- 리팩토링 → 마법 최적화(영창 제거, 3줄화)
- `async`/`await` → 무영창 즉시 시전
- `for` 루프 최적화 · `sleep` 제거 → 캐스팅 속도 폭증
- 메모리 누수 → 마나 누수(힐링 마법 등)
- `nmap` → 정찰 · `iptables ... -j DROP` → 적 마나 차단
- `kill -9` → 마법/프로세스 강제 종료
- `UPDATE ... SET` 쿼리 → 몬스터/대상 스탯 직접 수정
- `cron`/스크립트 → 자동화 마법(관개·오토 터렛 디펜스)
- 권한(`root`/`sudo`) → 신적 개입
- CI/CD·빌드 → 마법 시전/배포

## 집필 규칙

- 파일: `vol{NN}/ep{NNN}.md`. 회차 번호는 권을 가로질러 **연속**(1권 = ep001~025).
- 각 화 첫 줄: `# 제 N화: {부제}`
- `outline/`(시놉시스)와 `outline/seed-episodes/`(기존 집필분)는 **캐논 시드**. 사건·고유명사·캐릭터 일관성을 반드시 유지한다.
- 고유명사 표기 고정: 아르시안 폰 루멘 / 루멘 영지(`lumen-territory`) / 마리 / 세리아 / 루멘 남작 / Gemini(창조신) / 이그니스 아카데미 / 엘리스 / 하인즈.
