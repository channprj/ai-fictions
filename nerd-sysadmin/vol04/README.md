# 4권 — 왕국 시스템 아키텍처 재설계

**왕도와 왕성 지하 중앙 마나 제어소**

> "장애는 신성하지 않다. 재현 가능하면 고칠 수 있고, 고칠 수 있으면 배포하면 된다."

## 소개

남부 영지를 안정화한 아르시안이 왕도 중앙 코어에 접근한다. 국왕 독살 음모, 제1왕자의 반란, 낡은 왕실 네트워크 장악을 거쳐 왕국 전체를 분산 아키텍처로 재설계한다.

이 권은 전 권의 사건을 이어받아 더 큰 시스템 계층으로 올라간다. 아르시안은 적을 쓰러뜨리는 데서 멈추지 않고, 왜 그런 적이 반복해서 생기는지 구조를 바꾼다. 전투는 화려하지만 핵심은 늘 같다. 권한을 확인하고, 로그를 남기고, 레거시를 걷어내고, 다음 장애가 사람의 희생으로 막히지 않게 만든다.

## 등장인물

**아르시안 폰 루멘** — 왕국 중앙 네트워크를 장악하려는 root 관리자. 정치보다 로그를 먼저 본다.

**엘리스** — 왕도 귀족 인맥과 표준 규격 지식으로 중앙 코어 분석을 돕는 조수.

**세리아** — 왕성 경호와 반란군 제압을 맡는 물리 보안 책임자.

**마리** — 왕도에서도 아르시안의 생활 리듬과 인간적인 불안을 붙잡아 주는 메이드.

**국왕 아르투르** — 마나 독에 쓰러진 노왕. 왕국 코어 권한을 아르시안에게 위임한다.

**제1왕자 레온** — 중앙 권한을 사유화하려는 반란의 주모자.
