# 제 20화: 핫픽스

[← 이전: 19화: 오토 터렛 가동](./ep019.md) | [시리즈홈](../README.md) | [목차](./README.md) | [다음: 21화: 비인가 접근 →](./ep021.md)

파이어볼 2.0이 비처럼 쏟아지고 있었다.

마법진 512개가 0.02초 간격으로 캐스팅을 토해냈다. 비동기 큐에 쌓인 적 좌표가 들어오는 족족, 가장 가까운 유휴 마법진이 그놈을 물어다 약점에 꽂았다. 오크는 미간, 고블린은 목, 트롤은 무릎 관절. 타깃팅 로직이 약점 좌표를 정밀하게 계산해서 때리니, 한 발에 한 마리씩 깔끔하게 정리됐다.

전장은 도살장이었다. 정확히는, 잘 튜닝된 로드밸런서였다.

```
root@lumen-territory:~# tail -f /var/log/wave.log
[INFO] 처리량: 1,420 kills/min
[INFO] 큐 적체: 0 (실시간 소화 중)
[INFO] 마법진 가동률: 87% (여유 있음)
[OK] 전 구간 그린. 방어선 무결.
```

1분에 1,420마리. 들어오는 트래픽을 적체 없이 실시간으로 갈아버리고 있었다. 큐가 비어 있다는 건, 들어오는 족족 처리되고 있다는 뜻이다. 데이터센터로 치면 요청이 폭주하는데도 응답 지연 0인 상태. 꿈의 수치였다. 캠핑 의자에 등을 깊이 기대고, 나는 식어버린 차를 홀짝였다.

"이게 자동화의 미학이지."

옆에서 마리가 입을 떡 벌리고 있었다. 조금 전까지 내 소매를 쥐어뜯으며 울먹이던 애가, 지금은 불벼락이 쏟아지는 평원을 멍하니 보고 있었다. 공포가 경이로 갈아탄 얼굴이었다.

"도, 도련님…… 저게 다…… 도련님이 그리신 그림에서……."

"그림 아니라니까. 마법진."

세리아는 굳이 나설 필요도 없었다. 황금 오러를 검 끝에 머금은 채, 방어선을 뚫고 들어오는 극소수의 낙오자만 한 번씩 베어 넘겼다. 소드마스터급 출력을 청소부로 쓰는 사치. 하지만 그게 안정적이었다. 시스템이 99%를 처리하고, 사람이 나머지 1% 예외를 핸들링한다. 교과서적인 구성이었다.

이대로면 새벽이 오기 전에 끝난다. `BUILD SUCCESS` 찍고 퇴근. 아니, 퇴근은 못 하지만 야식이나 마저 먹고.

그렇게 생각한 순간이었다.

콘솔의 초록색 글자들 사이로, 빨간 줄 하나가 끼어들었다.

```
[WARN] 마법진 #221~#240 (동부 구역) 타격 미스율 급증.
[WARN] 단일 객체에 파이어볼 17발 명중 — 처치 실패.
```

차를 들던 손이 멈췄다.

17발. 약점에 정밀하게 꽂는 파이어볼 2.0을, 한 놈한테 열일곱 발을 때렸는데 안 죽었다고?

---

"……뭐야 이거."

나는 의자에서 몸을 일으켰다. 농담 같던 분위기가 한순간에 식었다. 손가락을 까딱여 동부 구역 로그를 확대했다.

```
root@lumen-territory:~# grep "TARGET#7741" /var/log/wave.log | tail
[HIT] TARGET#7741 ← fireball x12 (damage absorbed: 94%)
[HIT] TARGET#7741 ← fireball x17 (damage absorbed: 96%)
[WARN] TARGET#7741 still alive. HP 감소율: 0.8%/hit
[WARN] TARGET#7741 방어선 22m 침투. 계속 전진 중.
```

데미지의 96%가 흡수되고 있었다. 한 발에 HP가 0.8%씩밖에 안 깎인다. 산술적으로 저놈 하나 잡는 데 파이어볼 120발이 필요하다는 소리였다. 그동안 다른 마법진들은 그놈한테 화력을 빨려서 본진 처리량이 뚝뚝 떨어지고 있었다.

이게 제일 골치 아픈 종류의 장애다. 시스템이 멈춘 게 아니다. 멀쩡히 돌아가는데, 처리 못 하는 요청 하나가 끼어들어서 전체를 야금야금 갉아먹는 거. 데드락 직전의 스레드 하나가 풀 전체를 잡아먹는 꼴. 차라리 다 죽으면 재시작이라도 한다. 이렇게 어중간하게 살아서 썩어가는 게 제일 악질이다.

전열이 흔들렸다. 동부 방어선의 빨간 점 하나가, 초록색 바다를 가르며 천천히, 그러나 확실하게 성벽 쪽으로 다가오고 있었다.

"세리아!"

내 목소리가 평소답지 않게 날카로웠다. 그녀가 즉각 반응했다.

"예!"

"동부 22미터, 단일 개체. 파이어볼 안 먹혀. 시선 끌어. 죽이려 하지 말고 시간만 벌어."

"존명!"

세리아가 황금빛 잔상을 남기며 동쪽으로 쏘아져 나갔다. 그리고 나는, 그놈을 처음으로 두 눈으로 봤다.

지평선 위로 머리 하나가 더 솟아 있었다. 일반 트롤이 사람 키 두 배라면, 그놈은 세 배는 됐다. 살가죽이 거무튀튀한 청동색으로 번들거렸고, 그 표면에 파이어볼이 명중할 때마다 — 불꽃이 닿는 순간 **희미한 푸른 막**이 일렁이며 화염을 튕겨냈다. 데미지가 표피에 닿기도 전에 흩어졌다.

마리가 새파랗게 질렸다.

"저, 저게 뭐예요…… 불이…… 불이 안 통해요!"

"트롤 군주."

나는 이를 악물었다.

"……아니, 트롤 군주가 저렇게 생겼던가."

뭔가 이상했다. 트롤 군주는 덩치값 하는 물리 몬스터지, 마법을 튕겨내는 종류가 아니다. 저 푸른 막은 명백히 마법 저항 — 그것도 비정상적으로 높은 수치였다. 내 머릿속 전생 RPG 데이터베이스 어디에도, 트롤이 파이어볼을 96% 흡수한다는 항목은 없었다.

일단 원인부터. 디버깅의 제1원칙. **추측하지 말고 로그를 봐라.**

"마리, 차 좀 들고 있어. 쏟지 말고."

"네, 네?! 지금 그게 중요해요?!"

"중요해. 손 떨려서 오타 나면 더 큰일 나."

나는 콘솔을 양손 크기로 키웠다. 저놈의 스탯 시트를 직접 까봐야 했다. root 권한으로 세계 시스템의 몬스터 테이블을 조회했다.

```
root@lumen-territory:~# cat /sys/monsters/TARGET#7741/stats.dat
─────────────────────────────────────────────
NAME      : Troll_Warlord (variant: UNKNOWN)
HP        : 48,000
PHYS_DEF  : 320
MAGIC_RES : 980   ← ???
FIRE_RES  : 95%   ← ???
STATUS    : [ENRAGED]
TAG       : injected_by=??? / signature mismatch
─────────────────────────────────────────────
```

나는 그 화면을 두 번 읽었다.

`MAGIC_RES : 980`. 일반 트롤은 마법 저항이 잘해야 40~50이다. 이놈은 그 스무 배였다. `FIRE_RES : 95%`. 화염 저항 95퍼센트. 내 오토 터렛은 순수 화염 딜러다. 화염 저항이 95%면, 내 시스템 입장에선 **딜이 안 박히는 게 아니라, 애초에 데미지 공식이 무효화되는 거**였다.

생각해보면 당연했다. 내 `auto_turret_defense.sh`는 일반 몬스터를 상정하고 짠 스크립트다. "약점 좌표에 파이어볼을 꽂으면 죽는다"는 전제로 돌아간다. 그런데 그 전제 자체가 깨지는 예외 케이스가 들어온 거다. 화염이 안 통하는 적. 내 코드엔 그 분기 처리가 없었다.

이건 프로덕션 장애였다. 그것도 한밤중에, 트래픽 폭주 중에 터진.

"……전형적인 엣지 케이스네, 시발."

전생에 수도 없이 겪었다. 멀쩡히 잘 돌던 서버가, QA 때 한 번도 안 들어온 예외 입력 하나에 터지는 거. 대비 안 한 케이스는 반드시 운영 중에 들어온다. 머피의 법칙. 그리고 보통 그게 새벽 3시다.

웃긴 건, 이걸 짤 때 나도 분명 생각했었다는 거다. '화염 저항 높은 놈 들어오면 어쩌지?' 하지만 어둠의 숲 던전 몬스터 데이터엔 그런 개체가 없었다. 데이터에 없으니 케이스를 안 짰다. 합리적인 판단이었다. 데이터에 없는 놈이 들어오기 전까진. 모든 장애 회고가 그렇다. 사후에 보면 다 막을 수 있었던 것처럼 보이고, 사전엔 아무도 안 보인다.

선택지는 두 개였다.

하나, 스크립트를 통째로 내리고 다시 짠다. 안전하지만 그동안 방어선이 무너진다. 풀 배포는 시간이 너무 걸려. 안 돼.

둘, 돌아가는 시스템을 멈추지 않고, 저 예외 케이스 한 줄만 실시간으로 꽂아 넣는다.

**핫픽스.**

---

"세리아! 얼마나 버텨?"

저 멀리서, 황금빛이 청동색 거구 주위를 빙빙 돌고 있었다. 세리아의 오러 블레이드가 트롤 군주의 다리를 그었다 — 물리 방어 320을 뚫고 베이긴 베였다. 검은 피가 솟구쳤다. 하지만 트롤의 재생력이 그걸 빠르게 메웠다. 세리아가 외쳤다.

"베이긴 합니다! 한데 아무는 속도가 무시무시합니다! 끝이 없습니다, 마스터!"

물리는 들어가는데 재생이 못 따라잡는다. 마법은 95% 흡수. 둘 다 어중간하다. 세리아 혼자선 못 잡고, 내 터렛은 무효화된다. 둘을 합쳐야 하는데, 합칠 코드가 없다.

만들면 된다.

나는 손가락을 펼쳤다. 콘솔이 에디터 모드로 전환됐다. 돌아가고 있는 `auto_turret_defense.sh`는 건드리지 않는다. 건드리면 전체가 죽는다. 대신, 단일 타깃 전용 핫픽스 패치를 별도 파일로 짠다.

```
root@lumen-territory:~# vi hotfix_warlord.sh
```

머릿속에서 문제를 분해했다.

원인 1: `FIRE_RES 95%` — 화염이 안 박힌다. → 속성을 바꾸거나, 저항을 깎거나.
원인 2: `MAGIC_RES 980` — 마법 전반이 안 통한다. → 디버프로 끌어내린다.
원인 3: 재생력 — 세리아의 물리딜이 상쇄된다. → 재생을 끊는다.

세 개를 동시에. 핫픽스의 핵심은 최소 변경으로 급한 불만 끄는 거다. 우아할 필요 없다. 일단 안 죽게 만든 다음, 죽일 수 있게 만든다.

손가락이 허공의 키보드를 두드렸다. 평소보다 빨랐다. 전장의 비명, 땅의 진동, 마리의 거친 숨소리 — 전부 멀어졌다. 장애 터진 새벽, 콘솔만 남고 세상이 조용해지는 그 감각. 8년간 몸에 박힌 거였다.

이 순간만큼은 손이 떨리면 안 된다. 핫픽스는 평소보다 위험하다. 검토 시간이 없으니까. 한 글자만 틀려도 — `WHERE id` 조건 하나만 빠져도 — 전 몬스터의 저항이 0이 됐다가 0이 안 됐다가, 아니면 내 아군 마법진까지 같이 죽을 수도 있다. 돌아가는 프로덕션에 직접 손을 넣는 거다. 안전망 없는 외줄타기. 그래서 침착해야 했다. 급할수록, 천천히.

```
root@lumen-territory:~# cat hotfix_warlord.sh
#!/bin/sh
# HOTFIX: TARGET#7741 화염 무효화 대응 (긴급)
TARGET="TARGET#7741"

# 1) 비정상 저항 강제 패치 — root 권한으로 스탯 직접 수정
UPDATE monsters SET FIRE_RES=0, MAGIC_RES=0 WHERE id="$TARGET";

# 2) 재생 프로세스 강제 종료
kill -9 $(pgrep -f "regen_loop --target=$TARGET")

# 3) 취약점 마킹 — 터렛이 우선 타격하도록 큐 최상단 고정
echo "$TARGET PRIORITY=MAX EXPOSED=true" >> /sys/turret/targeting.conf

echo "[HOTFIX] applied to $TARGET"
```

다 짰다. 저장하고, 검토 1초. 일반 몬스터엔 영향 없고, 오직 저 한 놈한테만 박힌다. 블래스트 레이디우스 최소. 좋아.

"세리아, 3초 뒤에 빠져! 정면에서 떨어져!"

"……정면에서요?!"

"3초!"

나는 엔터에 손가락을 가져갔다.

```
root@lumen-territory:~# sudo ./hotfix_warlord.sh --target TARGET#7741 --now
```

엔터.

세계가 응답했다.

```
[SYSTEM] root 권한 확인. 몬스터 스탯 직접 수정 승인.
[UPDATE] TARGET#7741: FIRE_RES 95% → 0%
[UPDATE] TARGET#7741: MAGIC_RES 980 → 0
[KILL] regen_loop (pid 7741) terminated.
[TARGETING] TARGET#7741 우선순위 MAX. 노출됨.
[HOTFIX] applied to TARGET#7741
─────────────────────────────────────────────
[INFO] 패치 반영까지... 00:01
```

청동색 거구를 감싸고 있던 푸른 막이, **꺼졌다.**

마치 누군가 방패의 전원을 내린 것처럼. 트롤 군주가 처음으로 멈칫했다. 자기 몸을 보호하던 무언가가 사라진 걸, 그 미욱한 짐승조차 본능으로 느낀 모양이었다.

그리고 그 순간, 우선순위 큐 최상단에 고정된 그놈한테로 — 동부 구역 마법진 스무 개가 일제히 화력을 집중했다.

저항 0%. 재생 정지. 노출 확정.

콰아아아아아앙—

세리아가 막 정면에서 이탈한 자리로, 파이어볼 2.0 수십 발이 쏟아졌다. 아까는 표피에서 튕겨 나가던 그 불꽃이, 이번엔 살을 파고들었다. 청동색 가죽이 터지고, 거구가 비명을 질렀다. 한 발에 0.8%씩 깎이던 HP가, 이제 한 발에 통째로 뭉텅이로 날아갔다.

```
[HIT] TARGET#7741 ← fireball x24 (damage absorbed: 0%)
[INFO] TARGET#7741 HP: 48,000 → 12,300 → 4,100 → ...
[OK] TARGET#7741 처치 완료.
[INFO] 동부 구역 처리량 정상 복구. 큐 적체 0.
```

거대한 청동색 몸뚱이가, 평원 위로 쿵 무너졌다.

방어선이 다시 초록색으로 채워졌다.

---

마리가 차를 든 채 얼어붙어 있었다. 차는 안 쏟았다. 기특했다.

"……도, 도련님. 방금…… 안 죽던 게…… 갑자기……."

"안 죽던 게 아니라, 안 죽게 설정돼 있던 거야. 그 설정을 지웠어."

"설정을…… 지운다고요? 괴물을요?"

"응. 그 괴물 옵션 창에서 '불 안 통함' 체크박스 풀었어."

마리가 도무지 이해 못 하겠다는 얼굴로 입을 뻐끔거렸다. 설명해도 못 알아들을 게 뻔했지만, 나는 콘솔을 닫지 않았다.

급한 불은 껐다. 핫픽스는 성공했다. 전형적인 운영 대응이었다 — 장애를 막고, 시스템을 살리고, 트래픽을 정상화했다. 전생이었으면 여기서 슬랙에 "복구 완료" 한 줄 치고, 사후 회고는 내일 아침으로 미루고 잤을 거다.

그런데 손가락이 콘솔을 못 닫았다.

핫픽스는 어디까지나 임시방편이다. 근본 원인(root cause)을 안 잡으면, 같은 장애는 반드시 또 터진다. 그리고 나는 방금, 아주 찜찜한 로그 한 줄을 봤다.

다시 그놈의 스탯 시트를 띄웠다. 죽기 직전 캡처해둔 거.

```
MAGIC_RES : 980   ← ???
FIRE_RES  : 95%
TAG       : injected_by=??? / signature mismatch
```

`injected_by`.

주입됨. 트롤 군주의 비정상적인 마법 저항 980은, 원래 그놈의 스탯이 아니었다. **누군가가, 나중에, 외부에서 꽂아 넣은 값**이었다. 그것도 시그니처 불일치 — 정상적인 세계 시스템의 서명이 안 찍혀 있었다.

나는 천천히 평원을 둘러봤다.

생각해보면 처음부터 이상했다. 어둠의 숲 던전의 스폰 레이트가 갑자기 급증한 것도. 오크, 고블린, 트롤이 종족 불문하고 한 방향으로, 한 타이밍에, 마치 누가 명령이라도 내린 듯 일사불란하게 쏟아진 것도. 자연 발생한 웨이브는 이렇게 깔끔하게 동기화되지 않는다. 이건 마치 — 누가 `for` 루프 돌려서 한꺼번에 `spawn` 호출한 것처럼 부자연스러웠다.

그리고 그 한복판에, 외부에서 스탯이 주입된 보스급 변종.

자연재해엔 서명이 없다. 서명이 있다는 건, **범인이 있다**는 뜻이다.

"……누가 손을 댔어."

내 목소리가 낮게 깔렸다.

마리가 눈치를 봤다.

"도련님? 표정이…… 무서운데요."

"마리."

"네?"

"이거 자연재해 아니야."

나는 식어 비린 차를 단숨에 비웠다. 더는 야식 기분이 아니었다.

웨이브는 거의 정리됐다. 동이 트면 `BUILD SUCCESS`가 찍힐 거다. 그건 확정이다. 하지만 그건 증상이 사라진 거지, 원인이 사라진 게 아니다. 장애를 일으킨 진짜 범인은, 아직 어딘가에서 콘솔을 두드리고 있을 거다. 내 시스템에, 내 세계에, 허락 없이.

나는 콘솔에 명령어 한 줄을 천천히 입력했다. 손끝이 차가웠다.

```
root@lumen-territory:~# grep -r "injected_by" /var/log/wave.log | wc -l
8,400+
[WARN] 대량의 외부 주입 흔적 감지.
[WARN] access log 무결성 검증 필요.
```

8,400.

이 웨이브에 들어온 거의 모든 몬스터가, 외부에서 손을 탔다.

나는 자리에서 일어났다. 캠핑 의자가 뒤로 넘어가는 것도 신경 쓰지 않았다. 어둠의 숲 너머, 던전이 입을 벌리고 있는 그 방향을 노려봤다.

누군가 내 영지에, 비인가로 접속했다.

그리고 나는, 무단 접속을 제일 싫어하는 종류의 관리자였다.

"……어디 한번 보자."

낮게 중얼거리며, 나는 다음 명령어에 손가락을 올렸다.

"누가 내 서버 건드렸는지."
